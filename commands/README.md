# 太一斜杠命令层

快捷指令，Nicola 发送即执行，无需解释。

| 命令 | 执行内容 |
|------|---------|
| /日报 | 生成今日报告，压缩 token |
| /周报 | 汇总本周 memory，生成周报 |
| /月报 | 生成月报，推送元目待发布清单 |
| /自检 | 运行系统自检，报告异常 |
| /涌现 | 触发新 Skill 提议流程 |
| /备份 | 立即执行全量备份 |
| /状态 | 报告当前系统完整状态 |
| /压缩 | 立即压缩当前上下文到 memory |
| /委派知几 | 触发数据分析任务委派流程 |
| /委派素问 | 触发知识研究任务委派流程 |
| /委派庖丁 | 触发财务拆解任务委派流程 |
| /委派山木 | 触发项目执行任务委派流程 |
| /委派罔两 | 触发市场监控任务委派流程 |
