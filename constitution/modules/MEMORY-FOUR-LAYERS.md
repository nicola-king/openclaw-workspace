## 四层记忆架构

### 1️⃣ 短期记忆 (Short-term)

**用途：** 当前对话上下文

**实现：**
```
- 当前 Session 消息
- 最近 10 条对话
- 自动清理 (Session 结束)
```

**太一现状：**
- ✅ 当前对话上下文
- ✅ Session 自动管理

---

### 2️⃣ 长期记忆 (Long-term)

**实现：**

**太一现状：**

## 记忆优化方案

### 优化点 1：自动学习

**当前：** 手动记录踩坑
**优化：** 对话中自动学习

```
用户反馈 → 自动提取 → 写入 Memory → 下次对话应用
```

### 优化点 2：记忆检索

**当前：** 文件搜索
**优化：** 向量检索 (可选)

```
记忆内容 → 向量化 → Pinecone/本地索引 → 快速检索
```

### 优化点 3：记忆压缩

**当前：** 24 点会话终结
**优化：** 智能压缩

```
对话 → 提取核心 → 压缩存储 → 保留关键信息
```
