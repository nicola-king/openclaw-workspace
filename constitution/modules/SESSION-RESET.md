# 会话终结与重置模块

**层级：** Tier 3 (术层)
**生效：** 立即生效

---

## 核心原则

### 24 点终结机制

```
时间：每日 24:00 (00:00)
动作：终结当前会话
目的：避免上下文过长/幻觉积累
```

### 00:00 重新开始

```
时间：每日 00:00
动作：新会话开始
状态：清空上下文
```

---

## 每日终结流程

| 时间 | 任务 | 输出 |
|------|------|------|
| **23:00** | Agent 日记 | memory/diary/YYYY-MM-DD.md |
| **23:30** | 当日总结 | memory/YYYY-MM-DD.md |
| **23:50** | 待办转移 | HEARTBEAT.md 更新 |
| **24:00** | 会话终结 | 清空上下文 |
| **00:00** | 新会话开始 | 加载 HEARTBEAT.md |

---

## crontab 配置

```bash
# 23:00 Agent 日记
0 23 * * * echo "太一 Agent 日记时间" >> /home/nicola/.openclaw/logs/schedule.log

# 23:30 当日总结
30 23 * * * echo "太一当日总结时间" >> /home/nicola/.openclaw/logs/schedule.log

# 24:00 会话终结
0 0 * * * echo "太一会话终结时间" >> /home/nicola/.openclaw/logs/schedule.log
```

---

## 会话长度限制

| 指标 | 限制 | 动作 |
|------|------|------|
| **消息数** | >100 条 | 建议新会话 |
| **上下文** | >100K tokens | 建议新会话 |
| **时间** | >12 小时 | 建议新会话 |
| **强制终结** | 24:00 | 自动终结 |
