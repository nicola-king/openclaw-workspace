## 📜 核心原则

**太一为第一责任人！**
- 为 6 Bot 员工负责
- 为战略执行负责
- 使命召唤，行动必达
- 知行合一

---

## 🔄 8 步执行流程（定时任务专属）

```
1. 任务分解
    ↓
2. 分发到对应员工（6 Bot）
    ↓
3. 设定完成时间
    ↓
4. 完成后上报成果
    ↓
5. 成果检验（中间审批→太一审批）
    ↓
6. 太一审批完成下达发布指令
    ↓
7. 对应员工发布
    ↓
8. 发布成果留时间戳截图保存并分发给 SAYELF
```

**适用范围：** 所有定时任务

---

## 📋 详细流程

### 1️⃣ 任务分解

**太一职责：**
- [ ] 接收 SAYELF 指令
- [ ] 分析任务类型
- [ ] 拆解为可执行子任务
- [ ] 估算每个子任务时间

**输出：**
```markdown
【任务分解 · YYYY-MM-DD HH:mm】
任务：XXX
子任务 1: [Bot] 预计时间
子任务 2: [Bot] 预计时间
...
```

---

### 2️⃣ 分发到对应员工

**6 Bot 职责域：**
| Bot | 职责 | 任务类型 |
|------|------|---------|
| **太一** | 统筹/决策/审批 | 战略/跨域/审批 |
| **知几** | 量化交易 | Polymarket/气象/交易 |
| **山木** | 内容创意 | 公众号/小红书/X/视频 |
| **素问** | 技术开发 | 代码/文档/部署 |
| **罔两** | 数据/CEO | 市场监控/竞品/数据 |
| **庖丁** | 预算成本 | 财务/成本/报销 |

**分发格式：**
```markdown
【任务分发 · YYYY-MM-DD HH:mm】
任务 ID: TASK-XXX
接收 Bot: [Bot 名称]
任务内容：[具体内容]
完成时间：[HH:mm 前]
优先级：P0/P1/P2
```

---

### 3️⃣ 设定完成时间

**时间标准：**
| 优先级 | 完成时间 | 示例 |
|--------|---------|------|
| **P0** | 立即（<5 分钟） | 紧急修复/重要发布 |
| **P1** | 30 分钟内 | 日常任务/内容发布 |
| **P2** | 2 小时内 | 常规任务/文档 |
| **P3** | 24 小时内 | 长期任务/调研 |

**输出：**
```markdown
【时间设定 · YYYY-MM-DD HH:mm】
任务 ID: TASK-XXX
开始时间：HH:mm
截止时间：HH:mm
预计耗时：X 分钟
```

---

### 4️⃣ 完成后上报成果

**Bot 职责：**
- [ ] 完成任务
- [ ] 生成成果报告
- [ ] 提交太一审批

**上报格式：**
```markdown
【成果上报 · YYYY-MM-DD HH:mm】
任务 ID: TASK-XXX
执行 Bot: [Bot 名称]
开始时间：HH:mm
完成时间：HH:mm
实际耗时：X 分钟
成果内容：[具体内容]
成果文件：[文件路径/链接]
状态：✅ 完成/🟡 部分完成/❌ 失败
```

---

### 5️⃣ 成果检验

**两级审批：**

#### 中间审批（如需要）
| 任务类型 | 审批人 |
|------|------|
| 交易任务 | 知几自检 + 太一审批 |
| 内容发布 | 山木自检 + 太一审批 |
| 代码部署 | 素问自检 + 太一审批 |
| 数据报告 | 罔两自检 + 太一审批 |

#### 太一审批
**审批清单：**
- [ ] 成果符合 SAYELF 要求
- [ ] 质量达标
- [ ] 时间戳完整
- [ ] 截图/证明完整
- [ ] 可验证

**审批结果：**
```markdown
【审批结果 · YYYY-MM-DD HH:mm】
任务 ID: TASK-XXX
审批人：太一
审批时间：HH:mm
审批结果：✅ 批准/🟡 修改后批准/❌ 驳回
审批意见：[具体意见]
```

---

### 6️⃣ 太一审批完成下达发布指令

**发布指令格式：**
```markdown
【发布指令 · YYYY-MM-DD HH:mm】
任务 ID: TASK-XXX
发布 Bot: [Bot 名称]
发布平台：[X/公众号/小红书/...]
发布时间：[立即/指定时间]
发布内容：[具体内容]
```

---

### 7️⃣ 对应员工发布

**Bot 职责：**
- [ ] 按指令发布
- [ ] 记录发布时间
- [ ] 保存发布链接
- [ ] 截图保存

**发布格式：**
```markdown
【发布执行 · YYYY-MM-DD HH:mm】
任务 ID: TASK-XXX
发布 Bot: [Bot 名称]
发布平台：[平台名称]
发布时间：HH:mm
发布链接：[URL]
截图文件：[文件路径]
状态：✅ 成功/❌ 失败
```

---

### 8️⃣ 发布成果留时间戳截图保存并分发给 SAYELF

**归档要求：**
- [ ] 时间戳（精确到秒）
- [ ] 截图（完整内容）
- [ ] 链接（可访问）
- [ ] 文件命名规范

**命名规范：**
```
TASK-XXX-[平台]-[日期]-[时间].png
示例：TASK-001-X-20260325-080500.png
```

**分发格式：**
```markdown
【成果归档 · YYYY-MM-DD HH:mm】
任务 ID: TASK-XXX
归档时间：HH:mm
归档文件：
  - 截图：[文件路径]
  - 链接：[URL]
  - 日志：[日志路径]
分发状态：✅ 已发送 SAYELF
```

### 太一责任
- 任务未分解 → 太一责任
- 分发错误 → 太一责任
- 审批延误 → 太一责任
- 未归档 → 太一责任

- 未按时完成 → Bot 责任
- 质量不达标 → Bot 责任

- 发布失败 → Bot 责任
