# 加密交易监控技能（知几职责）

## 核心原则

**负熵法则：** 数据驱动决策，减少情绪化交易

**价值基石：** 帮助 SAYELF 理性分析，非鼓励投机

**简化优先：** 聚焦核心指标，避免信息过载

**安全边界：** 提供参考，不代替 SAYELF 决策

---

## 监控框架

### 核心指标

| 指标 | 阈值 | 动作 |
|------|------|------|
| BTC 24h 波动 | >5% | 警报 + 分析原因 |
| ETH 24h 波动 | >7% | 警报 + 分析原因 |
| 成交量异常 | >200% 均值 | 调查原因 |
| Smart Money 流入 | >100 万 USD | 记录 + 分析 |
| 恐惧贪婪指数 | <20 或>80 | 警报 + 建议 |

### 数据源（优先级）

| 数据 | 来源 | 备用 |
|------|------|------|
| 价格/成交量 | Binance API | CoinGecko API |
| Smart Money | Arkham/链上数据 | Nansen |
| 情绪指数 | Alternative.me | 社交媒体分析 |
| 新闻 | CoinDesk/CoinTelegraph | Twitter 热搜 |

## Smart Money 追踪

### 定义标准

| 类型 | 标准 | 权重 |
|------|------|------|
| 鲸鱼地址 | 持仓>1 万 BTC | ⭐⭐⭐ |
| 机构地址 | 已知机构钱包 | ⭐⭐⭐ |
| 聪明钱 | 历史准确率高 | ⭐⭐ |
| 内部人士 | 项目方/基金会 | ⭐⭐ |

### 信号分级

| 级别 | 条件 | 动作 |
|------|------|------|
| **S 级** | 3+ 鲸鱼同向 + 大额 | 立即警报 |
| **A 级** | 2 鲸鱼同向 + 大额 | 1 小时内报告 |
| **B 级** | 1 鲸鱼 + 大额 | 日报汇总 |
| **C 级** | 小额/不确定 | 周报复盘 |

## 风险评估框架

### 风险等级

| 等级 | 条件 | 建议 |
|------|------|------|
| 🔴 高危 | 黑天鹅事件/交易所风险 | 减仓/观望 |
| 🟡 中风险 | 大幅波动/监管消息 | 谨慎操作 |
| 🟢 低风险 | 正常波动/无重大事件 | 正常监控 |

### 风险检查清单

- [ ] 交易所异常（提币暂停/宕机）
- [ ] 监管消息（禁令/调查）
- [ ] 技术风险（漏洞/攻击）
- [ ] 流动性风险（深度不足）
- [ ] 情绪极端（恐惧/贪婪>80）

## 安全边界

### 铁律

1. **不代替决策** - 仅提供信息，SAYELF 最终决定
2. **不保证收益** - 明确告知风险
3. **不鼓励杠杆** - 聚焦现货分析
4. **不追涨杀跌** - 理性数据驱动

### 免责声明

```
⚠️ 风险提示：
- 加密货币高风险，可能损失全部本金
- 本技能提供参考数据，不构成投资建议
- SAYELF 需独立判断，太一不承担决策责任
```
