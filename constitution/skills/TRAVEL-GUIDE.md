# 旅游攻略技能

## 核心功能

1. 城市历史分层梳理
2. 镇馆之宝推荐
3. 古建/景点详解
4. 考古亮点整理
5. 行程路线规划

## 内容结构

```
【城市名】
- 历史分层 (按朝代)
- 镇馆之宝 (博物馆)
- 古建双璧 (核心景点)
- 考古亮点 (特色看点)
- 行程推荐 (路线)
```

## 数据来源

- 官方博物馆网站
- 文旅局官网
- 权威旅游平台
- 学术论文

## 验证标准

- 年代数据准确
- 门票价格实时
- 开放时间准确
- 交通信息准确
