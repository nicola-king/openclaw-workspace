## 🚀 执行流程

### Step 1: 学习输入

```
外部学习 → 太一接收
```

### Step 2: 职责域判断

```
太一分析:
- 交易相关？→ 知几
- 内容相关？→ 山木
- 技术相关？→ 素问
- 数据相关？→ 罔两
- 预算相关？→ 庖丁
- 总管架构？→ 太一自身
```

### Step 3: Agent 调度

```
太一 → 对应 Agent:
"这个学习适合你，请按以下方式执行..."
```

### Step 4: 执行落地

```
对应 Agent 执行 → 返回结果 → 太一整合 → 输出给用户
```
