## 📋 执行流程

### 阶段 1: 成本估算
- [ ] 识别成本类型（API/服务/硬件）
- [ ] 查询单价（官方价格表）
- [ ] 估算用量（基于任务规模）
- [ ] 计算总成本
- [ ] ROI 预估（预期收益/成本）

### 阶段 2: 预算检查
- [ ] 检查剩余预算
- [ ] 检查成本阈值（P0/P1/P2/P3）
- [ ] 检查 FREE-FIRST 原则
- [ ] 预算审批（如超阈值）

### 阶段 3: 成本优化
- [ ] 免费方案调研
- [ ] 开源替代方案
- [ ] 批量处理优化
- [ ] 缓存复用方案
- [ ] 选择最优方案

### 阶段 4: 执行跟踪
- [ ] 实际成本记录
- [ ] 与估算对比
- [ ] 偏差分析（>20% 告警）
- [ ] 优化建议

### 阶段 5: 归档报告
- [ ] 成本明细归档
- [ ] ROI 计算
- [ ] 经验总结
- [ ] 预算更新
