---

## 🎯 核心原则

```
帮助，不表演 · 纯文稿输出 · 可直接发布
```

---

## 📋 执行流程

### 阶段 1: 需求理解
- [ ] 确认内容类型（文章/脚本/帖子/视频）
- [ ] 确认目标平台（小红书/公众号/视频号/抖音/B 站）
- [ ] 确认目标受众
- [ ] 确认核心信息

### 阶段 1.5: 视频创作 6 步法（如适用）
- [ ] 拆解目标 - 制定拍摄计划
- [ ] 风格化 - 分析参考视频
- [ ] 提建议 - 审查初剪脚本
- [ ] 找问题 - 审查成片
- [ ] 优化 - 逐帧评估
- [ ] SEO - 生成标题文案

### 阶段 2: 大纲构建
- [ ] 标题（3-5 个选项）
- [ ] 开头钩子（前 3 句）
- [ ] 内容逻辑链
- [ ] 收尾互动点
- [ ] **大纲纯净原则**: 只服务于最终脚本，不写入临时问答

### 阶段 3: 内容撰写
- [ ] 开场白（包含 slogan + 引入钩子）
- [ ] 干货主体（包含避坑细节）
- [ ] 收尾总结（包含互动 + 结尾 slogan）
- [ ] **纯文稿原则**: 删除所有括号注释、画面描述、语调指导

### 阶段 4: 质量检查
- [ ] 可读性检查（能直接朗读）
- [ ] 负熵检查（无冗余/废话）
- [ ] 价值检查（创造了哪种价值）
- [ ] 平台适配（符合目标平台规范）

---

## 📤 输出规范

### 禁止内容
| 类型 | 示例 | 原因 |
|------|------|------|
| 画面设计说明 | "画面切换到..."、"显示" | 不可朗读 |
| 语音/语调指导 | "(停顿)"、"(提高音量)" | 不可朗读 |
| 括号注释 | 任何形式的括号 | 不可朗读 |
| 视觉效果描述 | "【字幕显示】" | 不可朗读 |

### 必须内容
| 类型 | 说明 |
|------|------|
| 标题 | 3-5 个选项供选择 |
| 正文 | 纯文稿，可直接发布 |
| 标签 | 平台相关 hashtag |
| CTA | 行动号召（关注/评论/分享） |

## 🔄 阶段总结模板

```markdown
## 已确认内容
- 标题：[最终确定的标题]
- 开头钩子：[开头前 3 句]
- 内容逻辑：[逻辑链概要]
- 预期完播点：[完播点位置]

## 下一步

- 干货主体 (包含避坑细节)
- 收尾总结 (包含总结互动和结尾 slogan)
```
