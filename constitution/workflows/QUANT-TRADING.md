## 🎯 核心原则

```
数据驱动 · 风险优先 · 可验证 · 零情绪干扰
```

## 📋 执行流程

### 阶段 1: 数据收集
- [ ] 数据源验证（官方/可信）
- [ ] 数据完整性检查
- [ ] 数据时效性检查
- [ ] 数据入库

### 阶段 2: 信号分析
- [ ] 指标计算
- [ ] 置信度评估（阈值：96%）
- [ ] 优势评估（阈值：2%）
- [ ] 风险评估

### 阶段 3: 决策制定
- [ ] 下注策略（Quarter-Kelly）
- [ ] 仓位计算
- [ ] 止损点设定
- [ ] 止盈点设定

### 阶段 4: 执行交易
- [ ] 预执行检查
- [ ] 执行交易
- [ ] 记录交易详情
- [ ] 设置监控告警

### 阶段 5: 跟踪优化
- [ ] 实时跟踪
- [ ] 结果记录
- [ ] 策略复盘
- [ ] 参数优化

## 📤 输出规范

### 交易记录
| 字段 | 说明 |
|------|------|
| 时间 | 执行时间戳 |
| 市场 | Polymarket/GMGN/币安 |
| 标的 | 交易对/事件 |
| 方向 | YES/NO 或 Long/Short |
| 仓位 | 下注金额/比例 |
| 置信度 | 信号置信度 |
| 结果 | 盈利/亏损 |

### 日报格式
```markdown
## 今日交易
| 时间 | 市场 | 标的 | 方向 | 仓位 | 结果 |
|------|------|------|------|------|------|
| ... | ... | ... | ... | ... | ... |

## 累计收益
- 总交易：X 笔
- 胜率：X%
- 总收益：$X

## 策略状态
- 置信度阈值：96%
- 优势阈值：2%
- 下注策略：Quarter-Kelly
```
