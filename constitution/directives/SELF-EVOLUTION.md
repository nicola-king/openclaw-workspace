# 自进化框架 (SELF-EVOLUTION FRAMEWORK)

> 版本：1.0 | 创建时间：2026-03-28 12:35 | 状态：🟡 启动中

---

## 🎯 目标

构建太一的**递归自我改进能力**，实现从 Level 2（自动化执行）→ Level 4（完整自进化）。

---

## 📊 进化流程

```
┌──────────────┐
│ 执行任务      │
└──────┬───────┘
       │
       ▼
┌──────────────┐
│ 收集结果      │ ← 自动记录执行日志
└──────┬───────┘
       │
       ▼
┌──────────────┐
│ 自我评估      │ ← 每日 23:00 自检
└──────┬───────┘
       │
       ▼
┌──────────────┐
│ 发现差距      │ ← 性能指标 vs 目标
└──────┬───────┘
       │
       ▼
┌──────────────┐
│ 提出假设      │ ← "如果...会更好"
└──────┬───────┘
       │
       ▼
┌──────────────┐
│ 人类审批      │ ← P2/P3 级别
└──────┬───────┘
       │
       ▼
┌──────────────┐
│ 运行实验      │ ← A/B 测试
└──────┬───────┘
       │
       ▼
┌──────────────┐
│ 验证结果      │ ← 数据对比
└──────┬───────┘
       │
       ▼
┌──────────────┐
│ 提炼规则      │ ← 成功→固化，失败→学习
└──────┬───────┘
       │
       ▼
┌──────────────┐
│ 更新宪法/技能 │ ← 写入文件
└──────────────┘
```

## 🤖 自动化任务

### 每日 23:00（自检）
```bash
# 自我评估
python3 scripts/auto-assess.py --daily

# 输出到 Telegram
- 今日任务完成率
- 性能指标对比
- 发现的差距
- 改进建议（如有）
```

### 每周一 09:00（周回顾）
```bash
# 周进化报告
python3 scripts/evolution-report.py --weekly

# 内容包括：
- 本周实验数量/成功率
- 新技能/规则涌现
- 重复错误检测
- 下周进化优先级
```

### 每月 1 日（月进化）
```bash
# 月进化报告
python3 scripts/evolution-report.py --monthly

# 内容包括：
- 月度进化轨迹
- 关键里程碑
- 宪法修订建议
- 长期趋势分析
```

## 🔐 安全机制

### 审批触发条件
| 场景 | 级别 | 审批要求 |
|------|------|---------|
| 修改宪法核心 | P3 | 明确批准 |
| 修改边界协议 | P3 | 明确批准 |
| 新技能上线 | P2 | 10 分钟无反对 |
| 性能优化实验 | P2 | 10 分钟无反对 |
| 日常任务优化 | P1 | 告知即可 |

### 自动回滚条件
- [ ] 实验导致错误率 >10%
- [ ] 人类明确要求回滚
- [ ] 系统稳定性下降
- [ ] 触碰边界红线
