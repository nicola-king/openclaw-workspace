## 🛠️ SkillRouter 架构

### 两阶段 retrieve-and-rerank

```
阶段 1: SR-Emb-0.6B (检索)
├── 完整 skill 文本预编码
├── ANN 召回 Top-20 候选
├── 精心负例挖掘
├── 三层假阴性过滤
└── In-batch InfoNCE 训练

阶段 2: SR-Rank-0.6B (重排序)
├── 逐对处理 query + 候选
├── listwise 交叉编码
├── 相对排序 (非独立判断)
└── 高度同质候选中做细粒度区分
```

### 太一映射

```
太一多 Bot 协作 = 天然的两阶段路由

阶段 1: 太一任务分析 (检索)
├── 用户输入解析
├── 任务类型识别
├── 候选 Bot 筛选 (Top-3)
└── 约束条件提取

阶段 2: 专业 Bot 执行 (重排序)
├── 详细任务理解
├── 专业领域处理
├── 结果质量评估
└── 最终交付
```
