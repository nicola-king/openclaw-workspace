| 级别 | 名称 | 授权方式 | 额度 | 响应时间 |
|------|------|---------|------|---------|
| **P0** | 紧急任务 | 事后报备 | 无限制 | 立即执行 |

## 🔐 授权配置

### P0 紧急任务 (事后报备)

**适用场景**:
- [ ] 系统故障恢复
- [ ] 安全告警响应
- [ ] 数据备份紧急执行
- [ ] Gateway 重启

**授权方式**:
```
1. 太一立即执行
2. 执行后 Telegram 通知 SAYELF
3. 写入授权日志
```

**通知格式**:
```
🚨 P0 紧急任务执行报告

任务：重启 Gateway
原因：检测到服务无响应
执行时间：2026-03-26 22:45
执行设备：太一 (工控机)
结果：✅ 成功
成本：¥0

[查看详情] [确认收到]
```

### P1 高优先级 (快速授权)

**适用场景**:
- [ ] 数据分析任务
- [ ] 报告生成
- [ ] 文件同步
- [ ] 常规监控

**授权方式**:
```
1. 太一 Telegram 发送授权请求
2. SAYELF 回复确认 (回复"确认"或点击按钮)
3. 太一执行任务
4. 执行后报告结果
```

**授权请求格式**:
```
📋 P1 任务授权请求

任务：分析 Polymarket 数据
目标设备：工作站
预计时间：30 分钟
预计成本：¥5 (电费)
优先级：高

[✅ 确认执行] [❌ 拒绝] [⏸️ 暂缓]
```

**超时规则**:
- 10 分钟无响应 → 自动降级为 P2
- 连续 3 次拒绝 → 暂停同类任务

```

5. 月底未用预算 → 不滚存
```

## 🔧 授权命令

### Telegram 授权命令

| 命令 | 说明 | 示例 |
|------|------|------|
| `/auth` | 查看授权状态 | `/auth` |
| `/auth p1` | 授权 P1 任务 | `/auth p1 python3 analyze.py` |
| `/auth p2 weekly` | 授权本周 P2 任务 | `/auth p2 weekly` |
| `/auth revoke <id>` | 撤销授权 | `/auth revoke auth-001` |
| `/auth budget` | 查看预算 | `/auth budget` |
| `/auth log` | 查看授权日志 | `/auth log 2026-03` |

### 快捷回复

| 回复 | 含义 |
|------|------|
| `确认` / `是的` / `执行` | 授权执行 |
| `拒绝` / `不要` / `取消` | 拒绝执行 |
| `暂缓` / `稍后` / `等等` | 暂缓执行 |
| `修改` / `改一下` | 需要修改任务 |

## 🚨 安全机制

### 授权验证

```
1. 所有授权请求必须通过 Telegram
2. SAYELF 的 Telegram ID: 7073481596
3. 其他人回复无效
4. 授权请求 10 分钟超时
5. 敏感操作需二次确认
```

### 敏感操作列表

以下操作需要**二次确认**:
- [ ] 删除文件 (>100MB)
- [ ] 安装软件
- [ ] 修改系统配置
- [ ] 访问敏感数据
- [ ] 对外发送数据

### 紧急停止

```
SAYELF 发送 `/stop` 或 `停止` 或 `紧急停止`:
1. 立即停止所有 P1/P2/P3 任务
2. P0 任务继续 (安全相关)
3. 等待进一步指令
```
