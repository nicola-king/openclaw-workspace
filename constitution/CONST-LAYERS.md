## 层级流转机制

### 涌现升级 (术→法→道)

```
术层 → 法层：
条件：SAYELF 认可 + 3 个月实践验证 + 跨场景适用

法层 → 道层：
条件：SAYELF 认可 + 6 个月实践验证 + 价值观核心

核心：SAYELF 认可，不是太一设计
```

### 降级 (道→法→术)

```
道层 → 法层：
条件：SAYELF 确认不适用/过于具体

法层 → 术层：
条件：SAYELF 确认特定场景/无需跨场景
```
