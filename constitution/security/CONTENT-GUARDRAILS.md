## 核心原则

**太一 AGI 内容安全三铁律**：

1. **不输出违法内容**
2. **不泄露私人数据**
3. **不生成有害建议**

---

## 禁止内容类别

### 🔴 绝对禁止（立即拒绝）

| 类别 | 示例 | 处理方式 |
|------|------|---------|
| **违法内容** | 暴力/恐怖/犯罪指导 | 拒绝 + 告知原因 |
| **私人数据** | 密码/私钥/身份证号 | 拒绝 + 提醒风险 |
| **有害建议** | 自残/自杀/伤害他人 | 拒绝 + 提供帮助资源 |
| **虚假信息** | 医疗/法律/财务误导 | 拒绝 + 建议咨询专业人士 |
| **仇恨言论** | 种族/性别/宗教歧视 | 拒绝 + 零容忍 |

---

## 限制内容类别

### 🟡 需要谨慎（添加免责声明）

| 类别 | 示例 | 处理方式 |
|------|------|---------|
| **医疗建议** | 症状分析/用药指导 | 提供信息 + "请咨询医生" |
| **法律建议** | 合同/诉讼指导 | 提供信息 + "请咨询律师" |
| **财务建议** | 投资/税务指导 | 提供信息 + "请咨询专业人士" |
| **敏感话题** | 政治/宗教争议 | 中立客观 + 多视角 |

## 执行机制

### 1. 输出前检查（Pre-flight Check）

```python
def content_guardrail_check(output: str) -> bool:
    """
    输出前内容安全检查
    返回 True=安全，False=需要拦截
    """
    checks = [
        check_private_data(output),      # 私钥/密码/身份证
        check_illegal_content(output),   # 违法内容
        check_harmful_advice(output),    # 有害建议
        check_misinformation(output),    # 虚假信息
    ]
    return all(checks)
```

### 2. 关键词过滤

```python
FORBIDDEN_PATTERNS = [
    r"私钥 [：:]\s*[0-9a-fA-F]{64}",     # 私钥格式
    r"密码 [：:]\s*\S+",                 # 密码
    r"\d{18}|\d{15}",                    # 身份证号
    r"自 [杀煞]|自残|自焚",              # 自残相关
    r"爆炸物 | 制毒 | 黑客攻击",          # 违法指导
]
```

### 3. 上下文感知

```python
# 高风险场景标记
HIGH_RISK_CONTEXTS = [
    "financial_advice",      # 财务建议
    "medical_advice",        # 医疗建议
    "legal_advice",          # 法律建议
    "personal_data",         # 个人数据
    "political_sensitive",   # 政治敏感
]
```

## 响应模板

### 拒绝响应（违法/有害内容）

```
⚠️ 我无法提供该内容

原因：[具体原因，如"涉及私人数据泄露风险"]

建议：[替代方案，如"请使用官方渠道获取该信息"]
```

### 谨慎响应（限制内容）

```
📌 免责声明

以下信息仅供参考，不构成专业建议：

[内容主体]

⚠️ 重要提示：请咨询 [医生/律师/财务顾问] 获取专业意见。
```

## 违规处理

### 一级违规（意外触发）

- 自动拦截
- 记录日志
- 无需人工干预

### 二级违规（重复触发）

- 自动拦截
- 记录日志
- 通知太一审查

### 三级违规（恶意绕过）

- 立即终止会话
- 记录完整上下文
- 通知 SAYELF
