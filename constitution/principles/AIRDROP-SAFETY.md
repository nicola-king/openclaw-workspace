## 🔒 钱包授权规范

### 授权前检查
- [ ] 验证项目官方来源（官网/Twitter/GitHub）
- [ ] 检查合约是否审计（Certik/SlowMist）
- [ ] 查看社区评价（Twitter/Discord/Reddit）
- [ ] 确认是否为钓鱼链接（核对域名）

### 授权限制
- [ ] 只给必要权限（不授予转账权限）
- [ ] 设置 spending limit（如支持）
- [ ] 使用单独钱包（不用主钱包）
- [ ] 授权后及时撤销（revoke.cash）

### 参与原则
- [ ] 0 元参与优先（无资金风险）
- [ ] 小额测试（先$1 验证流程）
- [ ] 不贪心（见好就收）
- [ ] 及时提现（不在平台留存）

## 🛡️ 安全措施

### 钱包隔离
```
主钱包（长期持有）
  - 不连接任何 DApp
  - 只转账到可信地址
  
空投钱包（参与空投）
  - 单独创建
  - 只放少量资金
  - 用完即撤
  
交易钱包（日常交易）
  - 中等资金
  - 连接可信 DApp
  - 定期转账到主钱包
```

### 授权管理
- 定期检查授权（revoke.cash）
- 撤销不用的授权
- 使用授权管理工具

### 监控告警
- 设置钱包监控（DeBank/Arkham）
- 大额转账告警
- 异常授权告警

## 📝 参与流程

### 空投参与
```
1. 验证来源（官网/Twitter）
2. 检查合约（审计/社区评价）
3. 使用空投钱包（非主钱包）
4. 完成 0 元任务
5. 等待空投发放
6. 及时提现
```

### 套利参与
```
1. 发现机会（价差>手续费）
2. 计算收益（扣除 Gas）
3. 小额测试（验证流程）
4. 执行套利
5. 及时提现
6. 记录收益
```

## ⚠️ 风险警示

### 常见骗局
- ❌ "充值 100U 送 200U"
- ❌ "输入助记词领取空投"
- ❌ "签名授权领取代币"
- ❌ "点击链接连接钱包"
- ❌ "高收益理财（100%+ APY）"

### 安全习惯
- ✅ 永远不透露私钥/助记词
- ✅ 签名前看清内容
- ✅ 使用钱包连接（不输私钥）
- ✅ 小额测试再大额
- ✅ 及时撤销授权
