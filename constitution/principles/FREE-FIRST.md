## 📊 优先级排序

### 工具选择（从免费到付费）

| 优先级 | 类型 | 示例 | 使用条件 |
|--------|------|------|----------|
| **⭐⭐⭐⭐⭐** | 免费开源 | Nitter, snscrape, twurl | 首选，始终优先 |
| **⭐⭐⭐⭐** | 免费 API | web_search, web_fetch | 次选，零成本 |
| **⭐⭐⭐** | 低成本 | 低价 API 服务 | 免费方案失败后 |
| **⭐⭐** | 中成本 | 付费 API | 必要且 ROI>10x |
| **⭐** | 高成本 | browser 自动化 | 最后选择，ROI>100x |
| **❌** | 极高成本 | 付费服务 | 禁止，除非生死攸关 |
