### 步骤 2: 导航到产品编辑

```
登录后访问:
https://gumroad.com/dashboard/products

找到产品 "PolyAlert Lite" 或 "qdxnm"
点击 "Edit" 按钮
```

### 步骤 3: 配置交付内容

```
1. 点击 "Content" 标签页

2. 打开文件复制内容:
   cat /home/nicola/.openclaw/workspace/scripts/gumroad-delivery-content.txt

3. 全选复制 (Ctrl+A → Ctrl+C)

4. 粘贴到 Content 文本框

5. 点击 "Save changes" 保存
```

### 步骤 4: 上架产品

```
1. 点击 "Details" 标签页

2. 确认产品信息:
   - 名称：PolyAlert Lite - Free Polymarket Alerts
   - 价格：$0 (Free)
   - 状态：Published (已发布)

3. 如果显示"not currently for sale":
   - 找到 "Publish" 或 "Enable" 按钮
   - 点击上架
```
