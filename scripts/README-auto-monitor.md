## 🔧 使用方式

### 全自动模式（推荐）
```bash
# 采集 + 推送 + 同步
python3 scripts/auto-social-monitor.py --auto
```

### 仅推送
```bash
python3 scripts/auto-social-monitor.py --send
```

### 强制重新估算
```bash
python3 scripts/auto-social-monitor.py --force --send
```
