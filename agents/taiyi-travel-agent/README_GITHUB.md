## 🚀 Quick Start

```bash
# Clone
git clone https://github.com/nicola-king/taiyi-travel-agent.git
cd taiyi-travel-agent

# Install
pip install -r requirements.txt

# Use
python3 -c "from taiyi_travel_agent import TaiyiTravelAgent; agent = TaiyiTravelAgent(); print('Ready!')"
```

## 📋 Provider Onboarding

```bash
# Hotel
python3 provider_cli.py hotel register --name "Hotel" --location "Tokyo" --price 800 --rating 4.5

# Guide
python3 provider_cli.py guide register --name "Guide" --location "Tokyo" --language "Chinese/English" --price_per_day 800

# Charter
python3 provider_cli.py charter register --name "Charter" --location "Tokyo" --car_types Comfort --price_per_day 600
```
