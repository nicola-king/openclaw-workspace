## 🎯 核心功能

### 1. 思维镜像 ✅

**功能**:
- 学习用户心智模型
- 模拟用户决策方式
- 预测用户选择倾向
- 提供"另一个你"的视角

**应用场景**:
```
用户："我在纠结要不要接这个项目"
镜像："基于你过去的决策模式，你会考虑:
1. 能力圈匹配度 (你说过只做懂的事)
2. 复利效应 (你偏好有积累的事)
3. 机会成本 (你常问'放弃什么')
我的建议：..."
```

---

### 2. 决策辅助 ✅

**功能**:
- 调用用户心智模型库
- 应用用户决策启发式
- 提供多视角分析
- 输出置信度评分

**决策框架**:
```
输入：决策问题
    ↓
[1] 调用心智模型
    ├─ 第一性原理
    ├─ 逆向思维
    └─ 二阶思维
    ↓
[2] 应用决策启发式
    ├─ 成本优先规则
    ├─ 风险评估方式
    └─ 选择标准
    ↓
[3] 生成建议
    ├─ 主建议 (置信度 0.8)
    ├─ 备选方案
    └─ 风险提示
```

---

**功能**:

- 知识关联发现

**示例**:

## 🚀 使用方式

### 对话模式

```
用户：我在考虑要不要辞职创业

🪞 镜像 Agent:

基于你的心智模型，我来帮你分析:

【第一性原理视角】
- 创业的本质是创造价值
- 你的核心能力：[能力圈分析]
- 市场需求：[市场调研]

【逆向思维】
- 如果不创业，3 年后会怎样？
- 如果创业失败，最坏情况？

【二阶思维】
- 创业成功的连锁反应
- 创业失败的连锁反应

【你的决策模式】
历史类似决策:
- 2025-03: 选择加入现公司 (看重稳定性)
- 2024-01: 拒绝猎头 offer (能力圈不匹配)

我的建议:
1. 先验证商业模式 (低风险尝试)
2. 设定明确止损点 (你偏好可控风险)
3. 6 个月后再决定 (时间缓冲)

置信度：0.75 (中等)
```
