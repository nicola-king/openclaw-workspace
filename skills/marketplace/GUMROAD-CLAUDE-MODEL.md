## 🎯 Claude Skills 极简哲学

```
1. 小即是美 (Small is Beautiful)
   - 每个技能解决 1 个具体问题
   - 代码<500 行
   - 文档<1000 字

2. 服务核心受众 (Serve Core Audience)
   - 聚焦 Polymarket 玩家
   - 深度 > 广度
   - 专业 > 通用

3. 自动化优先 (Automate First)
   - 交付自动化 (Telegram Bot)
   - 营销自动化 (内容分发)
   - 客服自动化 (FAQ)

4. 可持续增长 (Sustainable Growth)
   - 不烧钱获客
   - 口碑驱动
   - 复购优先
```
