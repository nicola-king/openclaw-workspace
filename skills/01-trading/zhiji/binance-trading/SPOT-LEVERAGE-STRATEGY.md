# 币安现货杠杆策略

> 版本：v1.0 | 创建：2026-03-28 20:34
> 铁律：现货杠杆 2-3 倍，不开合约

---

## 🎯 现货杠杆核心

**允许**:
- ✅ 现货杠杆 2-3 倍 (合理开)
- ✅ BTC/ETH 双币种
- ✅ 随借随还 (按日计息)

**禁止**:
- ❌ 合约交易 (Futures)
- ❌ 高杠杆 (>5 倍)
- ❌ 山寨币杠杆

---

## 📊 现货杠杆 vs 合约

| 维度 | 现货杠杆 | 合约 | 太一选择 |
|------|---------|------|---------|
| **杠杆倍数** | 2-5 倍 | 最高 125 倍 | ✅ 2-3 倍 |
| **爆仓风险** | 低 (有预警) | 高 (强平) | ✅ 低风险 |
| **资金费率** | 无 | 有 (持仓成本) | ✅ 无成本 |
| **持仓时间** | 无限制 | 有限制 | ✅ 无限制 |
| **利息** | 按日计息 (~0.02%/日) | 资金费率 | ✅ 透明 |
| **强平** | 有预警线 | 立即强平 | ✅ 更安全 |

## 🎯 杠杆使用场景

### ✅ 适合开杠杆

1. **高置信度信号** (知几-E 置信度>98%)
2. **趋势明确** (均线多头/空头排列)
3. **波动率低** (BTC<4%/日，ETH<6%/日)
4. **持仓时间短** (<7 天)

### ❌ 不适合开杠杆

1. **置信度低** (<96%)
2. **震荡市** (方向不明)
3. **波动率高** (BTC>6%/日)
4. **长期持仓** (>14 天)

## 🔒 杠杆风控规则

### 铁律 (必须遵守)

```
✅ 最大杠杆：3 倍
✅ 仅 BTC/ETH 可开杠杆
✅ 山寨币禁止杠杆
✅ 合约禁止开仓
```

### 杠杆止损规则

```
无杠杆止损：-2%
2 倍杠杆止损：-1% (相当于本金 -2%)
3 倍杠杆止损：-0.67% (相当于本金 -2%)

日止损 (杠杆后):
- 无杠杆：-5%
- 2 倍杠杆：-2.5%
- 3 倍杠杆：-1.67%
```

### 杠杆预警线

```
2 倍杠杆:
- 预警线：-15% (距离爆仓还有 35%)
- 强平线：-50%

3 倍杠杆:
- 预警线：-10% (距离爆仓还有 23%)
- 强平线：-33%
```

```

   ├─ 知几-E 置信度>98%

   ├─ 2 倍 (标准)

   └─ 确认还款时间

4. 交易执行
   ├─ 买入 BTC/ETH
   ├─ 设置止损
   └─ 设置预警

5. 持仓监控
   ├─ 实时盈亏
   ├─ 预警线检查
   └─ 利息累计

6. 平仓还款
   ├─ 卖出 BTC/ETH
   ├─ 归还借款
   └─ 支付利息
```

## ⚠️ 风险提示

### 现货杠杆风险

```
1. 利息成本：0.02%/日 (年化 7.3%)
2. 爆仓风险：-50% (2 倍杠杆)
3. 流动性风险：极端行情无法平仓
```

### 如何避免爆仓

```
1. 使用 2-3 倍杠杆 (不用高杠杆)
2. 设置预警线 (-15% 预警)
3. 设置止损 (-1% 止损，2 倍杠杆)
4. 不长期持仓 (<7 天)
5. 不满仓操作 (保留 20% 现金)
```
