## 🎯 核心能力

**女娲.skill** 通过多 Agent 系统蒸馏任何人的思维框架：

- ✅ **6 个并行调研 Agent** - 同时从多个角度研究目标人物
- ✅ **40+ 信息源** - 覆盖访谈/文章/演讲/社交媒体
- ✅ **三重验证** - 确保心智模型准确性
- ✅ **质量验证** - 生成前进行质量校验
- ✅ **SKILL.md 输出** - 标准化思维框架文档

---

## 🧠 六层蒸馏架构

| 层级 | 内容 | 说明 |
|------|------|------|
| **L1 表达 DNA** | 语言风格/口头禅/比喻习惯 | 如何表达思想 |
| **L2 心智模型** | 核心认知框架/思维模式 | 如何理解世界 |
| **L3 决策启发式** | 决策规则/判断标准 | 如何做选择 |
| **L4 反模式** | 避免的陷阱/错误 | 什么不该做 |
| **L5 价值观** | 核心信念/原则 | 什么最重要 |
| **L6 诚实边界** | 承认不知道的领域 | 认知边界 |

## 🔧 使用方法

```bash
# 安装女娲 Skill
npx skills add alchaincyf/nuwa-skill

# 蒸馏新人物
nuwa distill "Elon Musk" --output skills/elon-musk/

# 调用已蒸馏思维
nuwa ask "elon-musk" "如何用第一性原理分析这个问题？"

# 多顾问会议
nuwa meeting "elon-musk,charlie-munger,feynman" --topic "创业决策"
```
