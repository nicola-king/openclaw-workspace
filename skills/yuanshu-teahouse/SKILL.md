## 🚀 使用方式

### Python API

```python
from skills.yuanshu-teahouse.services.booking_service import BookingService

# 初始化
booking = BookingService()

# 查询可用空间
spaces = booking.query_spaces(
    date="2026-04-10",
    time_slot="14:00-17:00",
    party_size=4
)

# 预订
reservation = booking.create_reservation(
    space_id="room-001",
    customer="张三",
    phone="138****1234",
    date="2026-04-10",
    time_slot="14:00-17:00"
)

# 查询订单
orders = booking.get_reservations(customer_phone="138****1234")
```

### CLI 命令

```bash
# 查询可用空间
yuanshu spaces --date 2026-04-10 --time 14:00 --size 4

# 创建预订
yuanshu book --space room-001 --date 2026-04-10 --time 14:00-17:00

# 查询订单
yuanshu orders --phone 138****1234

# 会员查询
yuanshu member --phone 138****1234

# 茶叶产品
yuanshu teas --category sichuan

# 活动报名
yuanshu event join --id event-001
```

## 🎯 心智模型 (蒸馏太一体系)

### 茶文化理解

```
心智模型 1: "茶是媒介，社交是本质"
- 茶馆不仅是喝茶场所
- 核心价值是提供社交空间
- 体验 > 产品

心智模型 2: "场景决定消费"
- 商务会谈 → 包间/会议室
- 朋友小聚 → 散座/露台
- 文化活动 → 活动场地

心智模型 3: "会员即社群"
- 会员不是折扣卡
- 是社群身份认同
- 运营重点是活动而非销售
```

### 决策启发式

```
启发式 1: "空间利用率优先"
- 闲时促销 > 空着
- 活动填充 > 等待散客

启发式 2: "体验第一，销售第二"
- 先让客人体验好
- 销售自然发生

启发式 3: "社群口碑 > 广告投放"
- 老客带新客
- 活动出片传播
```
