## 🧠 智能行情判断

### 市场状态识别

| 状态 | 特征 | 杠杆 | 策略 |
|------|------|------|------|
| **单边上涨** | 均线多头 + 成交量放大 + 涨幅>2% | 2-3 倍 | 追涨 + 持有 |
| **单边下跌** | 均线空头 + 成交量放大 + 跌幅>2% | 1-2 倍 | 观望/做空 |
| **震荡市** | 均线纠缠 + 成交量萎缩 + 波动<2% | 1 倍 | 高抛低吸 |

## 📊 智能判断指标

### 1. 均线排列

```
单边上涨:
MA20 > MA50 > MA200 (多头排列)

单边下跌:
MA20 < MA50 < MA200 (空头排列)

震荡市:
MA20 ≈ MA50 ≈ MA200 (纠缠)
```

### 2. RSI 指标

```
超买：RSI > 70 (警惕回调)
中性：30 < RSI < 70 (正常)
超卖：RSI < 30 (警惕反弹)
```

### 3. 成交量

```
放大：成交量比率 > 1.2 (确认趋势)
正常：0.8 < 比率 < 1.2 (中性)
萎缩：比率 < 0.8 (趋势可能结束)
```

### 4. 波动率 (ATR)

```
高波动：24 小时波动 > 4% (单边市特征)
正常波动：2-4% (过渡)
低波动：<2% (震荡市特征)
```
