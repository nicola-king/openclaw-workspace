### 模块 1: 市场扫描器

```python
# 扫描适合网格的市场
def scan_grid_markets():
    """
    筛选标准:
    1. 流动性好 (日成交>$10K)
    2. 价格区间 0.10-0.90
    3. 波动率适中 (非极端事件)
    4. 到期时间 >7 天
    """
    markets = []
    
    # 调用 Polymarket API
    for market in get_active_markets():
        if market['last_price'] < 0.10 or market['last_price'] > 0.90:
            continue
        if market['volume_24h'] < 10000:
            continue
        if market['expiry_days'] < 7:
            continue
        
        markets.append(market)
    
    return markets[:50]  # 返回前 50 个
```

### 模块 2: 挂单管理器

```python
# 自动挂单
def place_grid_orders(market, budget=50):
    """
    网格挂单:
    - 买单：0.15 × budget
    - 卖单：0.85 × budget
    """
    buy_price = 0.15
    sell_price = 0.85
    buy_amount = budget * buy_price
    sell_amount = budget * (1 - sell_price)
    
    # 挂买单
    order_buy = clob_client.place_order(
        market=market['slug'],
        side='BUY',
        price=buy_price,
        amount=buy_amount
    )
    
    # 挂卖单
    order_sell = clob_client.place_order(
        market=market['slug'],
        side='SELL',
        price=sell_price,
        amount=sell_amount
    )
    
    return {
        'market': market['slug'],
        'buy_order': order_buy,
        'sell_order': order_sell
    }
```

### 模块 3: 盈利追踪器

```python
# 实时追踪盈亏
def track_pnl(orders):
    """
    追踪每个仓位的盈亏
    """
    total_pnl = 0
    realized_pnl = 0
    unrealized_pnl = 0
    
    for order in orders:
        if order['status'] == 'filled':
            # 已成交，计算已实现盈亏
            pnl = (order['sell_price'] - order['buy_price']) * order['amount']
            realized_pnl += pnl
        else:
            # 未成交，计算浮动盈亏
            current_price = get_current_price(order['market'])
            pnl = (current_price - order['price']) * order['amount']
            unrealized_pnl += pnl
    
    total_pnl = realized_pnl + unrealized_pnl
    
    return {
        'total_pnl': total_pnl,
        'realized_pnl': realized_pnl,
        'unrealized_pnl': unrealized_pnl,
        'roi': total_pnl / total_budget
    }
```

## 📈 风险控制

### 风险 1: 单边行情

```
问题：价格一直不回调，只成交一边

解决:
1. 设置最大仓位限制 (单市场<$100)
2. 分散多个市场 (50-200 个)
3. 动态调整挂单价格
```

### 风险 2: 流动性不足

```
问题：挂单无法成交

解决:
1. 只选高流动性市场 (日成交>$10K)
2. 挂单价格贴近市场价 (±10%)
3. 定期取消未成交订单，重新挂单
```

### 风险 3: 黑天鹅事件

```
问题：极端事件导致价格归零或归一

解决:
1. 避开政治/选举类高风险市场
2. 专注体育/娱乐/加密货币类
3. 单仓位上限$50
```
