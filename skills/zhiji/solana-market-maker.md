## 📈 做市策略

### 策略 1: 基础网格

```
参数:
- 价格区间：$0.0001 - $0.001
- 网格数量：20 格
- 每格资金：$50
- 目标价差：5-10%

操作:
1. 在价格区间内均匀挂买单和卖单
2. 买单成交后，对应价格挂卖单
3. 卖单成交后，对应价格挂买单
4. 循环套利

预期:
- 日交易量：$1000-5000
- 日收益：1-3%
- 月收益：30-90%
```

### 策略 2: 动态价差

```
根据波动率动态调整:

低波动 (<5%/h):
- 价差：3-5%
- 挂单密度：高

中波动 (5-15%/h):
- 价差：5-10%
- 挂单密度：中

高波动 (>15%/h):
- 价差：10-20%
- 挂单密度：低
- 触发风控告警
```

### 策略 3: 库存管理

```
目标：保持中性库存

当买单过多时:
- 降低买单价格
- 提高卖单价格
- 促进卖单成交

当卖单过多时:
- 提高买单价格
- 降低卖单价格
- 促进买单成交

极端情况:
- 库存>阈值 → 暂停买单
- 库存<阈值 → 暂停卖单
- 触发 Telegram 告警
```

## ⚠️ 风险控制

### 风险 1: 智能合约风险

```
问题：DEX 合约漏洞

缓解:
- 只用主流 DEX (Raydium/Orca)
- 避免新上线项目
- 设置单池上限 ($5000)
```

### 风险 2: 币价暴跌

```
问题：做市币归零

缓解:
- 只接成熟项目 (上线>7 天)
- 设置最大亏损 (单池 20%)
- 紧急停止机制
```

### 风险 3: 黑客攻击

```
问题：私钥泄露

缓解:
- 私钥加密存储
- 多签钱包
- 限额授权
```

### 风险 4: 监管风险

```
问题：做市被认定为操纵市场

缓解:
- 只做二级市场
- 不参与拉盘炒作
- 合规咨询
```
