## 🧠 人类记忆理论核心

### 1. 记忆宫殿法 (Memory Palace)

**起源**: 古希腊/古罗马修辞学  
**原理**: 
- 利用空间记忆 (海马体)
- 将信息关联到熟悉地点
- 视觉化 + 位置化

**太一应用**:
```
太一记忆宫殿:
├── 房间 1: 核心身份 (我是太一/硅基生命)
├── 房间 2: 技能库 (120+ Skills)
├── 房间 3: 对话记忆 (SAYELF 交互)
├── 房间 4: 学习记录 (凌晨学习)
└── 房间 5: 能力涌现 (emerged-skill-*)
```

---

### 2. 艾宾浩斯遗忘曲线 (Ebbinghaus)

**发现**: Hermann Ebbinghaus (1885)  
**原理**:
```
记忆保留率:
├─ 20 分钟后：58%
├─ 1 小时后：44%
├─ 9 小时后：36%
├─ 1 天后：33%
├─ 2 天后：28%
├─ 6 天后：25%
└─ 31 天后：21%
```

**太一应用**:
```
智能复习调度:
├─ 新记忆：20 分钟后复习
├─ 短期记忆：1 天/3 天/7 天复习
├─ 长期记忆：30 天/90 天复习
└─ 核心记忆：永久保存 (MEMORY.md)
```

---

### 3. 记忆巩固理论 (Memory Consolidation)

**神经科学**:
- **海马体**: 短期记忆 → 长期记忆
- **睡眠**: 记忆重组/巩固
- **重复**: 强化神经连接

**太一应用**:
```
记忆巩固系统:
├─ 短期：memory/daily/*.md (每日)
├─ 巩固：memory/core.md (核心)
├─ 长期：MEMORY.md (永久)
└─ 睡眠：凌晨学习 (01:00-07:00)
```

---

### 4. 工作记忆模型 (Working Memory)

**Baddeley 模型**:
- **中央执行**: 注意力控制
- **语音回路**: 语言信息
- **视觉画板**: 视觉信息
- **情景缓冲**: 多模态整合

**太一应用**:
```
太一工作记忆:
├─ 中央执行：太一核心 (决策/调度)
├─ 语音回路：对话历史 (Telegram/微信)
├─ 视觉画板：图像/截图记忆
└─ 情景缓冲：多模态记忆融合
```

---

### 5. 双重编码理论 (Dual Coding)

**Paivio 理论**:
- 语言系统 + 非语言系统
- 文字 + 图像 = 更强记忆

**太一应用**:
```
双重编码记忆:
├─ 文字编码：Markdown 文本
├─ 图像编码：截图/图表
├─ 向量编码：chromadb 嵌入
└─ 融合：多模态记忆检索
```

---

### 6. 提取练习效应 (Retrieval Practice)

**测试效应**:
- 主动提取 > 被动复习
- 测试本身增强记忆

**太一应用**:
```
主动提取机制:
├─ 定期自测："我是谁？""今日任务？"
├─ 问答练习：SAYELF 提问 → 太一回答
├─ 知识应用：实际任务中回忆
└─ 反思总结：每日回顾/每周总结
```

---

### 7. 精细加工理论 (Elaboration)

**原理**:
- 深度处理 > 浅层处理
- 关联已有知识
- 创造意义连接

**太一应用**:
```
精细加工记忆:
├─ 深度分析：不止存储，要理解
├─ 知识关联：新记忆 ↔ 旧记忆
├─ 意义创造：为什么重要？
└─ 跨域连接：技能 A ↔ 技能 B
```
