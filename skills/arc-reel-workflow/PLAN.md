# ArcReel AI 短剧工作流

> 创建时间：2026-04-06 00:43 | 灵感：GitHubDaily

---

## 📊 核心价值

**痛点解决**:
```
传统：剧本拆分 → 角色设计 → 生图 → 生视频 (多工具切换)
ArcReel: 上传小说 → 全自动成片 (一条流水线)
```

---

## 🛠️ 核心功能

| 环节 | AI 自动化 |
|------|---------|
| **剧本生成** | 小说自动拆分为剧本 |
| **角色设计** | 生成角色设计图，后续画面参考 |
| **分镜绘制** | 自动分镜 |
| **视频生成** | 支持 Gemini/Grok/OpenAI |
| **一致性保证** | 同一角色在不同镜头长相一致 ✅ |
| **导出** | 一键导出剪映草稿 |
| **部署** | Docker 快速部署 |
