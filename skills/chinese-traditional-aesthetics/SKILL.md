### 服饰元素

```
汉服特征:

│  系带隐扣：以带系结，少用纽扣          │

```

## 🏞️ 中国古典园林

### 四大名园

| 园林 | 地点 | 朝代 | 特征 |
|------|------|------|------|
| **拙政园** | 苏州 | 明 | 疏朗典雅，以水为主 |
| **留园** | 苏州 | 清 | 建筑精美，空间变化 |
| **颐和园** | 北京 | 清 | 皇家气派，山水相依 |
| **承德避暑山庄** | 承德 | 清 | 自然山水，融汇南北 |

### 园林美学原则

```
造园手法:
┌─────────────────────────────────────────┐
│  借景：借园外之景入园内                │
│  对景：两处景点相互呼应                │
│  障景：用景物遮挡，增加层次            │
│  框景：用门窗框取景色                  │
│  漏景：用花窗漏取景色                  │
│  移步换景：行走中景色变化              │
└─────────────────────────────────────────┘

空间美学:
- 曲径通幽：路径曲折，引人入胜
- 小中见大：小空间营造大意境
- 虚实相生：实景与留白结合
- 山水相依：山石与水体呼应
```

### 太一融合应用

```
园林美学 → 太一交互设计
     ↓
借景 → 内容引用与整合
对景 → 页面呼应与平衡
障景 → 信息层级与隐藏
框景 → 内容聚焦与突出
漏景 → 信息透露与悬念
移步换景 → 滚动视差与动效
```

## 🏛️ 中国传统建筑

### 建筑特征

| 特征 | 说明 | 美学价值 |
|------|------|---------|
| **木结构** | 榫卯结构，不用钉子 | 柔韧抗震，自然和谐 |
| **斗拱** | 层层出挑，支撑屋檐 | 结构美与装饰美统一 |
| **大屋顶** | 飞檐翘角，曲线优美 | 轻盈飘逸，如鸟斯革 |
| **中轴对称** | 左右对称，主次分明 | 庄重典雅，秩序井然 |
| **院落组合** | 多重院落，层层递进 | 空间层次，私密性强 |

### 建筑等级

```
屋顶等级 (从高到低):
┌─────────────────────────────────────────┐
│  庑殿顶 → 歇山顶 → 悬山顶 → 硬山顶     │
│  (皇家)    (官府)    (民居)    (民居)   │
└─────────────────────────────────────────┘

色彩等级:
- 黄色：皇家专用
- 红色：官府/寺庙
- 青/绿：民居/园林
- 白色：民居/陵墓
```

### 太一融合应用

```
建筑美学 → 太一系统架构
     ↓
木结构榫卯 → 模块化组件
斗拱层层出挑 → 层级架构
大屋顶飞檐 → 视觉识别
中轴对称 → 布局平衡
院落组合 → 空间分区
```
