## 🎯 核心能力

**Dreaming** 是 OpenClaw 的后台记忆整理系统，模拟人类 REM 睡眠：

- ✅ **三阶段处理** - Light → REM → Deep
- ✅ **自动输出** - DREAMS.md + 自动更新 MEMORY.md
- ✅ **实验性功能** - 可选开启
- ✅ **可解释性** - 梦境记录可审查
- ✅ **后台运行** - 不干扰日常工作

---

## 🧠 三阶段记忆整理

```
┌─────────────────────────────────────────────────────────────┐
│                    Dreaming 三阶段                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  【Light 整理】                                              │
│  - 收集当日对话/任务/决策                                   │
│  - 初步分类和标签化                                         │
│  - 输出：DREAMS.md (原始梦境日记)                           │
│                                                             │
│  ↓                                                          │
│                                                             │
│  【REM 反思】                                                │
│  - 关联已有记忆                                             │
│  - 提取模式和洞察                                           │
│  - 识别重要事件                                             │
│                                                             │
│  ↓                                                          │
│                                                             │
│  【Deep 提炼】                                               │
│  - 提炼长期记忆                                             │
│  - 更新 MEMORY.md                                           │
│  - 压缩/归档旧记忆                                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

---

## 📄 输出文件

### DREAMS.md（梦境日记）

```markdown
# DREAMS - 2026-04-08 03:00

## Light 整理
- 对话 1: ...
- 任务 2: ...
- 决策 3: ...

## REM 反思
- 模式识别：...
- 关联记忆：...
- 重要事件：...

## Deep 提炼
- 长期记忆更新：...
- MEMORY.md 变更：...
```

### MEMORY.md（自动更新）

Dreaming 自动将重要事件提炼到长期记忆，无需手动维护。

## 💡 太一集成场景

### 场景 1：自动记忆维护

```
每日对话 → Dreaming(凌晨 3 点) → 自动整理 → MEMORY.md 更新
```

**价值**：
- ✅ 无需手动维护记忆
- ✅ 避免记忆碎片化
- ✅ 自动提炼长期价值内容

### 场景 2：记忆关联增强

```
新记忆 → Dreaming → 关联已有记忆 → 知识网络增强
```

### 场景 3：模式识别

```
多日对话 → Dreaming → 模式识别 → 行为习惯洞察
```

### 场景 4：记忆压缩

```
大量对话 → Dreaming → 压缩提炼 → 节省 Token
```
