## 🎯 涌现来源

**技能组合**:
```
知几 (量化交易) + 罔两 (项目研究) + PolyAlert (机会监控)
```

**涌现功能**:
```
单一技能无法实现的能力:
❌ 知几：只能交易现有市场，无法发现新空投
❌ 罔两：只能研究项目，无法自动化操作
❌ PolyAlert: 只能监控价格，无法识别空投

猎手涌现:
✅ 新协议自动发现
✅ 空投资格自动检测
✅ 多账号批量操作
✅ 成本/收益自动计算
✅ 最佳策略自动选择
```

## 🔧 核心功能

### 1. 新协议自动发现

**功能描述**:
- 监控 DefiLlama 新协议
- 追踪 Twitter 项目公告
- 扫描 GitHub 新项目
- 识别有空投潜力的协议

**发现渠道**:
| 渠道 | 说明 | 权重 |
|------|------|------|
| DefiLlama | 新协议上线 | ⭐⭐⭐⭐⭐ |
| Twitter | 项目官方公告 | ⭐⭐⭐⭐⭐ |
| GitHub | 代码库活动 | ⭐⭐⭐⭐ |
| Discord | 社区公告 | ⭐⭐⭐⭐ |
| Medium | 项目博客 | ⭐⭐⭐ |

**输出示例**:
```

✅ 未发币

成本：$50 (Gas+ 本金)
预期收益：$500-5000
```

### 2. 空投资格自动检测

**功能描述**:

| 每周交互 | 5 分 | 每周至少 1 次 |

| 存款>$100 | 10 分 | 中等存款 |
| 存款>$10 | 5 分 | 小额存款 |
| 推荐好友 | 10 分 | 每个好友 |
| Discord 活跃 | 5 分 | 社区参与 |
| Twitter 关注 | 3 分 | 社交媒体 |

**输出示例**:

### 3. 多账号批量操作

**功能描述**:
- 管理多个钱包地址
- 批量执行交互操作
- 避免关联检测
- 最大化空投资格

**账号管理**:
| 账号 | 地址 | 状态 | 分数 |
|------|------|------|------|
| 主账号 | 0x678c1... | ✅ 活跃 | 85/100 |
| 副账号 1 | 0x8ba1f... | ✅ 活跃 | 72/100 |
| 副账号 2 | 0x742d3... | ✅ 活跃 | 68/100 |
| 副账号 3 | 0x9cd45... | 🟡 待交互 | 35/100 |

**防关联策略**:
```
1. 不同 IP 地址
   - 主账号：家庭 IP
   - 副账号：代理 IP

2. 不同时间操作
   - 间隔>1 小时
   - 避免同时交易

3. 不同金额
   - 主账号：$500-1000
   - 副账号：$50-200

4. 不同行为模式
   - 主账号：每周交互
   - 副账号：随机交互
```

**输出示例**:
```
🎯 多账号批量操作

项目：OpenLedger
执行时间：2026-03-27 10:00

账号 1 (主):
- 操作：存款$500
- Gas 预估：$5
- 状态：待执行

账号 2 (副 1):
- 操作：存款$100
- Gas 预估：$5
- 状态：待执行

账号 3 (副 2):
- 操作：借款$50
- Gas 预估：$8
- 状态：待执行

总成本：$68 (Gas+ 本金)
预期收益：$1500-6000 (3 个账号)
ROI: 22x-88x
```

## 📖 使用方法

### 启动猎手

```bash
cd ~/.openclaw/workspace/skills
python3 -m hunter.main
```

```bash
python3 -c "
from hunter.discover import AirdropDiscovery
discovery = AirdropDiscovery()
discovery.find_opportunities()
"
```
