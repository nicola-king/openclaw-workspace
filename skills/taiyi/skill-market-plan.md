## 技术实现

### 技能标准化

每个技能包含：
```
skill-name/
├── SKILL.md          # 技能说明（用户看）
├── README.md         # 使用教程
├── config.example    # 配置模板
├── scripts/          # 执行脚本
└── requirements.txt  # 依赖
```

### 安装流程

```bash
# 用户安装
npx clawhub install taiyi-weather-pro

# 技能自动：
# 1. 下载技能文件
# 2. 生成配置模板
# 3. 提示用户填写 API Key
# 4. 配置定时任务
# 5. 启动服务
```

### 付费验证

```python
# 技能启动时验证
def verify_subscription(api_key):
    response = requests.post(
        "https://taiyi.sayelf.com/api/verify",
        json={"api_key": api_key}
    )
    return response.json()["active"]
```
