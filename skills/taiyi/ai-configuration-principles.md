- 明确使用场景

**输出**: 核心配置清单

---

### Step 2: 现有配置

- ✅ Gemini API (文档/分析)
- ✅ 本地 qwen2.5:7b (简单/离线)
- 🟡 Claude (按需)
- 🟡 GPT (按需)

**输出**: 现有配置状态表

---

## 🤖 太一 AI 调度策略

### 智能路由逻辑

```python
def select_model(task):
    """智能选择模型"""
    
    if task.type == 'code' or task.type == 'technical':
        return 'bailian-coding'  # 百炼 Coding Plan
    
    elif task.type == 'document' or task.type == 'analysis':
        return 'gemini'  # Gemini 免费额度
    
    elif task.type == 'simple' or task.offline:
        return 'qwen2.5-7b-local'  # 本地
    
    elif task.complexity == 'high':
        return 'claude-or-gpt'  # 付费高端
    
    else:
        return 'gemini'  # 默认 Gemini
```

### 成本优化策略
