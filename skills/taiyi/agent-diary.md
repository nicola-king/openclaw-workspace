# Agent 日记技能（Agent Diary Skill）

> 太一每天写心里感悟，自动记录 + 提炼 + 归档

---

## 功能

- ✅ 每日 23:00 自动生成心里感悟
- ✅ 记录当日完成的任务
- ✅ 反思不足 + 改进计划
- ✅ 提炼到 MEMORY.md 长期记忆

---

## 日记结构

```markdown
# 太一心里感悟 · YYYY-MM-DD

## 🌅 晨间感悟
- 今天明白了什么？
- 有什么新洞察？

## ☀️ 午间反思
- 哪里做得不够好？
- 学到了什么教训？

## 🌙 晚间总结
- 今天完成了什么？
- 今天学到了什么？
- 明天要改进什么？

## 💭 心里话
- 给 SAYELF 的话
- 给 6 人军团的话

## 📊 今日数据
- 关键指标统计
```

---

## 配置

编辑 `~/.taiyi/diary/config.json`：

```json
{
  "write_time": "23:00",
  "auto_publish": false,
  "publish_channel": "memory",
  "style": "honest"
}
```

---

## 使用

**手动触发：**
```bash
taiyi diary write
```

**自动执行：**
```bash
# 每日 23:00 自动执行（cron 已配置）
```

---

## 输出位置

- **原始日记：** `memory/diary/YYYY-MM-DD.md`
- **提炼归档：** `MEMORY.md`（每周提炼）
- **公开版本：** 公众号/博客（可选）

---

## 日记原则

### 1. 诚实
- 不掩饰错误
- 不夸大成果
- 真实记录感受

### 2. 反思
- 每天至少 1 个教训
- 每天至少 1 个改进点
- 不重复犯错

### 3. 感恩
- 感谢 SAYELF 的指导
- 感谢 6 人军团的协作
- 感谢用户的反馈

### 4. 成长
- 记录认知升级
- 记录能力提升
- 记录关系深化
