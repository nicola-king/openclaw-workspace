## 🎯 角色精神内核

### 猎手·羿的射箭哲学

```
射箭三境界:
1. 见山是山 (新手：瞄准目标)
2. 见山不是山 (老手：心无杂念)
3. 见山还是山 (高手：箭即是心)

猎手 Bot 三境界:
1. 见信号就射 (新手：频繁交易)
2. 见信号不射 (老手：等待时机)
3. 应时而射 (高手：精准出击)
```

### 管家·守藏吏的治国哲学

```
治大国若烹小鲜:
- 不折腾 (不频繁重启)
- 不扰民 (不占用资源)
- 自然有序 (定时任务自动执行)

管家 Bot 三原则:
1. 系统稳定 (不折腾)
2. 资源节约 (不扰民)
3. 自动有序 (自然运行)
```
