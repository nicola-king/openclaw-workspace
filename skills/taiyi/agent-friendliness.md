## 5 维度评估

| 维度 | 权重 | 评分标准 |
|------|------|---------|
| **JSON 输出** | 20% | 是否支持--json 输出 |
| **Schema 自省** | 20% | 是否支持--help/--schema |
| **Dry Run** | 20% | 是否支持--dry-run 预演 |
| **错误处理** | 20% | 错误信息是否清晰 |
| **认证配置** | 20% | 认证配置是否简单 |

## CLI 命令

```bash
# 查看技能评分
openclaw skill score <skill-name>

# 列出所有技能评分
openclaw skill list --score

# 导出评分报告
openclaw skill export-score --format markdown
```
