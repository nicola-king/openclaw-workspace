# 太一轮桌会议 Skill

> 版本：v1.0 | 创建：2026-03-28 20:02
> 主持人：太一
> 参会 Bot: 知几/山木/素问/罔两/庖丁/猎手/管家

---

## 触发条件

满足以下任一条件时启动圆桌会议:
- 重大决策 (预算>¥1000)
- 跨域问题 (涉及≥3 个 Bot 职责)
- 用户明确要求
- 太一判断需要集体智慧

---

## 讨论流程

### Round 1: 问题剖析 (各 Bot 发表观点)
### Round 2: 方案讨论 (交叉质疑/补充)
### Round 3: 决策建议 (太一汇总)

---

**知几**: [交易角度分析]

**素问**: [技术角度分析]

## Round 3: 决策建议

**太一汇总**:

2. 分歧点：...
3. 建议方案：...

**投票结果**:
- 同意：X 票
- 反对：Y 票
- 弃权：Z 票

*会议记录：太一 | 归档：memory/2026-03-28.md*

---

## 使用方式

```bash
# 启动圆桌会议
openclaw roundtable start --topic "议题名称"

# 查看状态
openclaw roundtable status

# 导出纪要
openclaw roundtable export --format markdown
```
