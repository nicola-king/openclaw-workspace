## 🔧 核心功能

### 1. 聪明钱行为分析

**功能描述**:
- 追踪历史胜率>60% 的钱包
- 分析其持仓变化
- 识别建仓/平仓模式
- 生成跟单信号

**数据来源**:
- Polymarket Analytics API
- Guru 钱包分析
- Betmoar 持有者数据

**输出示例**:
```
🧠 聪明钱动向

钱包：0x742d35Cc6634C0532925a3b844Bc454e4438f44e
历史胜率：68.5%
总盈利：$125,430

最新操作:
- 买入 $5,000 YES @ 45¢ (市场：BTC $100K 2026)
- 买入 $3,000 YES @ 38¢ (市场：ETH $5K 2026)
- 卖出 $8,000 NO @ 72¢ (市场：Trump 2028)

信号：强烈看涨加密市场
置信度：85%
```

---

### 2. 内幕交易检测

**功能描述**:
- 监控新钱包大额买入
- 识别异常交易模式
- 关联多个钱包行为
- 预警潜在内幕交易

**检测规则**:
| 规则 | 阈值 | 说明 |
|------|------|------|
| 新钱包大额 | >$10K | 创建<7 天，单笔>$10K |
| 集中买入 | >5 钱包 | 5 个 + 钱包同时买入同一方向 |
| 时间异常 | 新闻前 5 分钟 | 重大新闻前 5 分钟内大额交易 |
| 胜率异常 | >80% | 新钱包连续 10 笔胜率>80% |

**输出示例**:
```
⚠️ 内幕交易预警

市场：Will Trump announce 2028 campaign?
方向：YES
异常类型：集中买入

检测到 7 个新钱包同时买入:
- 钱包 1: $15,000 @ 52¢ (创建 3 天)
- 钱包 2: $12,000 @ 53¢ (创建 5 天)
- 钱包 3: $18,000 @ 54¢ (创建 2 天)
...

总买入：$87,000
时间窗口：3 分钟内
置信度：92%

建议：跟随 YES 或避免交易
```

---

### 3. 市场情绪预测

**功能描述**:
- 分析多空双方持仓比
- 计算聪明钱倾向
- 监测资金流入流出
- 预测短期走势

**情绪指标**:
| 指标 | 计算方式 | 说明 |
|------|---------|------|
| 多空比 | YES 持仓/NO 持仓 | >1 看涨，<1 看跌 |
| 聪明钱倾向 | 高胜率钱包方向 | 跟随聪明钱 |
| 资金流 | 24h 净流入 | 正=流入，负=流出 |
| 热度 | 成交量/持仓量 | 高=关注度高 |

**输出示例**:
```
📊 市场情绪报告

市场：BTC $100K by end of 2026

多空比：1.8 (看涨)
聪明钱：68% 持仓 YES
资金流：+$245K (24h)
热度：高 (成交量$1.2M)

综合情绪：强烈看涨 📈
短期预测：概率可能上升至 55-60%
```

---

### 4. 套利机会发现

**功能描述**:
- 跨市场套利 (Polymarket vs Kalshi)
- 跨时间套利 (早期 vs 尾盘)
- 确定性套利 (结算收割)
- 概率偏差套利

**机会类型**:
| 类型 | 说明 | 风险 |
|------|------|------|
| 跨市场 | 同一事件不同平台价格差 | 低 |
| 跨时间 | 早期高赔率 vs 尾盘确定性 | 中 |
| 结算收割 | UMA 结算前低价买入 | 低 |
| 概率偏差 | 市场过度反应 | 中 |

**输出示例**:
```
💰 套利机会

类型：跨市场套利
事件：Fed rate cut March 2026

Polymarket: YES 72¢ (概率 72%)
Kalshi:     YES 78¢ (概率 78%)

套利策略:
- Polymarket 买入 YES $1000 @ 72¢
- Kalshi 买入 NO $1000 @ 22¢

无论结果如何:
- 收入：$1000
- 成本：$940
- 利润：$60 (6.4%)

风险：低 (平台风险)
时效：5 分钟内有效
```

---

### 5. 鲸鱼动向追踪

**功能描述**:
- 实时监控>$100K 持仓钱包
- 追踪买卖操作
- 分析持仓变化
- 预警大额交易

**鲸鱼分级**:
| 级别 | 持仓 | 说明 |
|------|------|------|
| 🐋 巨鲸 | >$1M | 市场操纵者 |
| 🐬 大鲸 | $500K-$1M | 专业交易者 |
| 🐟 小鲸 | $100K-$500K | 高净值个人 |

**输出示例**:
```
🐋 鲸鱼动向

巨鲸：0x8ba1f109551bD432803012645H...
持仓：$2.4M
历史胜率：72%

最新操作:
- 买入 $50,000 YES @ 48¢ (ETH $5K 2026)
- 卖出 $30,000 NO @ 65¢ (Trump 2028)
- 持仓变化：+8% ETH, -5% Trump

信号：强烈看涨 ETH
置信度：88%
```
