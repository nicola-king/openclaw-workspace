```

```

- 不因为"时间急"而跳过回测

## 🚩 红旗列表 - STOP and Start Over

**出现以下任何一条，立即停止：**

- [ ] "这个信号很明显，不用回测"
- [ ] "时间来不及，先下注"
- [ ] "感觉会涨/跌"
- [ ] "应该是这样"
- [ ] "之前一直有效"
- [ ] "聪明钱也在买"

**所有这些都意味着：你在 rationalize。**

---

## 📋 合理化表格

| 借口 | 现实 |
|------|------|
| "信号明显" | 明显信号也会失效 |
| "时间来不及" | 越急越要回测，错了更亏钱 |
| "感觉会涨" | 感觉不是策略 |
| "应该是这样" | "应该"= 不确定 = 验证 |
| "之前一直有效" | 过去不代表未来 |
| "聪明钱也在买" | 聪明钱也会错 |
