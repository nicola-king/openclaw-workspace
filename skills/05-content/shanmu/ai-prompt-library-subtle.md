## 🎨 隐性元素植入指南 (后期处理)

### 1. 文字水印 (1%-5% 透明度)

**Photoshop 操作**:
```
1. 新建文字图层
2. 输入：单字或 2-4 字 (如"道"/"无为"/"自在")
3. 字体：宋体/楷体 (细体)
4. 大小：10-12px (1080x1920 壁纸)
5. 位置：右下角或左下角
6. 透明度：5%-8%
7. 混合模式：正片叠底/叠加
```

**Canva 操作**:
```
1. 添加文字
2. 透明度滑块调至 5%
3. 位置：角落
4. 大小：最小号
```

---

### 2. 符号隐藏 (2%-3% 可见度)

**八卦符号植入**:
```
位置：画面边缘或背景纹理中
大小：8x8px - 12x12px
透明度：2%-3%
混合模式：叠加/柔光
```

**示例位置**:
- 云朵纹理中隐藏"☰"
- 水波纹中隐藏"☵"
- 山峦轮廓形成"☶"
- 麦田线条形成"☷"

## ⚠️ 注意事项

### 避免踩坑

| ❌ 错误做法 | ✅ 正确做法 |
|-----------|-----------|
| 文字占画面 10%+ | 文字占画面 1%-3% |
| 透明度 80%+ | 透明度 5%-8% |
| 中央位置 | 角落/边缘位置 |
| 粗体大字 | 细体小字 |
| 明显符号 | 隐藏纹理 |
| 直接说教 | 隐性暗示 |

### 平台审核安全

```
✅ 安全：
- 文化/美学角度
- 隐性元素 (<5%)
- 无宗教推广嫌疑

❌ 避免：
- 明显宗教符号 (>10%)
- 大段经文
- 迷信/算命用语
```
