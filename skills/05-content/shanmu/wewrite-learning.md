## 📋 选题评分系统 (新增)

### 评分维度

```python
def score_topic(topic):
    """
    选题综合评分 (0-100)
    """
    # 相关性评分 (30%)
    relevance = calc_relevance(topic, taiyi_business)
    
    # 热度评分 (25%)
    hotness = calc_hotness(topic, search_volume, discussion_count)
    
    # 竞争评分 (20%)
    competition = calc_competition(topic, content_saturation)
    
    # 变现评分 (25%)
    monetization = calc_monetization(topic, product_relevance)
    
    # 综合评分
    total = (
        relevance * 0.30 +
        hotness * 0.25 +
        (100 - competition) * 0.20 +  # 竞争越低分越高
        monetization * 0.25
    )
    
    return total

# 阈值
if total >= 80:
    return "强烈推荐，立即写"
elif total >= 60:
    return "推荐，可以写"
elif total >= 40:
    return "观望，看情况"
else:
    return "不推荐，跳过"
```

### 学习机制

```
用户修改 → Playbook 记录 → 下次应用 → 越用越像你

示例:
第 1 篇：AI 生成标题"震惊！Polymarket 套利月入$10000"
用户修改："Polymarket 套利实战：月入$10000 教程"
Playbook 记录：避免"震惊"等标题党词汇，偏好"实战/教程"
第 2 篇：AI 生成标题"Polymarket 做市指南：3 天赚 30%"
用户满意 → 强化该偏好
```
