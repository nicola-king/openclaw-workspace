## 📊 宝玉 Skill 功能

### 核心功能
| 功能 | 说明 |
|------|------|
| **文章配图** | 自动分析文章内容，生成配图方案 |
| **设计方案** | 配色/尺寸/风格提示词 |
| **生图集成** | MJ/Nano Banana 一键生图 |
| **多平台** | 小红书/公众号/微博 |

### 操作流程
```
1. Claude Code 安装 Skill
   ↓
2. 输入安装命令
   ↓
3. 给文章配图 → 生成设计方案
   ↓
4. 提示词放到 MJ/Nano Banana
   ↓
5. 一键直出
```

### 输出内容
- 🎨 配色方案
- 📐 尺寸规格
- 🖼️ 风格提示词
- 💡 设计建议
