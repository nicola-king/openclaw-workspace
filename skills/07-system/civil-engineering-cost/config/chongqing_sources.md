## 📊 费用组成标准 (重庆地区)

### 建筑安装工程费用项目组成

```
总造价 = 分部分项工程费 + 措施项目费 + 其他项目费 + 规费 + 税金
```

### 措施项目费

| 项目 | 计费基础 | 费率 | 说明 |
|------|---------|------|------|
| **安全文明施工费** | 分部分项费 | 2.0%-3.0% | 市政工程 |
| 夜间施工费 | 分部分项费 | 0.5%-1.0% | 按实计算 |
| 二次搬运费 | 分部分项费 | 0.3%-0.8% | 按实计算 |
| 冬雨季施工费 | 分部分项费 | 0.5%-1.0% | 按实计算 |
| 已完工程保护费 | 分部分项费 | 0.2%-0.5% | 按实计算 |

### 规费

| 项目 | 计费基础 | 费率 | 说明 |
|------|---------|------|------|
| 工程排污费 | - | 按实计算 | 环保部门核定 |
| 社会保险费 | 人工费 | 20% | 养老/医疗/失业/工伤/生育 |
| 住房公积金 | 人工费 | 8% | - |

### 税金

| 项目 | 计费基础 | 税率 | 说明 |
|------|---------|------|------|
| 增值税 | 税前造价 | 9% | 建筑行业 |

## 📝 使用说明

### 查询材料价格
```bash
python3 material_prices.py --action get -m "材料名称"
```

### 导入造价站数据
```bash
python3 material_prices.py --action import -f 重庆造价站_2026 年 3 月.csv
```

### 更新费用标准
```python
# 在 cost.py 中修改
self.measure_rate = 0.025  # 措施费 2.5%
self.regulation_rate = 0.28  # 规费 28%(人工费)
self.tax_rate = 0.09  # 税金 9%
```
