## 🔄 6 步演进流程 (每情景通用)

```
Step 1: 刚开始不对劲 / 努力没有回报 / 准备已完成 / 反复无效尝试
        ↓
Step 2: 逐渐察觉问题 / 加大投入无效 / 结果未出现 / 怀疑方法
        ↓
Step 3: 开始怀疑路径 / 产生焦虑 / 开始焦虑比较 / 意识理解偏差
        ↓
Step 4: 尝试调整方式 / 意识方向问题 / 理解环境未匹配 / 寻找新认知
        ↓
Step 5: 逐步适应 / 尝试切换 / 选择等待 / 重建逻辑
        ↓
Step 6: 进入新状态 / 找到新路径 / 机会出现 / 获得新解法
```
