## 🎨 STEP 3: UI 视觉体系

### 🧠 原则

```
❌ 不是好看
✅ 是"有气质 + 可截图"
```

---

### 🎯 视觉方向 (直接选一个)

#### ① 极简高级 (最推荐) ✅

```
白底 + 黑字 + 灰线

类似: Notion / 微信读书

风格参考:
┌──────────────────
│ 第 17 情境 跟随期
│
│ 你正在参考他人的路径，
│ 但缺少自己的判断
│
│ 🏷 路径学习型人格
│
│ #随机视角 #今日状态
└──────────────────
```

---

#### ② 情绪卡片风 (适合传播)

```
浅米色背景 + 小标题 + 留白

背景：#F5F5F0
文字：#333333
 accent: #999999
```

---

#### ③ 神秘感 (少用) ⚠️

```
深色 + 渐变 + 光效

⚠️ 容易被判玄学
不推荐
```

---

### 🧩 卡片设计 (直接照做)

```
尺寸：1080×1920 (竖屏)
边距：40px
行距：1.8

字体:
- 中文：系统默认 (最安全)
- 英文：Inter

配色:
- 背景：#FAFAFA
- 文字：#111111
- 辅助：#888888
- 强调：#000000
```
