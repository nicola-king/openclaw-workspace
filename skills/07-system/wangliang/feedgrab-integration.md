## 💡 整合方案

### 方案 A: 直接调用（推荐）
```bash
# 罔两热点搜索
feedgrab twitter --keyword "Polymarket" --output markdown
```

### 方案 B: Python 集成
```python
import feedgrab
result = feedgrab.grab("https://twitter.com/...")
```

### 方案 C: MCP 接入
```json
// MCP 配置
{
  "mcpServers": {
    "feedgrab": {
      "command": "feedgrab-mcp"
    }
  }
}
```
