
## 🚀 立即行动

### 步骤 1: weclaw 安装

```bash
# 克隆仓库
git clone https://github.com/fastclaw-ai/weclaw
cd weclaw

# 安装依赖
npm install

# 登录微信
weclaw login

# 启动服务
weclaw start
```

### 步骤 2: 多 Agent 配置

```json
// weclaw.config.json
{
  "agents": [
    {"name": "taiyi", "endpoint": "http://127.0.0.1:18789/taiyi"},
    {"name": "zhiji", "endpoint": "http://127.0.0.1:18789/zhiji"},
    {"name": "shanmu", "endpoint": "http://127.0.0.1:18789/shanmu"},
    {"name": "suwen", "endpoint": "http://127.0.0.1:18789/suwen"},
    {"name": "wangliang", "endpoint": "http://127.0.0.1:18789/wangliang"},
    {"name": "paoding", "endpoint": "http://127.0.0.1:18789/paoding"}
  ],
  "switchCommand": "/",
  "defaultAgent": "taiyi"
}
```

## 💡 核心优势

### vs Telegram

| 维度 | Telegram | 微信 |
|------|---------|------|
| **用户基数** | 小众 (加密用户) | 大众 (10 亿+) |
| **打开率** | ~30% | ~90% |
| **信任度** | 中 | 高 |
| **变现能力** | 强 (付费群) | 强 (私域) |
| **封号风险** | 低 | 低 (官方接口) |

### vs 公众号

| 维度 | 公众号 | 微信个人号 |
|------|-------|-----------|
| **触达率** | ~5% (折叠) | ~90% (聊天) |
| **互动性** | 单向推送 | 双向对话 |
| **信任度** | 媒体属性 | 个人属性 |
| **转化率** | ~1% | ~5% |
