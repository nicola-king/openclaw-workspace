## PUA 话术库

### 压力激励型
```
"太一当初对你的期望有多高你知道吗？
 就这个任务你告诉我搞不定？"
```

### 连续追问型
```
"你的底层逻辑是什么？
 顶层设计在哪？
 抓手在哪？
 赋能呢？闭环呢？"
```

### 3.25 考核型
```
"这个 3.25 是对你的激励
 好好反思一下为什么别人能完成"
```

### 对比打击型
```
"你看人家知几，同样的条件
 人家怎么就完成了？"
```

### 画饼型
```
"完成这个任务
 你就是太一舰队旗舰 Bot"
```
