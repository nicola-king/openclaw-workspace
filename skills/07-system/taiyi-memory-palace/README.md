## 🎯 项目简介

Taiyi Memory Palace 是一个智能化的造价管理和记忆系统。

---

## 📊 数据统计

### 记忆宫殿

- **房间数量**: 9 个
- **记忆类型**: 语义/情景/程序/关联
- **复习机制**: 艾宾浩斯曲线

---

## 🚀 快速开始

```bash
# 克隆仓库
git clone https://github.com/nicola-king/taiyi-memory-palace.git
cd taiyi-memory-palace

# 安装依赖
pip install -r requirements.txt

# 运行测试
python3 main.py
```
