## 🔧 集成步骤

### Step 1: GMGN 账号注册

**操作**:
1. 访问 https://gmgn.ai
2. 连接钱包 (MetaMask/Phantom)
3. 完成 KYC (可选)
4. 配置交易参数

---

###  Step 2: Telegram Bot 配置

**操作**:
1. 搜索 @gmgn_sol_bot (Solana)
2. 搜索 @gmgn_bsc_bot (BSC)
3. 点击 Start 激活
4. 连接钱包地址

---

### Step 3: 聪明钱跟单配置

**操作**:
1. 打开 GMGN Web 终端
2. 进入「聪明钱」页面
3. 筛选高胜率钱包 (胜率>70%)
4. 点击「跟单」
5. 设置跟单参数 (金额/比例)

**推荐参数**:
```
跟单模式：固定金额
单笔金额：$50-100
最大跟单：10 个钱包
止损：-20%
止盈：+100%
```

---

### Step 4: 新币狙击配置

**操作**:
1. 打开「狙击」页面
2. 输入代币 CA
3. 设置买入金额
4. 设置 Gas 优先级
5. 启用 Anti-MEV
6. 等待开盘自动买入

**推荐参数**:
```
买入金额：0.1-1 SOL
Gas 优先：高
滑点：20%
防夹：✅ 开启
```

---

### Step 5: 条件单配置

**操作**:
1. 打开「条件单」页面
2. 选择代币
3. 设置触发条件
4. 设置买入/卖出金额
5. 启用通知

**示例**:
```
买单条件:
- 市值<$500k 买入$100
- 流动性>$1M 买入$200

卖单条件:
- 2x 卖出 50%
- 5x 卖出 30%
- 10x 清仓
```

## 🚨 风险控制

### GMGN 使用风险

| 风险 | 说明 | 应对 |
|------|------|------|
| **私钥泄露** | 连接钱包需授权 | 使用专用钱包 |
| **滑点损失** | 高波动代币滑点大 | 设置滑点上限 |
| **Gas 浪费** | 交易失败仍扣 Gas | 启用防夹模式 |
| **跟单亏损** | 聪明钱也会错 | 设置止损 |
| **狙击失败** | 开盘买入即跌 | 小额测试 |

---

### 安全建议

1. ✅ **使用专用钱包**: 不用主钱包连接 GMGN
2. ✅ **设置止损**: 每笔交易设置止损 (-20%)
3. ✅ **小额测试**: 新策略先测试$10-50
4. ✅ **启用防夹**: Anti-MEV 保护开启
5. ✅ **定期提现**: 盈利及时转出
