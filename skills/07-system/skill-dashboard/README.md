## 📊 API 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/skills` | GET | 获取 Skills 列表 |
| `/api/skills/<id>` | GET | 获取 Skill 详情 |
| `/api/skills/<id>/start` | POST | 启动 Skill |
| `/api/skills/<id>/stop` | POST | 停止 Skill |
| `/api/skills/create` | POST | 创建新 Skill |
| `/api/system/status` | GET | 系统状态 |
