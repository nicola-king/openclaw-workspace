## 🚀 快速开始

### Step 1: 环境准备

```bash
# 安装 Python 3.10+
python3 --version

# 安装 Ollama (本地模型)
curl -fsSL https://ollama.com/install.sh | sh

# 下载 Qwen 2.5 7B
ollama pull qwen2.5:7b-instruct-q4_K_M
```

---

### Step 2: 配置路由

```python
# 编辑 smart_router.py
LOCAL_MODEL = "qwen2.5:7b-instruct-q4_K_M"

# 运行测试
python3 smart_router.py "你好"
```

---

### Step 3: 测试效果

```bash
# 测试响应时间
time python3 smart_router.py "1+1 等于几"

# 预期输出：<1 秒
```
