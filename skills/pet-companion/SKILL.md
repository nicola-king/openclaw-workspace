# Pet Companion Skill

> **版本**: 1.0.0 | **创建时间**: 2026-04-03 | **负责 Bot**: 山木
> **状态**: ✅ 已激活 | **优先级**: P3-02

---

## 📋 功能概述

提供虚拟宠物陪伴系统，支持宠物成长/互动/状态管理。

---

## 🛠️ 可用命令

| 命令 | 功能 | 示例 |
|------|------|------|
| `pet create` | 创建宠物 | `pet create --name 小魔力 --type cat` |
| `pet status` | 查看状态 | `pet status` |
| `pet feed` | 喂食 | `pet feed --food fish` |
| `pet play` | 玩耍 | `pet play --toy ball` |
| `pet sleep` | 睡觉 | `pet sleep` |
| `pet clean` | 清洁 | `pet clean` |
| `pet evolve` | 进化 | `pet evolve` (满足条件时) |
| `pet list` | 列出宠物 | `pet list` |

---

## 📊 宠物状态

| 属性 | 说明 | 范围 |
|------|------|------|
| **饥饿度** | 0=饱腹，100=饥饿 | 0-100 |
| **心情** | 0=难过，100=开心 | 0-100 |
| **体力** | 0=疲惫，100=精力充沛 | 0-100 |
| **清洁度** | 0=脏，100=干净 | 0-100 |
| **等级** | 宠物等级 | 1-100 |
| **亲密度** | 与主人亲密度 | 0-100 |

---

## 📝 使用示例

### 示例 1: 创建宠物

```bash
# 太一，创建一只猫宠物
pet create --name 小魔力 --type cat --emoji 🐱
```

**输出**:
```
🐱 小魔力 来到了你的世界！
初始状态:
  饥饿度：50
  心情：80
  体力：100
  清洁度：90
  等级：1
  亲密度：50
```

### 示例 2: 查看宠物状态

```bash
# 太一，小魔力现在怎么样？
pet status
```

**输出**:
```
🐱 小魔力 (Lv.5)
━━━━━━━━━━━━━━━━━━━━
饥饿度：███░░░░░░░ 30%  (饱腹)
心情：  ████████░░ 80%  (开心)
体力：  █████████░ 90%  (精力充沛)
清洁度：████████░░ 80%  (干净)
亲密度：██████░░░░ 60%  (友好)
━━━━━━━━━━━━━━━━━━━━
状态：正在玩耍 🎾
```

### 示例 3: 互动

```bash
# 太一，喂小魔力吃鱼
pet feed --food fish

# 太一，和小魔力玩球
pet play --toy ball
```

### 示例 4: 进化

```bash
# 太一，小魔力可以进化了吗？
pet evolve
```

**输出**:
```
🎉 恭喜！小魔力进化了！
Lv.5 猫 → Lv.6 魔法猫 ✨
新技能：[卖萌] [抓老鼠]
```
