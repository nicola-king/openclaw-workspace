## 📝 手动添加白名单

**方法 1**: 编辑配置文件
```json
{
  "whitelist": ["新联系人名称"]
}
```

**方法 2**: 使用命令
```
/whitelist add 联系人名称
```

**方法 3**: 自动学习
```
连续聊天 3 次 → 自动加入白名单
```
