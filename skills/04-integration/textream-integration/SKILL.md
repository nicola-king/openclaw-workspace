## 🔧 使用方式

### 方式 1: Mac App

```bash
# 安装 Textream
brew install --cask textream

# 启动
open -a Textream

# 加载脚本
textream load script.txt
```

### 方式 2: API 集成

```python
from textream import TextreamApp

app = TextreamApp()

# 加载文本
app.load_text("这是提词内容")

# 开始提词
app.start()

# 控制速度
app.set_speed(1.0)  # 1.0x

# 实时高亮
app.highlight_current_line()
```

### 方式 3: 太一集成

```python
@taiyi.skill
def video_narration(script):
    """视频 narratio n 辅助"""
    textream = TextreamApp()
    textream.load_text(script)
    textream.start()
    
    # 等待录制完成
    wait_for_recording()
    
    textream.stop()
```
