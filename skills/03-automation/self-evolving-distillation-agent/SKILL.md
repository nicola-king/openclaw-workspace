### 自学习循环

```
执行蒸馏 → 收集数据 → 分析效果 → 优化规则 → 再次执行
    ↓
能力涌现 → 创建新技能 → 整合到系统 → 提升蒸馏效率
```

### 进化指标

**蒸馏效率进化**:
```python
效率提升 = (本次 ΔS - 上次 ΔS) / 上次 ΔS * 100%
目标：每周提升 >5%
```

**规则优化进化**:
```python
规则准确率 = 正确决策数 / 总决策数 * 100%
目标：准确率 >90%
```

**技能涌现进化**:
```python
涌现技能数 = 每周自动创建的技能数
目标：每周 >3 个有价值技能
```

```python
class 规则学习器:
    def 优化规则 (self, 历史数据):
        # 分析历史决策效果
        有效规则 = 筛选有效规则 (历史数据)
        无效规则 = 筛选无效规则 (历史数据)
        
        # 强化有效规则
        for 规则 in 有效规则:
            规则。权重 += 0.1
        
        # 弱化无效规则
        for 规则 in 无效规则:
            规则。权重 -= 0.1
        
        # 生成新规则
        新规则 = 生成新规则 (历史数据)
        
        return 更新后的规则集
```
