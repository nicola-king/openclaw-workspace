## 📋 Prompt 使用指南

### 1. 调整色彩

```
莫兰迪色系：添加 "Morandi color scheme"
温暖系：添加 "warm color palette"
冷色系：添加 "cool color palette"
单色：添加 "monochromatic [color]"
```

### 2. 调整风格

```
水墨风：添加 "Chinese ink wash painting style"
极简风：添加 "minimalism, clean lines"
水彩风：添加 "watercolor painting style"
摄影风：添加 "photography style, realistic"
```

### 3. 调整氛围

```
禅意：添加 "zen atmosphere, tranquil"
治愈：添加 "healing, soothing"
高级感：添加 "premium feel, elegant"
梦幻：添加 "dreamy, ethereal"
```

### 4. 调整构图

```
留白：添加 "negative space, extensive white space"
对称：添加 "symmetrical composition"
中心：添加 "central composition"
三分法：添加 "rule of thirds"
```
