## 📊 融合理念

### 核心原则

| 原则 | 说明 | 示例 |
|------|------|------|
| **自然融入** | 不刻意说教，潜移默化 | 壁纸中有卦象但不解释 |
| **视觉优先** | 美感第一，哲学第二 | 好看>深刻 |
| **低门槛** | 无需基础也能欣赏 | 不懂易经也能觉得美 |
| **情绪共鸣** | 先治愈，后启发 | 先打动心，再打动脑 |

## ☯️ 易经元素融合

### 1. 八卦符号视觉化

**融合方式**:
```
- 将八卦符号作为壁纸背景纹理
- 用现代极简风格重新设计卦象
- 配合自然元素 (山/水/风/雷)
- 色彩低饱和度 (莫兰迪色系)
```

| **坤☷** | 地/柔顺 | 大地/棕色/方形 | 包容/滋养 | "地势坤，君子以厚德载物" |

### 3. 易经金句壁纸

**融合方式**:
```
- 一句易经名言
- 配合意境图
- 现代字体设计
- 适合发朋友圈
```

**爆款文案库**:

```
1. "天行健，君子以自强不息" —— 奋斗/早安
2. "地势坤，君子以厚德载物" —— 包容/晚安
3. "穷则变，变则通，通则久" —— 改变/转型
4. "一阴一阳之谓道" —— 平衡/中庸
5. "积善之家，必有余庆" —— 善良/感恩
6. "君子藏器于身，待时而动" —— 沉淀/等待
7. "二人同心，其利断金" —— 合作/团队
8. "物极必反，否极泰来" —— 希望/坚持
9. "知周乎万物，而道济天下" —— 智慧/格局
10. "形而上者谓之道，形而下者谓之器" —— 哲学/思考
```

## ⚠️ 注意事项

### 避免踩坑

| 错误做法 | 正确做法 |
|---------|---------|
| 大段经文堆砌 | 精选一句 + 意境图 |
| 说教式讲解 | 故事/案例带入 |
| 宗教色彩过浓 | 文化/美学角度 |
| 强制信仰 | 尊重多元/开放包容 |
| 低质拼贴 | 高质量 AI 生成/原创设计 |

### 敏感词规避

```
避免：
- 迷信/算命/占卜 (平台敏感)
- 绝对化用语 (最/第一/ guaranteed)
- 宗教推广嫌疑

使用：
- 传统文化/东方哲学
- 美学/艺术/设计
- 情绪价值/治愈/成长
```
