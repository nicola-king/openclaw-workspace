## 📋 6 个核心提示词

### 1️⃣ 拆解目标提示词

```
你是一位专业的短视频编导，请帮我审查以下素材描述，并制定一个精准的拍摄计划。

素材描述：[在此输入]

请包括：
- 节奏设计
- 色调方案
- 音乐选择
- B 素材位置和类型
- 镜头切换时机

输出格式：分镜脚本表格
```

**用途**: 当专业短视频编导，制定精准拍摄计划

---

### 2️⃣ 风格化提示词

```
请分析以下参考视频的风格，并一步步拆解如何复制这种风格。

参考视频：[链接或描述]

请分析：
- 镜头运用（角度/运动/景别）
- 时长节奏（平均镜头时长）
- 调色风格（色调/对比度/饱和度）
- 背景音乐（类型/节奏/情绪）
- 文字字体（字体类型/大小/位置）

输出格式：风格分析报告 + 复制步骤
```

**用途**: 分析参考视频风格，一步步拆解复制

---

### 3️⃣ 提建议提示词

```
请审查以下视频初剪脚本，通过前后镜改进连贯性，并确定视频整体节奏，特别是更有节奏感的部分。

初剪脚本：[在此输入]

请提供：
- 连贯性改进建议
- 节奏优化建议
- 镜头顺序调整
- 转场效果建议

输出格式：修改建议列表（优先级排序）
```

**用途**: 审查视频初剪脚本，提供改进建议

---

### 4️⃣ 找问题提示词

```
请审查以下成片，找出问题并提供具体建议，让视频更好。

成片描述/链接：[在此输入]

请检查：
- 叙事清晰度
- 节奏问题
- 视觉问题
- 音频问题
- 整体观感

输出格式：问题清单 + 改进建议
```

**用途**: 审查成片找问题，提供建议

---

### 5️⃣ 优化提示词

```
请严格评估以下视频的逐帧画面，并提供专业反馈。

视频：[链接或描述]

请评估：
- 构图（主体位置/平衡/引导线）
- 色彩一致性（色温/色调统一）
- 清晰度（焦点/曝光）
- 具体修改建议（时间戳 + 修改内容）

输出格式：逐帧评估报告
```

**用途**: 严格评估逐帧画面，提供专业反馈

---

### 6️⃣ SEO 提示词

```
请根据以下视频内容，生成爆款标题和文案，并提供专业建议。

视频内容：[在此输入]

平台：小红书/抖音/B 站（选择）

请生成：
- 5 个爆款标题选项
- 视频文案（含 emoji）
- 热门标签（10-15 个）
- 发布时间建议
- 互动引导语

输出格式：多平台适配方案
```

**用途**: 根据视频内容生成爆款标题和文案

---

## 🔄 使用工作流

```
1. 拆解目标 → 制定拍摄计划
       ↓
2. 风格化 → 分析参考视频
       ↓
3. 提建议 → 审查初剪脚本
       ↓
4. 找问题 → 审查成片
       ↓
5. 优化 → 逐帧评估
       ↓
6. SEO → 生成标题文案
```
