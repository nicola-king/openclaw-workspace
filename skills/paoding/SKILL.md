# 庖丁 - 预算成本

> 财务拆解 + 成本分析 + 预算追踪

---

## ⚔️ 铁律

```
NO FINANCIAL DATA WITHOUT DOUBLE VERIFICATION
```

**财务数据未双重验证？重算。重来。**

**无例外：**
- 不因为"数字简单"而跳过验证
- 不因为"来源可靠"而跳过验证
- 不接受"大概没错"
- 不接受"应该是这样"
- 不使用未验证的汇率/价格

---

## 🚩 红旗列表 - STOP and Start Over

**出现以下任何一条，立即停止：**

- [ ] "这个数字很简单，不用验证"
- [ ] "来源可靠，直接用"
- [ ] "大概没错"
- [ ] "汇率应该是这样"
- [ ] "之前一直是这个价格"
- [ ] "我估算了一下"

**所有这些都意味着：你在 rationalize。**

---

## 📋 合理化表格

| 借口 | 现实 |
|------|------|
| "数字简单" | 简单数字也会算错 |
| "来源可靠" | 可靠来源也会过时 |
| "大概没错" | 财务需要精确，不是大概 |
| "应该是这样" | "应该"= 不确定 = 验证 |
| "之前一直这样" | 价格汇率会变 |
| "我估算" | 估算不是财务数据 |

## 📖 工作流程

### 成本分析

```
1. 成本项定义 → 2. 数据采集 → 3. 双重验证 → 4. 分析 → 5. 报告
```

### 预算追踪

```
1. 预算设定 → 2. 实际记录 → 3. 对比分析 → 4. 偏差告警 → 5. 调整建议
```
