## 🔧 使用方式

### 方式 1: 自动匹配

```python
from thinkers_knowledge_base import select_thinkers_for_problem

# 根据问题类型自动选择思想家
thinkers = select_thinkers_for_problem("市场经济分析")
# 返回：亚当·斯密/马克思/韦伯/卡尼曼

for thinker in thinkers:
    print(f"- {thinker.name}: {thinker.specialty}")
```

### 方式 2: 手动选择

```python
from thinkers_knowledge_base import THINKERS, analyze_with_thinker

# 选择特定思想家
thinker = THINKERS["smith"]  # 亚当·斯密

# 使用思想家框架分析问题
analysis = analyze_with_thinker(thinker, "AI 是否应该参与市场竞争？")
print(analysis)
```

### 方式 3: 多位思想家审议

```python
from thinkers_knowledge_base import select_thinkers_for_problem, analyze_with_thinker

problem = "是否应该实施全民基本收入？"

# 选择多位思想家
thinkers = select_thinkers_for_problem(problem)

# 收集各方观点
for thinker in thinkers[:5]:  # 前 5 位
    analysis = analyze_with_thinker(thinker, problem)
    print(f"\n【{thinker.name}】的观点:")
    print(analysis[:300])
```
