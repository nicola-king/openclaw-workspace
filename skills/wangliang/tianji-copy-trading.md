## 📊 聪明钱筛选标准

### 筛选条件

```python
def is_smart_money(wallet):
    """
    判断是否为聪明钱钱包
    
    标准:
    1. 交易次数 > 50 次
    2. 胜率 > 55%
    3. 总盈利 > $1000
    4. 平均持仓 > $100
    5. 最近 7 天活跃
    """
    if wallet.trade_count < 50:
        return False
    if wallet.win_rate < 0.55:
        return False
    if wallet.total_pnl < 1000:
        return False
    if wallet.avg_position < 100:
        return False
    if wallet.last_active_days > 7:
        return False
    
    return True
```

## 📈 信号生成逻辑

### 信号类型

**1. 已收到 (通知)**
```
触发条件:
- 监控钱包有新交易
- 无论价格是否合适

内容:
- 交易员地址
- 买卖方向
- 成交价格
- 成交金额

目的：信息透明，让用户自己判断
```

**2. 不追 (警告)**
```
触发条件:
- 当前价格比成交价高>20%
- 套利空间已消失

内容:
- 当前 ask/bid 价格
- spread 点差
- 追价成本

目的：避免用户高位接盘
```

**3. 可买 (推荐)**
```
触发条件:
- 当前价格接近成交价 (<10% 溢价)
- spread 合理 (<0.05)
- 套利空间>5%

内容:
- 推荐等级：🟢 可买
- 目标价位
- 预期收益

目的：明确推荐，促进转化
```
