**版本**: v1.0
**创建**: 2026-03-26
**作者**: 太一

---

## 📊 功能概述

### 核心功能
1. **数据采集**: 对接直播平台 API，实时采集数据
2. **可视化看板**: 自动生成 Markdown 格式数据报告
3. **多维度分析**: 流量/互动/转化三大维度评分
4. **问题诊断**: 自动识别问题并给出优化建议
5. **迭代进化**: 持续学习历史数据，优化分析模型

---

## 🔧 使用方法

### 基本用法
```python
from live-analytics-agent import LiveAnalyticsAgent

# 创建 Agent
agent = LiveAnalyticsAgent(platform="douyin")

# 采集数据
live_data = agent.collect_live_data("live_id")

# 分析数据
analysis = agent.analyze_performance(live_data)

# 生成报告
report = agent.generate_report(live_data, analysis)

# 保存记录（触发进化）
agent.save_live_record(live_data, analysis)
```

### 命令行用法
```bash
python3 live-analytics-agent.py
```

## 📈 分析维度

### 1. 流量维度（权重 25%）
| 指标 | 基准 | 权重 |
|------|------|------|
| 观看人数 | 5000+ | 30 分 |
| 最高在线 | 1000+ | 30 分 |
| 转粉率 | 2%+ | 40 分 |

### 2. 互动维度（权重 20%）
| 指标 | 基准 | 权重 |
|------|------|------|
| 互动率 | 5%+ | 50 分 |
| 分享率 | 2%+ | 30 分 |
| 礼物收入 | ¥500+ | 20 分 |

### 3. 转化维度（权重 20%）
| 指标 | 基准 | 权重 |
|------|------|------|
| 转化率 | 2%+ | 40 分 |
| GPM | ¥50+ | 40 分 |
| 成交金额 | ¥10000+ | 20 分 |

## 🔄 迭代进化机制

### 进化流程
```
直播 1 → 数据记录 → 分析学习
直播 2 → 数据记录 → 分析学习
直播 3 → 数据记录 → 分析学习
...
直播 N → 数据汇总 → 模型优化 → 权重调整
```

### 进化触发条件
- 直播记录 ≥ 5 场
- 自动分析高分直播特征
- 优化指标权重
- 更新建议策略
