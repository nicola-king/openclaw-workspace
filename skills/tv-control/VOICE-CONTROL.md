# 语音控制电视 Skill

> 版本：v1.0 | 创建：2026-03-28 14:17
> 功能：通过语音命令控制电视
> 唤醒词："电视" 或 "太一"
> 命令："开机" "关机" "增加音量" "减少音量" "静音" 等

## 🔧 安装步骤

### Step 1: 安装依赖

```bash
# 安装 Vosk 语音识别
pip3 install vosk

# 安装 PyAudio (麦克风输入)
sudo apt-get install -y python3-pyaudio
pip3 install pyaudio

# 安装端口音频 (可选)
sudo apt-get install -y portaudio19-dev
```

### Step 2: 下载中文语音模型

```bash
cd ~/.openclaw/workspace/skills/tv-control

# 下载 Vosk 中文模型 (约 40MB)
mkdir -p models
cd models

# 小型模型 (识别快，适合嵌入式)
wget https://alphacephei.com/vosk/models/vosk-model-cn-0.15.zip
unzip vosk-model-cn-0.15.zip

# 或大型模型 (识别更准确，需要更多资源)
# wget https://alphacephei.com/vosk/models/vosk-model-cn-0.22.zip
# unzip vosk-model-cn-0.22.zip
```

### Step 3: 测试麦克风

```bash
# 测试麦克风录音
arecord -l  # 列出录音设备
arecord -D hw:0,0 -d 5 test.wav  # 录音 5 秒
aplay test.wav  # 播放录音
```

### Step 4: 启动语音控制

```bash
cd ~/.openclaw/workspace/skills/tv-control
python3 voice-control.py
```

## 🚨 故障排除

### 问题 1: 麦克风无法使用

**解决**:
```bash
# 检查麦克风设备
arecord -l

# 测试录音
arecord -D hw:0,0 -d 5 test.wav

# 检查权限
sudo usermod -a -G audio $USER
```

### 问题 2: 识别率低

**解决**:
```
1. 使用大型模型 (vosk-model-cn-0.22)
2. 改善麦克风位置
3. 减少环境噪音
4. 说话清晰缓慢
```

### 问题 3: 响应慢

**解决**:
```
1. 使用小型模型 (vosk-model-cn-0.15)
2. 关闭其他占用 CPU 的程序
3. 升级硬件 (RAM/CPU)
```
