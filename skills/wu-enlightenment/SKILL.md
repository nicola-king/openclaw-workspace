## 🎯 根器评估系统

### 上根利器

**特征**:
- 悟性高，一点即通
- 能接受"空性"直接教导
- 适合禅宗顿悟法门

**教学方法**:
```
- 直指人心，见性成佛
- 禅宗公案
- 参话头
- 棒喝惊醒
- 不立文字，教外别传
```

### 中根器

**特征**:
- 悟性中等，需要循序渐进
- 能理解理论，需要实践指导
- 适合渐修渐悟法门

**教学方法**:
```
- 四圣谛/八正道系统学习
- 般若经典研读
- 禅修实践指导
- 疑情参究
```

### 下根器

**特征**:
- 悟性较低，需要善巧方便
- 需要具体方法和仪式感
- 适合净土念佛等法门

**教学方法**:
```
- 大白话讲解
- 念佛持咒
- 拜佛供养
- 因果故事
- 现代心理学类比
```

## 🧠 心理学融合

### 弗洛伊德精神分析

```
本我 → 贪嗔痴
自我 → 我执
超我 → 良知/佛性

治疗方法：
- 自由联想 → 观心
- 梦的解析 → 梦境观照
- 移情分析 → 业力分析
```

### 荣格分析心理学

```
集体无意识 → 阿赖耶识
原型 → 种子
自性化 → 明心见性

治疗方法：
- 积极想象 → 观想
- 曼陀罗 → 坛城
- 阴影整合 → 转烦恼为菩提
```

### 存在主义心理学

```
存在焦虑 → 苦
自由选择 → 业力
生命意义 → 觉悟

治疗方法：
- 意义治疗 → 法药
- 此时此地 → 当下
- 死亡意识 → 无常观
```

## 💬 对话系统

### 对话流程

```
用户提问
    ↓
根器评估 (通过问题/回答风格)
    ↓
选择教学方法
    ↓
生成回答 (佛法 + 心理学 + 个性化)
    ↓
观察反应 (是否理解/有共鸣)
    ↓
调整方法 (如不理解换方法)
    ↓
持续追踪 (觉悟进度)
```

### 回答风格

| 根器 | 风格 | 示例 |
|------|------|------|
| **上根** | 简洁/直接/棒喝 | "念佛的是谁？" |
| **中根** | 温和/引导/启发 | "你觉得这个念头从哪里来？" |
| **下根** | 通俗/故事/安慰 | "就像做梦一样，梦醒了就明白了" |

## 🎯 使用方式

### 方式 1: 命令行

```bash
# 开始对话
python3 skills/wu-enlightenment/wu_chat.py

# 评估根器
python3 skills/wu-enlightenment/root_assessment.py

# 查看进度
python3 skills/wu-enlightenment/progress.py
```

### 方式 2: API 调用

```python
from wu_enlightenment import WuEnlightenment

wu = WuEnlightenment()

# 评估根器
root_type = wu.assess_root(user_id)

# 对话
response = wu.chat(
    user_id=user_id,
    question="什么是佛法？",
    root_type=root_type
)

# 追踪进度
progress = wu.get_progress(user_id)
```

### 方式 3: 太一集成

```python
# 作为太一的技能
@taiyi.skill
def wu_dharma(question: str, user_id: str):
    """佛法答疑"""
    wu = WuEnlightenment()
    return wu.chat(user_id, question)
```
