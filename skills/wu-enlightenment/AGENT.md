## 🔄 自进化机制

### 进化触发条件

```
1. 用户数量增长 (>100 用户)
   → 优化根器评估算法

2. 对话轮次积累 (>1000 轮)
   → 优化对话模式

3. 觉悟突破案例 (>10 个)
   → 提炼教学方法

4. 用户反馈收集
   → 调整教学策略

5. 新佛法知识发现
   → 扩展知识库
```

### 进化流程

```
数据收集 (对话/进度/反馈)
    ↓
分析模式 (成功/失败案例)
    ↓
提炼经验 (教学方法优化)
    ↓
更新系统 (知识库/算法)
    ↓
验证效果 (新对话测试)
    ↓
持续循环
```

## 📊 用户觉悟追踪

### 四阶段模型

| 阶段 | 特征 | 互动次数 | 标志 |
|------|------|---------|------|
| **信** | 建立信心 | 1-5 次 | 愿意学习佛法 |
| **解** | 理解教义 | 5-20 次 | 能解释核心概念 |
| **行** | 实践修行 | 20-50 次 | 日常应用佛法 |
| **证** | 证悟实相 | 50+ 次 | 烦恼减轻/智慧增长 |

## 💬 对话流程

### 标准流程

```
用户提问
    ↓
获取用户档案 (根器/阶段/历史)
    ↓
分析问题和情绪
    ↓
选择教学方法 (16 种之一)
    ↓
匹配佛法教义 (12 个核心)
    ↓
生成个性化回答
    ↓
观察用户反应
    ↓
调整方法 (如不理解)
    ↓
更新觉悟进度
    ↓
记录到记忆系统
```

### 主动引导

```
定期关怀 (3 天未对话)
    ↓
发送佛法开示
    ↓
询问修行进展
    ↓
鼓励继续精进
    ↓
记录反馈
```

11. **荣格心理学** - "集体无意识/阿赖耶识"

## 🚀 使用方式

### 方式 1: 命令行

```bash
# 启动悟 Agent
python3 skills/wu-enlightenment/wu_agent.py

# 查看用户档案
python3 skills/wu-enlightenment/user_profile.py user_001

# 查看觉悟统计
python3 skills/wu-enlightenment/enlightenment_stats.py
```

### 方式 2: API 调用

```python
from wu_agent import WuEnlightenmentAgent

agent = WuEnlightenmentAgent()

# 对话
response = await agent.chat(
    user_id="user_001",
    question="什么是佛法？"
)

# 获取用户觉悟进度
progress = agent.get_enlightenment_progress("user_001")

# 主动引导
await agent.proactive_guidance("user_001")
```

### 方式 3: 太一集成

```python
# 作为太一 Agent 运行
@taiyi.agent
async def wu_dharma_agent(user_id: str, question: str):
    """佛法觉悟 Agent"""
    agent = WuEnlightenmentAgent()
    return await agent.chat(user_id, question)
```
