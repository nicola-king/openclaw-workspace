## 📊 太一安全措施

### 1. IP 合规 ✅

**措施：**
- ✅ 使用固定代理（Clash 127.0.0.1:7890）
- ✅ 避免频繁切换 IP
- ✅ 不使用公共代理

**配置：**
```bash
# .bashrc
export http_proxy=http://127.0.0.1:7890
export https_proxy=http://127.0.0.1:7890
```

---

### 2. 行为合规 ✅

**措施：**
- ✅ 随机延迟（2-5 分钟）
- ✅ 避免固定间隔
- ✅ 模拟人类行为

**实现：**
```bash
# zhiji-auto.sh
DELAY=$((120 + RANDOM % 180))  # 2-5 分钟随机
sleep $DELAY
```

---

### 3. 频率控制 ✅

**措施：**
- ✅ 每 5 分钟检查一次（非高频）
- ✅ API 调用限速
- ✅ 批量请求缓存
