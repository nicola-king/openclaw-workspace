- 🟡 引导私下交易
- 🟡 第三方平台引流

#### 推荐内容
- ✅ 真实体验分享
- ✅ 高质量原创内容
- ✅ 有价值信息
- ✅ 精美图片/视频
- ✅ 积极正面内容

### 2. 流量机制（2026 新版）

#### 流量分发逻辑
```
审核收录 → 打标签匹配 → 阶梯式流量池 → CES 评分 → 搜推互通
```

#### 流量池层级
| 层级 | 曝光量 | 晋级条件 |
|------|--------|---------|
| **初始池** | 100-500 | 审核通过 |
| **一级池** | 500-3000 | 互动率>5% |
| **二级池** | 3000-10000 | 互动率>8% |
| **三级池** | 10000-50000 | 互动率>10% |
| **热门池** | 50000+ | 互动率>15% |

#### CES 评分公式（2026 版）
```
CES = 点赞×1 + 收藏×2 + 评论×3 + 分享×4 + 停留时长×5

新算法重点：
- 停留时长权重提升
- 评论热度权重增加
- 单纯点赞权重降低
```

### 4. 笔记发布规范

#### 标题规范
- 字数：≤20 字
- 禁止：夸张/虚假/诱导
- 推荐：清晰/真实/有价值

#### 图片规范
- 数量：1-9 张
- 尺寸：3:4 或 1:1
- 质量：高清/无水印
- 禁止：二维码/联系方式

#### 标签规范
- 数量：≤10 个
- 相关：与内容强相关
- 禁止：蹭热度/无关标签

---

### 5. 商业化规则

#### 品牌合作
- **蒲公英平台**: 官方合作渠道
- **报备笔记**: 必须通过平台报备
- **佣金比例**: 10-30%
- **结算周期**: T+7

#### 广告投放
- **信息流广告**: 按点击/展示付费
- **搜索广告**: 关键词竞价
- **开屏广告**: 品牌曝光
- **最低预算**: ¥500/天

#### 违规处罚
| 违规类型 | 处罚措施 |
|---------|---------|
| 轻微违规 | 限流/降权 |
| 中度违规 | 禁言 7-30 天 |
| 严重违规 | 封号/清零 |
| 商业违规 | 扣除保证金 |

## ⚠️ 风险提醒

### 封号高风险行为
- ❌ 刷粉/刷数据
- ❌ 私下交易引流
- ❌ 发布违禁内容
- ❌ 恶意攻击他人
- ❌ 虚假宣传

### 限流常见原因
- 🟡 内容同质化
- 🟡 过度营销
- 🟡 标题党/封面党
- 🟡 频繁修改笔记
- 🟡 异常互动行为

### 合规建议
1. **内容真实**: 不夸大/不虚假
2. **遵守规则**: 熟悉平台规范
3. **优质原创**: 提升内容质量
4. **良性互动**: 真实用户互动
5. **长期主义**: 不追求短期爆发
