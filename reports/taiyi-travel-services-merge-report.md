## 🎯 合并内容

### 1. 地陪服务 + 落地导游 = 地陪导游服务

**合并前**:
```
地陪服务:
- 本地导游陪同
- 行程规划
- 景点讲解

落地导游服务:
- 导游服务
- 语言翻译
- 文化讲解
```

**合并后**:
```
地陪导游服务:
✅ 本地导游陪同
✅ 行程规划
✅ 景点讲解
✅ 语言翻译
✅ 文化讲解
✅ 美食推荐
✅ 拍照指导
✅ 可包含包车服务 (可选)
```

**使用方式**:
```python
guide = services.search_local_guide(
    destination="东京",
    days=3,
    language="中文",
    travelers=2,
    tour_type="深度游",
    include_car_service=False  # 可选：是否包含包车
)
```

### 2. 落地包车 + 落地接机 = 包车接机服务

**合并前**:
```
落地包车:
- 多日包车服务
- 专业司机
- 燃油/过路/停车

落地接机:
- 机场接机
- 航班追踪
- 举牌接机
```

**合并后**:
```
包车接机服务:
✅ 多日包车服务
✅ 专业司机
✅ 燃油/过路/停车
✅ 机场接机 (可选)
✅ 航班追踪
✅ 举牌接机
✅ 协助搬运行李
```

**使用方式**:
```python
charter = services.search_charter_car(
    destination="东京",
    days=5,
    car_type="舒适型",
    travelers=2,
    include_airport_pickup=True,  # 可选：是否包含接机
    airport="成田国际机场",        # 接机机场
    flight_number="CA181"         # 航班号
)
```
