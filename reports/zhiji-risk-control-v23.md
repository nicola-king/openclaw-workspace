## 📊 三层风控体系

### 第一层：事前风控（预防）

| 措施 | 参数 | 说明 |
|------|------|------|
| **置信度阈值** | 96% | 只执行高分策略 |
| **优势阈值** | 4.5% | 覆盖 2.5% 成本后仍盈利 |
| **仓位控制** | Quarter-Kelly | 25% 凯利公式（保守） |
| **最大暴露** | 5% | 单笔不超过 5% 资金 |

---

### 第二层：事中风控（监控）

| 措施 | 阈值 | 动作 |
|------|------|------|
| **日损熔断** | -10% | 停止当日交易 |
| **周损熔断** | -20% | 停止本周交易 |
| **最大回撤** | -15% | 清仓并重新评估策略 |
| **流动性监控** | < $1000 | 跳过该市场 |

**执行逻辑：**
```python
if daily_pnl <= -0.10:
    halt_trading("日损熔断触发")
    notify_user()

if weekly_pnl <= -0.20:
    halt_trading("周损熔断触发")
    notify_user()

if drawdown >= 0.15:
    close_all_positions()
    halt_trading("最大回撤熔断触发")
    notify_user()
```
