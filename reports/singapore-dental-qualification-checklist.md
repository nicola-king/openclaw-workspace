## 📄 详细文件要求

### QUAL-001: 公司注册证书 (UEN)

**要求**:
- 新加坡会计与企业管制局 (ACRA) 颁发
- 显示完整公司名称和 UEN 号码
- 清晰彩色扫描件 (PDF)

**获取方式**:
```
https://www.acra.gov.sg/
```

**备注**: 如为中国公司，需提供营业执照 + 公证翻译件

---

### QUAL-011: BCA 承包商执照

**等级要求**:
- **CR13 级** (装修工程) - 最高限额 SGD 85 万
- **CR09 级** (建筑工程) - 最高限额 SGD 85 万
- 或 **L6 级** (最高等级，无限额)

**申请条件**:
- 公司注册资本 ≥ SGD 25 万
- 技术人员 ≥ 5 名
- 过去 3 年完成项目 ≥ SGD 200 万

**申请链接**:
```
https://www1.bca.gov.sg/industry-professionals/licensing-and-registration/contractor-licence
```

**费用**: SGD 300-600/年

---

### QUAL-012: PPVC 制造商认证

**认证类型**:
- **PPVC MAS** (Manufacturer Accreditation Scheme)
- 由新加坡混凝土学会 (SCI) 管理

**申请要求**:
- 工厂面积 ≥ 5000㎡
- 年产能 ≥ 1000 模块
- 质量管理体系认证
- 至少 2 个 PPVC 项目经验

**认证流程**:
```
1. 提交申请 (2 周)
2. 工厂审核 (4 周)
3. 样品测试 (2 周)
4. 获得认证 (2 周)
```

**总时间**: 10 周  
**费用**: SGD 5,000-10,000

**认证机构**:
```
Singapore Concrete Institute (SCI)
https://scinst.org.sg/ppvc-manufacturer-accreditation-scheme/
```

---

- 最近 3 年财务报表
