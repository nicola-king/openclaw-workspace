## 🎬 详细分镜脚本

### 镜头 1: 开场震撼 (0-3 秒)

**画面描述**:
- **起始帧**: 折叠状态 (紧凑的集装箱形态)
- **转场**: 快速延时摄影展示展开过程
- **结束帧**: 完全展开的现代化房屋

**拍摄要点**:
- 使用无人机航拍或高机位
- 延时摄影压缩 2 小时过程到 3 秒
- 对比强烈，突出"快"

**字幕动画**:
```
0.0s: "2 小时" (放大弹出)
0.5s: "建成" (从左滑入)
1.0s: "一栋房？！" (从右滑入 + 感叹号跳动)
2.0s: 三个词组合成完整句子
3.0s: 淡出
```

**音效**:
- 0.0s: "叮!" (提示音)
- 0.5-2.5s: 快速"嗖嗖"展开音效
- 3.0s: "咚!" (落地重音)

---

### 镜头 2: 快速部署 (3-8 秒)

**画面描述**:
- **3-5s**: 2-3 名工人轻松推动房屋侧板
- **5-7s**: 液压系统自动展开特写
- **7-8s**: 房屋主体结构完成锁定

**拍摄要点**:
- 中景拍摄工人操作 (突出轻松不费力)
- 特写液压杆/锁定机构 (展示技术细节)
- 快节奏剪辑 (每个镜头 1-2 秒)

**字幕动画**:
```
3.0s: "快速部署" (从下往上滑入)
5.0s: "2 小时" (数字放大强调)
7.0s: "完成主体" (打字机效果)
```

**旁白** (可选):
> "宫阙折叠式集成房屋，2 小时完成主体搭建"

**音效**:
- 机械展开声"咔嚓咔嚓"
- 锁定声"咔哒"
- BGM: 快节奏电子音乐 (120-140BPM)

---

### 镜头 3: 内部装修 (8-12 秒)

**画面描述**:
- **8-9s**: 推门进入，展示客厅/卧室全景
- **9-10s**: 现代厨房特写 (橱柜 + 台面)
- **10-11s**: 卫浴间特写 (淋浴 + 马桶)
- **11-12s**: 卧室/办公区 (床/桌椅 + 窗户采光)

**拍摄要点**:
- 使用稳定器平滑推进
- 每个空间 1 秒快速切换
- 突出"现代化"和"精装修"
- 自然光 + 室内灯光结合

**字幕动画**:
```
8.0s: "现代精装" (从左滑入)
9.0s: "厨房" (小字标注在画面角落)
10.0s: "卫浴" (小字标注)
11.0s: "卧室/办公" (小字标注)
```

**旁白** (可选):
> "高品质内饰，即装即用"

**音效**:
- 开门声"吱呀"
- 轻柔"叮"声标注每个空间
- BGM 继续

### 镜头 4: 外观全景 (12-16 秒)

## 🎨 后期制作规范

### 视频参数

| 参数 | 设置 |
|------|------|
| **分辨率** | 1080x1920 (竖屏 9:16) |
| **帧率** | 30fps (或 60fps 慢动作) |
| **码率** | 15-20 Mbps |
| **格式** | MP4 (H.264) |
| **色彩** | Rec.709 (标准) |

### 调色风格

- **整体**: 明亮 / 高对比 / 饱和度适中
- **室内**: 暖色调 (温馨感)
- **外观**: 冷色调 (科技感)
- **LOGO**: 金属质感 + 轻微发光

### 字幕规范

| 属性 | 设置 |
|------|------|
| **字体** | 思源黑体 / 站酷高端黑 |
| **字号** | 主标题 48pt / 副标题 32pt |
| **颜色** | 白色 + 黑色描边 |
| **位置** | 安全区域内 (避免被平台 UI 遮挡) |
| **动画** | 简洁为主 (滑入/淡入/打字机) |

### 转场特效

- 场景切换：**快速闪白** 或 **擦除**
- 时间过渡：**延时摄影**
- 重点强调：**放大弹出**
- 避免使用：花哨 3D 转场
