### 核心思想（12 大观点）

#### 1. 人的行动学 (Praxeology)

```
核心定义：
"人的行动是有目的的行为"

基本公理：
1. 人行动（不可否认的自明真理）
2. 行动有目的
3. 人偏好手段达成目的
4. 时间稀缺
5. 未来不确定

从这一公理演绎出全部经济学
```

**方法论特点**:
| 特点 | 说明 |
|------|------|
| 先验演绎 | 从公理演绎，非经验归纳 |
| 定性分析 | 非数学建模 |
| 个人主义 | 只有个人行动 |
| 主观主义 | 价值是主观的 |

#### 2. 主观价值论

```
核心观点：
- 价值不是客观的
- 价值存在于行动者心中
- 价值通过选择显示（显示性偏好）
- 无法人际比较
```

**例证**:
```
一杯水对沙漠中的人价值 > 对河边的人
同一商品，不同人价值不同
同一人，不同时间价值不同
```

#### 3. 边际效用递减

```
核心观点：
- 每增加一单位，效用递减
- 解释需求曲线向下倾斜
- 解释价值悖论（水 vs 钻石）
```

#### 4. 机会成本

```
核心观点：
- 成本是放弃的次优选择
- 成本是主观的
- 成本在行动时决定
```

#### 5. 时间偏好

```
核心观点：
- 人偏好现在而非未来
- 利息是时间偏好的体现
- 资本积累降低时间偏好
- 强制储蓄扭曲时间偏好
```

#### 6. 经济计算问题（社会主义批判）

```
核心论证：

1. 理性经济计算需要价格
2. 价格需要市场交换
3. 生产资料公有制无市场
4. 因此社会主义无法计算

结论：
"社会主义是计划的经济混乱"
```

**米塞斯挑战 (1920)**:
```
"社会主义经济中，如何决定生产什么、如何生产、为谁生产？
没有价格信号，计划者无法知道资源的最优配置。"
```

#### 7. 商业周期理论 (ABCT)

```
核心机制：

1. 央行人为降低利率
2. 企业家误判信号，过度投资
3. 资本品行业过度扩张
4. 消费者时间偏好未变
5. 资源错配暴露
6. 经济危机爆发

结论：
"经济危机是央行干预的结果，而非市场的失败"
```

**周期过程**:
```
低利率 → 过度投资 → 资源错配 → 危机 → 清算 → 恢复
```

#### 8. 方法论个人主义

```
核心观点：
- 只有个人行动
- 集体不行动
- "国家""社会"是抽象概念
- 分析必须从个人出发
```

#### 9. 自由市场

```
核心观点：
- 市场是自愿合作的秩序
- 价格协调分散知识
- 企业家发现机会
- 政府干预扭曲信号
```

#### 10. 政府干预的后果

```
核心观点：
"干预主义自我挫败"

干预链条：
价格管制 → 短缺 → 更多管制 → 全面控制 → 社会主义

例证：
- 最低工资 → 失业
- 租金管制 → 住房短缺
- 关税 → 消费者受损
```

#### 11. 货币理论

```
核心观点：
- 货币非中性
- 货币注入有分配效应
- 通胀是货币现象
- 金本位限制政府通胀
```

**通胀定义**:
```
"通胀是货币供给增加，而非物价上涨"
物价上涨是通胀的结果，不是通胀本身
```

#### 12. 自由意志主义

```
政治主张：
- 最小政府（守夜人）
- 私有财产神圣
- 契约自由
- 不侵犯原则
```

### 核心思想（12 大观点）

#### 1. 知识问题 (The Knowledge Problem)

```
核心论证：

1. 知识分散在数百万人手中
2. 知识是特定时间地点的
3. 中央计划者无法获取全部知识
4. 因此中央计划注定失败

"经济问题不是如何分配给定资源，
而是如何利用分散的知识。"
```

**知识类型**:
| 类型 | 特点 | 例子 |
|------|------|------|
| **科学知识** | 可编码、可传播 | 物理定律 |
| **特定知识** | 时间地点特定 | 本地市场情况 |
| **默会知识** | 难以言传 | 技能、经验 |

#### 2. 价格作为信号

```
核心观点：
"价格是知识的浓缩装置"

价格传递：
- 稀缺程度
- 需求强度
- 机会成本
- 替代可能性

"一个观察价格的人，不需要知道为什么
价格变化，就可以调整行为。"
```

**例证**:
```
锡价格上涨 → 用户节约用锡
用户不需要知道：
- 锡矿罢工
- 新需求出现
- 供应减少

价格自动协调行为
```

#### 3. 自发秩序 (Spontaneous Order)

```
核心观点：
"秩序可以来自自发，而非设计"

自发秩序特点：
- 无中心设计者
- 个体追求自身目标
- 产生整体秩序
- 比设计的更复杂

例子：
- 语言
- 法律（普通法）
- 市场
- 道德
```

**两种秩序**:
| 类型 | 来源 | 例子 |
|------|------|------|
| **人造秩序** | 设计创造 | 组织、企业 |
| **自发秩序** | 演化形成 | 市场、语言、法律 |

#### 4. 通往奴役之路

```
核心论证：

1. 经济计划需要统一目标
2. 统一目标需要强制
3. 强制需要权力集中
4. 权力集中导致极权

"计划经济必然导致政治专制"
```

**逻辑链条**:
```
经济计划 → 资源控制 → 人身控制 → 思想控制 → 极权
```

#### 5. 自由的价值

```
核心观点：
- 自由是最高政治价值
- 自由允许试错和创新
- 自由社会更繁荣
- 自由不是手段，是目的
```

**自由定义**:
```
"自由是不受他人专断意志强制的状态"
```

#### 6. 法治 vs 人治

```
核心区分：

**法治 (Rule of Law)**:
- 普遍适用的规则
- 事先已知
- 平等适用
- 限制政府

**人治 (Rule of Men)**:
- 专断命令
- 事后决定
- 特权存在
- 政府无限
```

#### 7. 社会正义的幻象

```
核心观点：
"社会正义在市场秩序中无意义"

论证：
- 市场结果非人为设计
- 非设计结果无所谓正义
- "社会正义"是空洞概念
- 追求社会正义破坏法治
```

#### 8. 文化演化

```
核心观点：
- 制度是演化产物
- 成功制度被模仿
- 传统承载智慧
- 理性建构危险
```

**演化机制**:
```
变异 → 竞争 → 选择 → 传承
```

#### 9. 货币非国家化

```
核心主张：
- 政府垄断货币导致通胀
- 允许私人货币竞争
- 良币驱逐劣币
- 限制政府通胀能力
```

#### 10. 企业家精神

```
核心观点：
- 企业家发现机会
- 市场是发现过程
- 竞争是发现程序
- 创新来自边缘
```

#### 11. 有限政府

```
政府正当职能：
- 保护私有财产
- 执行合同
- 国防
- 有限公共品

政府不当职能：
- 经济计划
- 价格管制
- 收入再分配
- 产业政策
```

#### 12. 致命的自负

```
核心批判：
"社会主义是致命的自负"

自负表现：
- 认为可以设计社会
- 认为可以掌握全部知识
- 认为可以计划经济
- 忽视演化智慧
```
