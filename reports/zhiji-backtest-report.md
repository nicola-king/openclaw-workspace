## 附录 A：代码片段

### 信号生成

```python
def generate_signal(confidence, edge):
    if confidence >= 0.96 and edge >= 0.02:
        bet_size = kelly_quarter(edge, confidence)
        return Signal(action='BUY', size=bet_size)
    return Signal(action='HOLD', size=0)
```

### Kelly 公式

```python
def kelly_quarter(edge, confidence):
    # Quarter-Kelly: 1/4 Kelly
    kelly = (edge * confidence) / 4
    return min(kelly, 0.02)  # Cap at 2%
```
