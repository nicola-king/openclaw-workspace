### 2. 流量算法机制（2026 版）

#### 双引擎推荐算法
```
社交推荐 (40%) + 机器推荐 (60%) = 综合流量
```

#### 社交推荐逻辑
```
好友点赞 → 好友圈曝光 → 好友的好友 → 破圈传播

权重：
- 好友点赞：权重最高
- 好友评论：次高
- 好友分享：高
- 好友完播：中
```

#### 机器推荐逻辑
```
内容标签 → 用户画像 → 匹配推荐 → 数据反馈 → 流量调整

核心指标：
- 完播率 (35%)
- 互动率 (25%)
- 停留时长 (20%)
- 转粉率 (10%)
- 分享率 (10%)
```

#### 流量池层级
| 层级 | 曝光量 | 晋级条件 |
|------|--------|---------|
| **初始池** | 200-500 | 审核通过 |
| **一级池** | 500-3000 | 完播率>30% |
| **二级池** | 3000-10000 | 完播率>40% + 互动>5% |
| **三级池** | 10000-50000 | 完播率>50% + 互动>8% |
| **热门池** | 50000+ | 完播率>60% + 互动>10% |
| **爆款池** | 100 万 + | 社交裂变 + 高留存 |

### 4. 视频发布规范

#### 视频规格
| 项目 | 要求 |
|------|------|
| **时长** | 1 秒 -30 分钟 |
| **尺寸** | 9:16 (竖屏) 或 16:9 (横屏) |
| **分辨率** | ≥720p (推荐 1080p) |
| **格式** | MP4/MOV |
| **大小** | ≤4GB |
| **封面** | 3:4 比例，清晰无水印 |

#### 文案规范
- **字数**: ≤1000 字
- **话题**: ≤3 个
- **位置**: 可添加
- **链接**: 仅限公众号/小程序

#### 发布时间建议
| 时段 | 流量 | 建议 |
|------|------|------|
| 06:00-08:00 | 早高峰 | ✅ 推荐 |
| 12:00-14:00 | 午休 | ✅ 推荐 |
| 18:00-20:00 | 晚高峰 | ✅ 最佳 |
| 21:00-23:00 | 睡前 | ✅ 推荐 |
