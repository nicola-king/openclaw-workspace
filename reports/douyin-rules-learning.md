### 4. 视频发布规范

#### 视频规格
| 项目 | 要求 |
|------|------|
| **时长** | 15 秒 -30 分钟 |
| **尺寸** | 9:16 (竖屏) 或 16:9 (横屏) |
| **分辨率** | ≥720p (推荐 1080p) |
| **格式** | MP4/MOV |
| **大小** | ≤4GB |
| **封面** | 3:4 比例，清晰无水印 |

#### 文案规范
- **标题**: ≤55 字
- **话题**: ≤5 个
- **位置**: 可添加
- **@好友**: ≤5 个

## 📈 运营策略

### 1. 内容创作策略

#### 爆款公式（2026 版）
```
爆款 = 黄金 3 秒开头 + 价值内容 + 互动引导 + 社交属性

黄金 3 秒：
- 制造悬念
- 提出痛点
- 展示结果
- 引发好奇
```

#### 热门内容类型
1. **情感共鸣类**: 故事/感悟/励志
2. **知识干货类**: 教程/技巧/科普
3. **生活记录类**: Vlog/日常/旅行
4. **娱乐搞笑类**: 段子/剧情/才艺
5. **垂直专业类**: 行业/技能/经验

#### 更新频率
- **新号**: 日更 1-2 条 (养号期 7-14 天)
- **成长号**: 周更 3-5 条
- **成熟号**: 周更 2-3 条
- **直播**: 周播 2-3 场
