# 变更台账 Excel 自动化工具 - 使用说明

> **编制时间**: 2026-04-14  
> **适用软件**: Microsoft Excel 2016+ / WPS 表格 / Google Sheets

## 📊 工具功能

### 核心功能
| 功能 | 描述 | 自动化程度 |
|------|------|-----------|
| **台账登记** | 变更/签证信息录入 | 手动录入 |
| **状态跟踪** | 审批状态实时更新 | 自动计算 |
| **金额统计** | 申报/审批金额汇总 | 自动求和 |
| **预警提示** | 超时未批自动预警 | 自动判断 |
| **证据完整性** | 证据链评分自动计算 | 公式计算 |
| **进度款关联** | 变更纳入进度款申报 | 自动关联 |
| **报表生成** | 周报/月报自动生成 | 一键生成 |

## 📄 各表说明

### 1️⃣ 台账总表（主表）

**功能**：所有变更签证的汇总视图

**字段**：
| 列号 | 字段名 | 数据类型 | 说明 |
|------|-------|---------|------|
| A | 序号 | 自动 | 自动编号 |
| B | 变更编号 | 文本 | BG-2026-001 |
| C | 变更名称 | 文本 | 经纬大道管径变更 |
| D | 变更类型 | 下拉 | A 类/B 类/C 类/D 类 |
| E | 申报日期 | 日期 | 2026-04-14 |
| F | 位置/桩号 | 文本 | K0+120~K0+350 |
| G | 变更原因 | 下拉 | 管线冲突/地质变化/居民协调/设计优化 |
| H | 申报金额 | 数字 | ¥800,000 |
| I | 审批金额 | 数字 | ¥750,000 |
| J | 核减金额 | 公式 | =H-I |
| K | 核减率 | 公式 | =J/H |
| L | 审批状态 | 下拉 | 已批/审批中/被拒/已撤销 |
| M | 审批日期 | 日期 | 2026-04-20 |
| N | 审批天数 | 公式 | =M-E |
| O | 超时天数 | 公式 | =IF(L="已批",IF(N>审批时限,N-审批时限,0),TODAY()-E-审批时限) |
| P | 预警级别 | 公式 | =IF(O<=3,"🟢正常",IF(O<=7,"🟡关注",IF(O<=14,"🟠预警","🔴警报"))) |
| Q | 证据完整性 | 数字 | 0-100 分 |
| R | 证据等级 | 公式 | =IF(Q>=90,"🟢完整",IF(Q>=70,"🟡基本完整",IF(Q>=50,"🟠高风险","🔴缺失"))) |
| S | 是否纳入进度款 | 下拉 | 是/否 |
| T | 进度款期数 | 文本 | 第 3 期 |
| U | 责任人 | 文本 | 张三 |
| V | 备注 | 文本 | - |

**条件格式**：
- 预警级别列：🟢绿色 / 🟡黄色 / 🟠橙色 / 🔴红色
- 超时天数>7：红色字体
- 证据等级="🔴缺失"：整行红色背景

## 🔧 核心公式

### 超时天数计算
```excel
=IF(L2="已批",
    IF(N2>审批时限，N2-审批时限，0),
    TODAY()-E2-审批时限
)
```

### 预警级别自动判断
```excel
=IF(O2<=3,"🟢正常",
    IF(O2<=7,"🟡关注",
        IF(O2<=14,"🟠预警","🔴警报")
    )
)
```

### 证据完整性评分
```excel
=书面证据得分*40% + 影像证据得分*35% + 第三方证据得分*25%
```

### 核减率计算
```excel
=IF(H2=0,0,(H2-I2)/H2)
```

### 审批天数计算
```excel
=IF(AND(M2<>"",E2<>""),M2-E2,"")
```

## 📋 使用流程

### 日常使用
```
1. 发生变更 → 6️⃣数据录入表 → 填写表单 → 提交
2. 自动进入 1️⃣台账总表 → 状态为"审批中"
3. 跟踪审批 → 更新状态/金额/日期
4. 4️⃣预警看板 → 自动更新预警信息
5. 每周五 → 5️⃣统计分析 → 生成周报
6. 每月底 → 5️⃣统计分析 → 生成月报
```

### 进度款申报
```
1. 筛选"已批准"且"未纳入进度款"的变更
2. 勾选"纳入进度款" → 填写期数
3. 导出进度款变更清单
4. 附变更签证单 → 申报进度款
```

### 预警处理
```
1. 4️⃣预警看板 → 查看🟠🔴预警项
2. 按责任人分配 → 跟踪处理
3. 超时>7 天 → 商务经理催办
4. 超时>14 天 → 项目经理协调
5. 超时>21 天 → 上报业主分管领导
```
