## 💰 成本核算

### 出口成本构成

```
总成本 = 采购成本 + 国内费用 + 国际运费 + 保险费 + 关税 + 其他

详细分解：
1. 采购成本：工厂出厂价
2. 国内费用：
   - 拖车费：¥500-2000
   - 报关费：¥200-500
   - 港杂费：¥500-1500
   - 商检费：¥500-2000（法检产品）
3. 国际运费：
   - 海运：$500-5000/柜
   - 空运：$5-15/kg
   - 铁路：$1000-3000/柜
4. 保险费：货值×0.3%
5. 关税：根据目的国（0-15%）
6. 其他：
   - 认证费：¥5000-50000
   - 翻译费：¥500-2000
   - 银行手续费：0.1%
```

### 利润核算

```
FOB 报价 = 采购成本 + 国内费用 + 利润

CIF 报价 = FOB + 国际运费 + 保险费

DDP 报价 = CIF + 关税 + 清关费 + 派送费

建议利润率：
- 小型发动机：15-25%
- 3D 打印机：20-30%
- 变压器：15-25%
- 储能系统：25-40%
- 电池：15-25%
- 小型工程机械：20-30%
```

## 📋 出口流程

### 完整流程（12 步）

```
1. 客户开发 → 2. 询盘报价 → 3. 商务谈判 → 4. 签订合同
       ↓
5. 收取定金 → 6. 安排生产 → 7. 产品检验 → 8. 订舱报关
       ↓
9. 装运发货 → 10. 收取尾款 → 11. 单据移交 → 12. 售后跟踪
```

### 详细步骤

#### 1. 客户开发
- **渠道**: 阿里巴巴国际站、展会、Google 广告、社交媒体
- **成本**: ¥10000-100000/年
- **周期**: 1-6 个月

#### 2. 询盘报价
- **内容**: 产品规格、数量、价格、交期
- **报价方式**: FOB/CIF/DDP
- **有效期**: 7-30 天

#### 3. 商务谈判
- **重点**: 价格、付款方式、交期、售后
- **技巧**: 阶梯报价、样品策略

#### 4. 签订合同
- **合同类型**: 销售合同 (Sales Contract)
- **关键条款**: 产品规格、价格、付款、交期、质保
- **付款方式**: T/T 30% 定金 +70% 见提单副本

#### 5. 收取定金
- **比例**: 30-50%
- **方式**: 电汇、信用证
- **确认**: 到账后安排生产

#### 6. 安排生产
- **周期**: 7-45 天（根据产品）
- **跟进**: 生产进度、质量检验
- **文档**: 生产记录、质检报告

#### 7. 产品检验
- **自检**: 工厂质检
- **第三方**: SGS/BV/Intertek
- **法检**: 商检局检验（法检产品）

#### 8. 订舱报关
- **订舱**: 货代订舱
- **报关资料**: 合同、发票、箱单、报关单
- **商检**: 法检产品需商检

#### 9. 装运发货
- **方式**: 海运/空运/铁路
- **单据**: 提单、保险单
- **通知**: 发送装运通知给客户

#### 10. 收取尾款
- **条件**: 见提单副本
- **方式**: 电汇
- **风险**: 客户不付款（买保险）

#### 11. 单据移交
- **单据**: 提单、发票、箱单、原产地证
- **方式**: 快递/电放
- **确认**: 客户收到单据

#### 12. 售后跟踪
- **内容**: 产品使用情况、问题处理
- **周期**: 质保期内
- **目标**: 返单、转介绍

## ⚠️ 风险管理

### 主要风险

| 风险类型 | 说明 | 应对措施 |
|---------|------|---------|
| **政治风险** | 战争/政变/制裁 | 出口信用保险 |
| **商业风险** | 客户违约/破产 | 定金 + 信用证 |
| **物流风险** | 延误/丢包/损坏 | 购买保险 |
| **汇率风险** | 汇率波动 | 锁汇/人民币结算 |
| **质量风险** | 产品不合格 | 质检 + 质保 |
| **认证风险** | 认证不通过 | 提前认证 |

### 出口信用保险

| 保险公司 | 产品 | 费率 | 赔付 |
|---------|------|------|------|
| **中国信保** | 短期险 | 0.5-1% | 90% |
| **平安保险** | 出口险 | 0.3-0.8% | 80-90% |
| **人保财险** | 贸易险 | 0.4-0.9% | 80-90% |

**推荐**: 中国信保（政策性强、赔付快）
