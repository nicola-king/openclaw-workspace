## 📊 Polymarket 基础规则

### 1. 平台概述

**Polymarket** 是一个去中心化的预测市场平台，建立在 Polygon 区块链上。

| 特点 | 说明 |
|------|------|
| **区块链** | Polygon (以太坊 Layer 2) |
| **结算货币** | USDC (稳定币) |
| **市场类型** | 二元预测 (Yes/No) |
| **价格范围** | $0.01 - $0.99 |
| **结算方式** | 事件结果确定后自动结算 |

---

### 2. 交易规则

#### 买入规则
- **最小买入**: $0.01
- **最大买入**: 根据市场流动性
- **价格=概率**: $0.60 = 60% 发生概率
- **Yes + No = $1**: 无论结果如何，正确预测获得$1

#### 卖出规则
- **随时卖出**: 可在交易时间内随时卖出持仓
- **市价卖出**: 按当前市场价立即成交
- **限价卖出**: 设定目标价格等待成交

#### 手续费
- **交易手续费**: 2% (从盈利中扣除)
- **Gas 费**: Polygon 网络费用 (约 $0.01-0.10)
- **无入金/出金费用**: USDC 转账费用

---

### 3. 市场类型

| 类型 | 说明 | 例子 |
|------|------|------|
| **事件预测** | 特定事件是否发生 | "X 会赢得选举吗？" |
| **价格预测** | 资产价格区间 | "BTC 会突破$100K 吗？" |
| **天气预测** | 天气相关事件 | "某地温度会超过 X 度吗？" |
| **体育预测** | 体育赛事结果 | "某队会赢得冠军吗？" |
| **财经预测** | 经济指标 | "美联储会加息吗？" |

## 📈 交易策略

### 1. 套利策略 (Arbitrage)

**原理**: 利用不同市场之间的价格差异

**示例**:
- 市场 A: Yes $0.55
- 市场 B: Yes $0.60
- 操作：在 A 买入，在 B 卖出
- 利润：$0.05/股 (扣除手续费后)

**风险**: 
- 市场流动性不足
- 价格快速变化
- 手续费侵蚀利润

---

### 2. 做市策略 (Market Making)

**原理**: 同时挂买单和卖单，赚取差价

**示例**:
- 挂买单：Yes @ $0.58
- 挂卖单：Yes @ $0.62
- 差价：$0.04/股
- 利润：扣除手续费后约 $0.02/股

**风险**:
- 库存风险 (单向持仓过多)
- 信息不对称风险
- 市场波动风险

**建议**:
- 保持库存平衡
- 设置止损点
- 监控市场新闻

---

### 3. 价值投资策略

**原理**: 基于基本面分析，寻找被错误定价的市场

**步骤**:
1. 研究事件背景
2. 评估真实概率
3. 对比市场价格
4. 发现价值偏差
5. 下注等待价值回归

**示例**:
- 市场分析：某事件真实概率 70%
- 市场价格：Yes $0.50 (50% 概率)
- 操作：买入 Yes
- 预期收益：$0.50/股 (100% 回报)

**风险**:
- 分析错误
- 黑天鹅事件
- 时间成本

---

### 4. 趋势跟踪策略

**原理**: 跟随市场趋势，顺势而为

**信号**:
- 价格突破关键位
- 成交量放大
- 新闻事件驱动

**操作**:
- 上升趋势：买入 Yes
- 下降趋势：买入 No
- 设置止损

**风险**:
- 趋势反转
- 假突破
- 滞后性

---

### 5. 对冲策略

**原理**: 同时持有相反头寸，降低风险

**示例**:
- 持仓：Yes 100 股 @ $0.60
- 对冲：No 50 股 @ $0.40
- 结果：
  - Yes 赢：赚 $40 (100×$1 - $60 - $20)
  - No 赢：亏 $20 (50×$1 - $40 - $60)

**适用场景**:
- 不确定性高的事件
- 保护已有利润
- 降低波动

## ⚠️ 风险管理

### 1. 仓位管理

| 规则 | 说明 |
|------|------|
| **单笔最大仓位** | 不超过总资金的 5% |
| **单市场最大仓位** | 不超过总资金的 10% |
| **总仓位** | 不超过总资金的 50% |
| **止损** | 亏损达 20% 立即止损 |

### 2. 资金管理

**Kelly 公式** (建议 Quarter-Kelly):
```
f* = (p × b - q) / b

f* = 下注比例
p = 胜率
q = 败率 (1-p)
b = 赔率
```

**保守策略**:
- Quarter-Kelly: 计算结果的 25%
- 避免过度下注
- 长期稳定增长

### 3. 市场选择

**优质市场特征**:
- ✅ 高流动性 (日成交量>$10K)
- ✅ 明确结算条件
- ✅ 可靠数据源
- ✅ 合理到期时间

**避免市场**:
- ❌ 低流动性 (日成交量<$1K)
- ❌ 模糊结算条件
- ❌ 数据来源不可靠
- ❌ 到期时间过长 (>3 个月)

## 📋 操作流程

### 1. 注册与入金

1. **注册账户**: 连接钱包 (MetaMask 等)
2. **充值 USDC**: 从交易所转账到 Polygon 网络
3. **验证身份**: 完成 KYC (如需要)

### 2. 选择市场

1. **浏览市场**: 按类别筛选
2. **研究背景**: 阅读市场描述和规则
3. **分析概率**: 评估真实发生概率
4. **对比价格**: 寻找价值机会

### 3. 下单交易

1. **选择方向**: Yes 或 No
2. **输入金额**: 设定下注金额
3. **确认订单**: 检查价格和手续费
4. **执行交易**: 确认并等待成交

### 4. 持仓管理

1. **监控市场**: 关注价格变化
2. **调整仓位**: 根据情况加减仓
3. **止盈止损**: 达到目标价格及时卖出
4. **到期结算**: 等待事件结果

### 5. 出金提现

1. **卖出持仓**: 将份额换回 USDC
2. **提现到钱包**: 从 Polymarket 提现
3. **交易所变现**: 将 USDC 换成法币
