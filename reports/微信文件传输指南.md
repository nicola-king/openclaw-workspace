## ⚠️ 常见问题

### Q1: 微信扫码后还是无法发送
**A**: 等待 30 秒，token 需要时间同步。或者重启 gateway：
```bash
openclaw gateway restart
```

### Q2: 文件太大发送失败
**A**: PDF 限制 100MB，当前 580KB 远小于限制。如果更大，使用云存储。

### Q3: 手机打不开 PDF
**A**: 安装 WPS Office 或 Adobe Acrobat Reader。

### Q4: 微信 token 过期
**A**: 每 30 天需要重新登录一次。设置提醒：
```bash
# 添加到 crontab
0 0 1 * * echo "微信 token 需要更新" | mail -s "太一提醒" 285915125@qq.com
```
