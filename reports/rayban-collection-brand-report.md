## 🔍 Collection 系列解析

**Ray-Ban Collection** 是雷朋的高端特别系列，特点：

| 特征 | 说明 |
|------|------|
| **双品牌标识** | Ray-Ban + Collection |
| **premium 材质** | 皮革、金属、高级板材 |
| **限量设计** | 特别款/季节款 |
| **完整配件** | 硬盒 + 皮袋 + 清洁布 |
| **价格定位** | 高于经典款 30-50% |

| 位置 | 标识 | 含义 |
