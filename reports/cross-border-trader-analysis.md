### 第 4 步：报价与谈判（1 周）

**报价公式**:
```
FOB 报价 = 采购成本 + 国内费用 + 利润

CIF 报价 = FOB + 国际运费 + 保险费

DDP 报价 = CIF + 关税 + 清关费 + 派送费
```

**谈判要点**:
| 要点 | 策略 |
|------|------|
| **价格** | 阶梯报价（量大从优） |
| **付款方式** | 30% 定金 +70% 见提单 |
| **交期** | 留有余地（工厂交期 +7 天） |
| **质保** | 1 年质保，易损件除外 |
| **样品** | 样品费可退（下单后） |

---

### 第 5 步：订单执行（2-4 周）

**执行流程**:
```
1. 收定金 → 2. 下工厂订单 → 3. 生产跟进
       ↓
4. 质量检验 → 5. 订舱报关 → 6. 发货
       ↓
7. 收尾款 → 8. 单据移交 → 9. 售后跟踪
```

**关键节点**:
| 节点 | 注意事项 |
|------|---------|
| **收定金** | 确认到账再下单 |
| **生产跟进** | 每周跟进一次进度 |
| **质量检验** | 发货前验货（可第三方） |
| **收尾款** | 见提单副本付款 |
