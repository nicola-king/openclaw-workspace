### 核心思想

#### 1. 康波周期定义

```
核心定义：
"资本主义经济存在 50-60 年的长周期"

每个周期包含：
- 上升期（25-30 年）
- 下降期（25-30 年）
```

#### 2. 历史康波（已确认 5 轮）

| 康波 | 时间 | 主导技术 | 上升期 | 下降期 |
|------|------|---------|--------|--------|
| **第一波** | 1780-1840 | 纺织、蒸汽机 | 1780-1815 | 1815-1840 |
| **第二波** | 1840-1890 | 钢铁、铁路 | 1840-1870 | 1870-1890 |
| **第三波** | 1890-1940 | 电力、化工 | 1890-1920 | 1920-1940 |
| **第四波** | 1940-1980 | 汽车、石化 | 1940-1965 | 1965-1980 |
| **第五波** | 1980-2020 | 信息技术 | 1980-2000 | 2000-2020 |
| **第六波** | 2020-2070 | AI、新能源 | 2020-2045 | 2045-2070 |

#### 3. 周期四阶段

```
每个康波包含四个阶段：

1. 繁荣 (Spring) - 15-20 年
   - 新技术扩散
   - 投资增长
   - 物价温和上涨

2. 衰退 (Summer) - 5-10 年
   - 增长放缓
   - 通胀上升
   - 金融危机

3. 萧条 (Autumn) - 10-15 年
   - 经济停滞
   - 通缩压力
   - 债务清算

4. 回升 (Winter) - 10-15 年
   - 新技术萌芽
   - 产能出清
   - 新周期准备
```

#### 4. 驱动因素

```
核心驱动：
1. 技术革命（最根本）
2. 资本积累
3. 人口变化
4. 资源发现
5. 战争与革命
```

#### 5. 价格波动

```
康波价格规律：
- 上升期：物价上涨
- 下降期：物价下跌
- 转折点：通胀/通缩转换
```

### 基本信息

---

### 主要著作

| 著作 | 年份 | 主题 |
|------|------|------|
| **《经济发展理论》** | 1911 | 创新理论奠基 |
| **《经济周期》** | 1939 | 周期理论体系 |
| **《资本主义、社会主义与民主》** | 1942 | 创造性破坏 |

---

### 核心思想（10 大观点）

#### 1. 创新理论

```
核心定义：
"创新是生产要素的新组合"

五种创新：
1. 新产品
2. 新生产方法
3. 新市场
4. 新供应来源
5. 新组织形式
```

#### 2. 企业家精神

```
核心观点：
"企业家是创新的主体"

企业家功能：
- 发现机会
- 承担风险
- 组织资源
- 实现创新

企业家利润：
- 创新带来的超额利润
- 随模仿者进入而消失
```

#### 3. 创造性破坏 (Creative Destruction)

```
核心概念：
"资本主义的本质是创造性破坏"

过程：
新技术 → 摧毁旧产业 → 创造新产业 → 经济增长

例子：
- 汽车摧毁马车业
- 数码摧毁胶卷业
- 电商摧毁零售业
```

**名言**:
```
"资本主义创造性破坏的过程是其本质特征。"
```

#### 4. 经济周期三模型

```
熊彼特三周期模型：

1. 短周期（基钦周期）- 3-4 年
   - 库存周期

2. 中周期（朱格拉周期）- 7-11 年
   - 投资周期

3. 长周期（康德拉季耶夫周期）- 50-60 年
   - 技术革命周期

关系：
1 个康波 = 6 个朱格拉 = 18 个基钦
```

#### 5. 技术革命驱动

```
核心观点：
"技术革命是长周期的根本驱动"

技术革命特点：
- 颠覆性
- 广泛影响
- 长周期
- 集群出现
```

#### 6. 经济周期阶段

```
四阶段模型：

1. 繁荣
   - 创新扩散
   - 投资高潮
   - 信贷扩张

2. 衰退
   - 创新饱和
   - 利润下降
   - 信贷收缩

3. 萧条
   - 企业破产
   - 失业上升
   - 债务清算

4. 复苏
   - 新技术出现
   - 投资恢复
   - 新周期开始
```

#### 7. 信贷与周期

```
核心观点：
"信贷扩张放大周期波动"

机制：
创新 → 信贷扩张 → 过度投资 → 泡沫 → 破裂 → 萧条
```

#### 8. 垄断与创新

```
核心观点：
"一定程度的垄断有利于创新"

论证：
- 垄断利润提供研发资金
- 竞争过度削弱创新动力
- 熊彼特悖论：竞争 vs 创新
```

#### 9. 资本主义的未来

```
核心预测：
"资本主义将自我毁灭"

原因：
- 企业家功能官僚化
- 知识分子敌视
- 家庭财富观念变化
- 民主政治压力

结论：
"资本主义将过渡到社会主义"
```

#### 10. 方法论

```
核心方法：
- 历史分析
- 动态视角
- 制度因素
- 非均衡分析
```

```
- 2004-2015：衰退期

- 2025-2045：回升期
```

#### 3. 萧条期特征

```
萧条期特点：
```

```
不同周期阶段资产配置：

繁荣期：股票 > 商品 > 债券
衰退期：债券 > 现金 > 股票
萧条期：现金 > 债券 > 股票
回升期：股票 > 商品 > 债券

2015-2025 建议：

- 配置债券
```

```
核心观点：
"大宗商品是康波最敏感指标"
```
