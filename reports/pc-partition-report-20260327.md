## ✅ 优化建议

### 高优先级（本周）

1. **清理 p1 分区（系统盘）**
   ```bash
   # 清理旧内核
   sudo apt autoremove --purge
   
   # 清理 Snap 旧版本
   sudo du -sh /snap/* | sort -h | tail -10
   
   # 清理缓存
   sudo apt clean
   ```

2. **重新规划 p3 分区**
   ```bash
   # 方案 A: 作为备用 Swap
   sudo mkswap /dev/nvme0n1p3
   sudo swapon /dev/nvme0n1p3
   
   # 方案 B: 作为备份盘
   sudo mkdir -p /media/nicola/backup
   sudo mount /dev/nvme0n1p3 /media/nicola/backup
   ```

3. **启用 p7 数据盘**
   ```bash
   # 创建数据目录
   sudo mkdir -p /data/{polymarket,logs,backups,docker}
   sudo chown -R nicola:nicola /data
   ```
