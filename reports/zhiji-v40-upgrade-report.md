## 🎯 P1 优化（已完成）

### 1. 动态仓位管理 ✅

**模式**:
- `fixed`: 固定 $0.50/笔（默认）
- `kelly`: Quarter-Kelly 比例下注

**Kelly 公式**:
```
Kelly = (p * (b+1) - 1) / b
p = 置信度，b = 赔率 -1
```

**上限控制**:
- 最大 Kelly 仓位：1%
- 最大单笔风险：0.5%
- 最小下注：$0.01
