## 🔑 核心技术

### 1. 全双工注意力机制

**原理**:
```
传统方案：听→理解→说 (串行)
PersonaPlex: 听 + 理解 + 说 (并行)

关键:
- 同时关注输入和输出序列
- 动态调整注意力权重
- 实时检测打断信号
```

**优势**:
- ✅ 延迟降低 40%
- ✅ 打断响应更快
- ✅ 对话更自然

---

### 2. 多 Token 预测

**原理**:
```
传统：每次预测 1 个 token
PersonaPlex: 每次预测 N 个 token

关键:
- 并行解码
- 置信度验证
- 错误回滚机制
```

**优势**:
- ✅ 推理速度提升 N 倍
- ✅ 延迟显著降低
- ✅ 保持生成质量

---

### 3. 联合模型架构

**传统 Pipeline**:
```
ASR 模型 → LLM → TTS 模型
   ↓        ↓       ↓
 200ms    500ms   300ms
总延迟：1000ms+
```

**PersonaPlex**:
```
单一 7B 模型
   ↓
 170ms
总延迟：170ms
```

**优势**:
- ✅ 延迟降低 83%
- ✅ 模型更小
- ✅ 部署更简单
