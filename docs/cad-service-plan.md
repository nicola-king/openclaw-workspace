## 🎯 服务定位

**服务形式**: 在线工具 + API + 定制服务

## 🚀 部署方案

### 方案 A: 本地服务 (快速启动)
**优点**: 零成本、快速上线、可控  
**缺点**: 需手动操作、规模有限

**部署步骤**:
1. ✅ LibreCAD/FreeCAD 已安装
2. 🟡 创建转换脚本 (`cad-converter.py`)
3. 🟡 配置微信/Telegram 接单通道
4. 🟡 定价/服务说明文档

**预计上线**: 1 天

---

### 方案 B: Web 服务 (中期目标)
**优点**: 自动化、可扩展、用户体验好  
**缺点**: 需服务器、开发成本

**技术栈**:
- 前端：React/Vue
- 后端：FastAPI + Python
- 处理：LibreCAD CLI + FreeCAD CLI
- 存储：MinIO/阿里云 OSS

**预计上线**: 2-4 周

---

### 方案 C: SaaS 平台 (长期愿景)
**优点**: 规模化、订阅收入、品牌效应  
**缺点**: 高投入、运维复杂

**参考竞品**:
- CADSoftTools
- AnyCAD
- CloudConvert

**预计上线**: 3-6 个月
