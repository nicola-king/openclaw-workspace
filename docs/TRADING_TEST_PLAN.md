## ⚠️ 风险控制

### 止损机制

**硬止损**:
```
✅ 单笔交易止损：5%
✅ 单日总止损：10%
✅ 单周总止损：20%
✅ 单月总止损：30%
```

**动态止损**:
```
✅ 追踪止损：盈利后回撤 50% 平仓
✅ 时间止损：持仓>7 天无盈利平仓
✅ 事件止损：黑天鹅事件立即平仓
```

### 资金管理

**仓位计算**:
```python
position_size = (account_balance * risk_per_trade) / (entry_price - stop_loss)
```

**参数**:
- `risk_per_trade`: 1-2% (保守)
- `max_position`: 10% (单币种)
- `max_leverage`: 5x (最大)

## 📈 性能评估

### 核心指标

| 指标 | 目标 | 计算方式 |
|------|------|---------|
| 月收益率 | >15% | (月末 - 月初) / 月初 |
| 最大回撤 | <10% | max(峰值 - 谷值) / 峰值 |
| 夏普比率 | >2.0 | (收益 - 无风险) / 波动率 |
| 胜率 | >55% | 盈利交易 / 总交易 |
| 盈亏比 | >2.0 | 平均盈利 / 平均亏损 |

**每日**:

## 📝 应急处理

### 应急预案

**场景 1: 系统故障**
```
1. 立即暂停所有交易
2. 检查系统日志
3. 修复问题
4. 模拟测试验证
5. 逐步恢复交易
```

**场景 2: 极端行情**
```
1. 触发硬止损
2. 全部平仓
3. 等待市场稳定
4. 分析原因
5. 调整策略参数
```

**场景 3: 黑天鹅事件**
```
1. 立即平仓
2. 暂停交易
3. 评估影响
4. 调整风控
5. 逐步恢复
```
