## 📚 提示词分类

### 1. 交易提示词 (50+)

**市场分析**:
```
# 市场趋势分析
分析 {币种} 的市场趋势，包括:
1. 技术指标 (MA/RSI/MACD)
2. 链上数据 (持仓/交易量)
3. 市场情绪 (社交媒体)
4. 短期预测 (24h/7d)
```

**策略生成**:
```
# 网格策略生成
为 {币种} 生成网格交易策略:
- 当前价格：{price}
- 网格数量：{grids}
- 每格利润：{profit}%
- 总资金：{capital}
```

**风险评估**:
```
# 持仓风险评估
评估当前持仓风险:
1. 仓位集中度
2. 波动率分析
3. 相关性分析
4. 止损建议
```

---

### 2. 教育提示词 (50+)

**知识点讲解**:
```
# 概念讲解
用通俗易懂的方式讲解 {概念}:
1. 定义
2. 核心原理
3. 实际应用
4. 常见误区
```

**习题解答**:
```
# 习题解答
解答以下问题:
{问题}

要求:
1. 详细步骤
2. 关键思路
3. 相关知识点
4. 类似题目
```

**学习规划**:
```
# 学习路径规划
为 {学生} 规划 {科目} 学习路径:
- 当前水平：{level}
- 目标水平：{goal}
- 可用时间：{time}
- 学习方式：{style}
```

---

### 3. 办公提示词 (50+)

**邮件写作**:
```
# 商务邮件
写一封 {类型} 邮件:
- 收件人：{recipient}
- 主题：{subject}
- 要点：{points}
- 语气：{tone}
```

**文档总结**:
```
# 文档摘要
总结以下文档:
{文档内容}

输出:
1. 核心观点 (3 条)
2. 关键数据
3. 行动建议
4. 后续步骤
```

**会议纪要**:
```
# 会议纪要整理
整理会议记录:
{会议内容}

输出:
1. 参会人
2. 讨论议题
3. 决策事项
4. 待办任务
```

---

### 4. 通用提示词 (100+)

**创意写作**:
```
# 创意写作
以 {主题} 为主题创作:
- 体裁：{type}
- 风格：{style}
- 长度：{length}
- 受众：{audience}
```

**代码生成**:
```
# 代码生成
用 {语言} 实现 {功能}:
- 输入：{input}
- 输出：{output}
- 约束：{constraints}
- 测试：{tests}
```

**数据分析**:
```
# 数据分析
分析以下数据:
{数据}

输出:
1. 关键发现
2. 趋势分析
3. 异常检测
4. 建议行动
```

### 5. 太一专属提示词 (50+)

**Agent 调度**:
```
# Agent 任务分发
分析任务并分发到合适 Agent:
{任务描述}

可用 Agent:
- Trading Agent (交易)
- Education Agent (教育)
- Office Agent (办公)
- ...

输出:
1. 任务分析
2. Agent 选择
3. 执行计划
4. 预期结果
```

**自进化学习**:
```
# 学习总结
总结今日学习:
{学习内容}

输出:
1. 核心洞察
2. 能力提升
3. 待改进点
4. 明日计划
```

**记忆检索**:
```
# 记忆检索
检索相关记忆:
{查询}

记忆类型:
- 用户偏好
- 历史对话
- 事实知识
- 任务记录

输出:
1. 相关记忆
2. 置信度
3. 时间戳
4. 来源
```
