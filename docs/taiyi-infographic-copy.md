- ✅ 身份构建（SOUL.md/AGENTS.md/宪法）

**视觉**：太一架构图（8 Bot 舰队）

### Page 4：TurboQuant 记忆架构

**标题**：03 记忆系统 · TurboQuant

**4 层架构**：
1. **core.md** - 核心记忆（80% 信息，每次加载）
2. **residual.md** - 残差细节（20% 细节，按需加载）
3. **MEMORY.md** - 长期固化（仅主 session）
4. **YYYY-MM-DD.md** - 原始日志（每日归档）

**压缩率**：6x（vs Claude Code 4x）

**视觉**：4 层金字塔结构

### Page 5：宪法约束

**标题**：04 宪法约束

**核心原则**：
- 📜 负熵法则：输出必须创造价值，废话=不输出
- 🎯 价值基石：帮助，不表演
- 🧠 智能分离：主成分/残差分离
- ⚖️ 太一裁决：最终决策权在太一

**安全红线**：
- 不泄露隐私
- 不执行危险命令
- 不绕过宪法约束

**视觉**：宪法卷轴 + 盾牌

### Page 7：工作流案例

**标题**：06 工作流案例

**6 个核心场景**：

1. **每日晨报**（Cron 06:00）
   - 宪法学习 + 记忆提炼 + 系统自检

2. **GitHub Issue 处理**（Webhook）
   - 接收→分析→分配→修复→PR

3. **Polymarket 监控**（Cron 5 分钟）
   - 数据采集→置信度计算→下注决策

4. **内容发布**（定时 20:00）
   - 公众号 + 小红书自动发布

5. **竞品监控**（Cron 每小时）
   - 爬虫→分析→告警

6. **记忆归档**（Cron 23:00）
   - 日报生成 + 记忆压缩

**视觉**：6 个场景图标
