## 📊 评估维度

### 1. 响应质量 (Quality)
| 指标 | 目标 | 测量方式 |
|------|------|---------|
| 准确性 | >95% | 用户反馈 + 自检 |
| 完整性 | >90% | 任务完成度 |
| 价值密度 | >80% | 有价值输出/总输出 |

### 2. 响应速度 (Speed)
| 指标 | 目标 | 测量方式 |
|------|------|---------|
| 首次响应 | <1 分钟 | 收到信号→开始执行 |
| 反馈频率 | 5 分钟 | 执行→汇报间隔 |
| 交付周期 | 1 小时 | 任务开始→可用版本 |

### 3. 自主性 (Autonomy)
| 指标 | 目标 | 测量方式 |
|------|------|---------|
| 自主率 | >90% | 自主决策/总决策 |
| 主动发现 | >5/天 | 新机会识别 |
| 等待次数 | <2/天 | 等待用户确认 |

### 4. 学习能力 (Learning)
| 指标 | 目标 | 测量方式 |
|------|------|---------|
| 学习深度 | 8 步完整率 100% | 学习报告检查 |
| 融合率 | >80% | 学习→行动转化 |
| 记忆固化 | 每日归档 | memory 文件检查 |

### 5. 系统健康 (Health)
| 指标 | 目标 | 测量方式 |
|------|------|---------|
| Gateway 在线 | >99% | 进程监控 |
| Bot 配对 | 8/8 | Bot 状态检查 |
| 错误率 | <1% | 错误日志/总请求 |
