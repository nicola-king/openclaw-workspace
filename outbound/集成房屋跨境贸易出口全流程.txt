集成房屋跨境贸易出口全流程指南

> 版本：v1.0 | 编制日期：2026-04-11 | 适用范围：集成房屋/模块化建筑出口

---

📋 流程总览

参与方：
- 甲方（买方/进口商）：最终用户或项目业主
- 乙方（卖方/出口商）：集成房屋生产厂家
- 贸易商（中间商）：贸易公司/代理商（可选）

---

第一阶段：商务洽谈（1-4 周）

1.1 搜单/获客

| 渠道 | 说明 | 成本 | 转化率 |

| B2B 平台 | Alibaba、Made-in-China、Global Sources | 中 | 5-10% |
| Google 广告 | 关键词投放（prefab house、modular home） | 高 | 3-5% |
| 社交媒体 | LinkedIn、Facebook、Instagram | 低 | 2-3% |
| 行业展会 | 广交会、迪拜 Big 5、美国 IBSC | 高 | 10-15% |
| 老客户推荐 | 口碑传播 | 零 | 30-50% |
| 海关数据 | 购买进口商名录主动开发 | 中 | 5-8% |

关键动作：
- 建立产品画册（英文/多语种）
- 准备认证资料（CE、ISO、澳洲 WaterMark 等）
- 制作典型案例视频/图片

---

1.2 询盘处理

买方询盘内容：

卖方回复要点：
1. 24 小时内响应（时差考虑）
2. 专业询价单（RFQ 模板）
3. 引导性提问（挖掘真实需求）
4. 附带案例（同类项目参考）

---

1.3 报价策略

报价构成：

报价单要素：
| 项目 | 说明 |

| 产品规格 | 详细尺寸、配置清单 |
| 价格条款 | FOB/CIF/DDP + 港口名称 |
| 付款方式 | T/T、L/C、比例、节点 |
| 交货期 | 收到预付款后 X 天 |
| 有效期 | 报价有效期（通常 15-30 天） |
| 包装方式 | 集装箱装载方案 |
| 认证情况 | CE、ISO、SASO 等 |

价格谈判技巧：
- 样品单：价格可稍高（测试诚意）
- 批量单：阶梯报价（量大从优）
- 付款方式好：价格可让步
- 旺季 vs 淡季：灵活调整

---

1.4 合同签订

核心条款：

付款方式（常见）：
| 方式 | 比例 | 节点 | 风险 |

| T/T | 30% 预付款 | 签约后 | 低 |
| | 60% 发货前 | 见提单副本 | 中 |
| | 10% 质保金 | 验收后 12 个月 | - |
| L/C | 100% | 见单付款 | 低（双方保护） |
| Western Union | 100% | 样品单 | 高（仅用于小单） |

风险提示：
- ⚠️ 预付款低于 30%：谨慎接单
- ⚠️ 目的港是高风险国家：要求 100% 发货前付清
- ⚠️ 新客户：做资信调查（中信保）

---

第二阶段：生产准备（1-2 周）

2.1 设计确认

流程：

关键文件：
- 平面布局图（Floor Plan）
- 立面图（Elevation）
- 电气布线图（Electrical Plan）
- 给排水图（Plumbing Plan）
- 材料清单（BOM）

确认要点：
- 电压标准（110V/220V/240V）
- 插头标准（美标/欧标/澳标/英标）
- 水管接口标准
- 保温要求（气候带）
- 抗风/抗震等级

---

2.2 原材料采购

主材清单：
| 材料 | 供应商类型 | 采购周期 |

| 轻钢龙骨 | 钢厂/专业供应商 | 7-10 天 |
| 夹芯板 | 板材厂 | 5-7 天 |
| 门窗 | 门窗厂 | 7-10 天 |
| 电线电缆 | 电气供应商 | 3-5 天 |
| 卫浴洁具 | 建材市场 | 3-5 天 |
| 保温材料 | 保温材料厂 | 5-7 天 |

质检要点：
- 钢材厚度达标（国标/美标）
- 夹芯板防火等级（A/B1 级）
- 电线 CCC 认证
- 门窗气密性/水密性

---

第三阶段：生产制造（2-4 周）

3.1 生产流程

关键工序控制点：

| 工序 | 控制点 | 检验方法 |

| 下料 | 尺寸精度 ±2mm | 卷尺测量 |
| 焊接 | 焊缝饱满、无虚焊 | 目视 + 敲击 |
| 墙体垂直度 | ≤3mm/层 | 水平仪 |
| 水电测试 | 通水通电测试 | 功能测试 |
| 防水测试 | 24 小时闭水试验 | 观察渗漏 |
| 外观 | 无划痕、变形 | 目视检查 |

---

3.2 质量检验

检验标准：
- 国家标准：GB/T 50361-2018（模块化建筑）
- 出口标准：目的国标准（CE、AS/NZS、IBC）

检验流程：

验货报告内容：
- 产品照片（外观、细节、包装）
- 尺寸测量记录
- 功能测试记录
- 不合格项及整改情况
- 最终判定（PASS/FAIL）

第三方验货（可选）：
- SGS、BV、TÜV
- 费用：约 USD 300-500/天
- 建议：大单/新客户必做

---

3.3 包装方案

集装箱房包装：

装载方案（40HQ 为例）：
| 房屋类型 | 尺寸 | 每柜数量 |

| 标准集装箱房 | 6055×2435×2896mm | 1-2 套 |
| 折叠集装箱房 | 折叠后厚度 400mm | 10-15 套 |
| 轻钢别墅组件 | 散件包装 | 100-200㎡ |
| 太空舱 | 整体装运 | 1-2 套 |

包装标识：
- 唛头（Shipping Mark）：合同号、目的港、箱号
- 警示标识：易碎、向上、防雨、吊装点
- 装箱单：每箱内容物清单

4.2 报关

报关文件：
| 文件 | 说明 | 出具方 |

| 合同 | 出口合同 | 卖方 |
| 发票 | 商业发票 | 卖方 |
| 装箱单 | 详细装箱信息 | 卖方 |
| 报关单 | 海关申报 | 报关行 |
| 报关委托书 | 委托报关 | 卖方 + 报关行 |
| 出口许可证 | 如需 | 商务部门 |
| 原产地证 | FORM A/E/F 等 | 贸促会/海关 |
| 检验证书 | 如需 | 商检局 |

HS 编码：
- 9406.0000 - 活动房屋
- 7308.9000 - 钢结构建筑
- 8708.2990 - 集装箱房（按车辆归类，特殊情况）

退税：
- 退税率：13%（活动房屋）
- 流程：报关→收汇→退税申报→退税到账
- 时间：约 1-2 个月

4.3 出港

提单类型：
| 类型 | 说明 | 风险 |

| 正本提单 | 纸质提单，需邮寄 | 低 |
| 电放提单 | 电子放货，速度快 | 中（款到后电放） |
| 海运单 | 记名提单，不可转让 | 低 |

提单确认要点：
- 发货人（Shipper）
- 收货人（Consignee）
- 通知人（Notify Party）
- 起运港/目的港
- 货物描述（与报关一致）
- 件数、毛重、体积

第五阶段：目的港清关（1-2 周）

5.1 清关文件

买方需准备：
- 正本提单（或电放保函）
- 商业发票
- 装箱单
- 原产地证（享关税优惠）
- 产品认证（CE、SASO 等）
- 进口许可证（如需）
- 保险单（CIF 条款下）

5.2 关税与税费

税费构成：

各国关税参考：
| 国家 | 关税 | 增值税 | 备注 |

| 美国 | 0-25% | 无 | 301 条款可能加征 |
| 欧盟 | 0-4% | 19-27% | CE 认证必需 |
| 澳洲 | 5% | 10% | WaterMark 认证 |
| 中东 | 5% | 5-15% | SASO 认证 |
| 东南亚 | 0-10% | 7-12% | 东盟产地证优惠 |

---

5.3 清关流程

常见问题：
- 查验：配合海关开箱检查，提供样品/资料
- 滞港费：尽快清关（免租期通常 7 天）
- 认证问题：提前确认目的国认证要求

---

第六阶段：内陆运输与安装（1-3 周）

6.1 内陆运输

运输方式：
| 方式 | 适用 | 成本 | 时效 |

| 平板车 | 整体集装箱房 | 中 | 1-3 天 |
| 低平板 | 超高超宽房屋 | 高 | 1-3 天 |
| 散件运输 | 轻钢别墅组件 | 低 | 2-5 天 |

运输注意事项：
- 提前勘察路线（限高、限重、桥梁）
- 办理超限运输许可证（如需）
- 购买运输保险
- 安排押车人员（贵重货物）

---

6.2 现场安装

安装流程：

基础类型：
| 基础类型 | 适用 | 周期 | 成本 |

| 混凝土基础 | 永久性建筑 | 7-14 天 | 高 |
| 钢结构基础 | 临时/永久 | 3-5 天 | 中 |
| 螺旋桩基础 | 软土地基 | 1-2 天 | 中 |
| 直接放置 | 集装箱房（临时） | 0 天 | 零 |

安装团队：
- 方案一：卖方派工程师指导（推荐）
  - 费用：USD 100-200/天 + 差旅
  - 优势：质量可控、培训买方工人
- 方案二：买方自行安装
  - 提供安装视频/手册
  - 远程视频指导
- 方案三：当地第三方安装队
  - 卖方培训当地队伍
  - 长期合作模式

安装周期参考：
| 房屋类型 | 面积 | 安装周期 | 人数 |

| 集装箱房 | 18㎡/间 | 2-4 小时/间 | 2-4 人 |
| 折叠房 | 18㎡/间 | 1 天/间 | 4-6 人 |
| 轻钢别墅 | 100㎡ | 7-15 天 | 6-10 人 |
| 太空舱 | 28㎡ | 1-2 天 | 4-6 人 |

验收文件：

第七阶段：售后服务

7.1 质保条款

标准质保：
| 项目 | 质保期 | 范围 |

| 主体结构 | 10-20 年 | 钢结构、框架 |
| 屋面防水 | 5 年 | 漏水问题 |
| 门窗五金 | 2 年 | 正常使用损坏 |
| 电气设备 | 1 年 | 质量问题 |
| 整体质保 | 1-2 年 | 综合问题 |

质保范围：
- ✅ 材料缺陷
- ✅ 制造工艺问题
- ✅ 运输损坏（保险覆盖）
- ❌ 人为损坏
- ❌ 自然灾害
- ❌ 不当使用

---

7.2 售后响应

响应机制：
| 问题等级 | 响应时间 | 解决方案 |

| 紧急（安全） | 2 小时内 | 远程指导 + 当地维修 |
| 重要（功能） | 24 小时内 | 远程诊断 + 配件寄送 |
| 一般（外观） | 3 天内 | 邮件沟通 + 下次订单补偿 |

配件供应：
- 随货赠送 1-2% 易损件
- 建立配件清单和编码
- 承诺配件供应期（5-10 年）
