## 注册步骤

### 1. 访问官网

```
https://console.anthropic.com/
```

### 2. 注册账号

| 字段 | 填写内容 |
|------|----------|
| **邮箱** | 你的邮箱 |
| **密码** | 设置密码 |
| **手机号** | 验证手机 |

### 3. 绑定支付方式

- 信用卡/借记卡
- 按量付费或订阅

### 4. 获取 API Key

```
Dashboard → API Keys → Create Key
复制并保存（只显示一次）
```
