## 📊 技术流程（4 步）

### 1. 语料准备
- **来源**：多个 AI（Gemini 3 Pro + Claude Opus 4.5）情感对话
- **清洗**：得到约 2300 条高质量 instruction-response 训练对
- **格式**：Alpaca 格式
- **平均长度**：每条 AI 回复 1100 多字符
- **总量**：约 2.5M 字符训练数据

### 2. 底模选择
- **模型**：Google Gemma 4 E4B（4B 参数）
- **选择原因**：
  - 4B 是硬件能支持的本地流畅推理上限
  - Gemma 4 是从 Gemini 3 蒸馏出来的（血统优势）
- **优势**：小模型 + 高质量蒸馏 = 本地可用

### 3. 微调方式
- **平台**：Colab Pro + A100 GPU
- **工具**：Unsloth 跑 QLoRA
- **时间**：大约一小时跑完
- **训练效果**：Training loss 稳定在 2.1-2.2，没有过拟合
- **成本**：Colab Pro $10/月

### 4. 导出部署
- **格式**：合并 LoRA adapter 到 base model
- **量化**：GGUF 格式（Q4_K_M 量化）
- **文件大小**：最终 10GB
- **运行**：下载到 Mac，用 Ollama 注册并运行
