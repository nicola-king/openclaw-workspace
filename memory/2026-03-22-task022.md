## 工作内容

### 1. 占地面积利用率计算规则（P0 聚焦）

```
占地面积利用率 = 建筑物占地面积 / 用地总面积 × 100%

相关指标：
- 建筑密度 = 建筑基底面积 / 用地面积 × 100%
- 容积率 = 总建筑面积 / 用地面积
- 绿化率 = 绿化面积 / 用地面积 × 100%
- 道路率 = 道路面积 / 用地面积 × 100%
```

### 2. 标准化计算公式

```markdown
## 用地面积
- 用地边界（闭合多段线）→ 几何面积计算
- 单位转换：CAD 单位 → 平方米（需比例尺）

## 建筑占地面积
- 建筑轮廓（闭合多段线/填充区域）→ 面积
- 多层建筑：仅计算基底面积（非楼层总和）

## 利用率计算
- 公式：利用率 = 建筑占地 / 用地总面积 × 100%
- 规范参考：《城市居住区规划设计标准》GB50180-2018
```
