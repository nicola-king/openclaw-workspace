## 🔍 Telegram 媒体下载失败 - 根本原因

**时间**: 2026-03-29 22:34-22:37
**状态**: 🔍 已诊断，待修复
**负责**: 素问

---

### 问题现象

```
⚠️ Failed to download media. Please try again.
```

---

### 根本原因

**不是** Telegram Gateway 掉包问题
**而是** OpenClaw URL Fetch 安全限制

**错误日志**:
```
blocked URL fetch (url-fetch) target=https://api.telegram.org/file/bot...
reason=Blocked: resolves to private/internal/special-use IP address
```

---

### 修复方案

**需要**: 将 `api.telegram.org` 加入 URL Fetch 白名单

**配置文件**: `~/.openclaw/credentials/url-fetch-allowlist.json`

**白名单内容**:
```json
{
  "allowlist": [
    "api.telegram.org",
    "*.telegram.org",
    "generativelanguage.googleapis.com",
    "coding.dashscope.aliyuncs.com"
  ]
}
```

---

### 执行命令

```bash
mkdir -p ~/.openclaw/credentials
cat > ~/.openclaw/credentials/url-fetch-allowlist.json << 'EOF'
{
  "allowlist": [
    "api.telegram.org",
    "*.telegram.org",
    "generativelanguage.googleapis.com",
    "coding.dashscope.aliyuncs.com"
  ]
}
EOF
openclaw gateway restart
```
