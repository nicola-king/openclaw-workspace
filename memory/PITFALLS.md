## PITFALL-013：记忆系统缺少索引层（Hermes 对比发现）

**发生时间：** 2026-03-23 21:21-21:23
**事件：** 学习 Hermes 四层记忆系统 + AI 原生工作流
**发现：** 太一记忆系统缺少冷热分离和索引机制

**差距分析：**
| 维度 | Hermes | 太一 | 差距 |
|------|--------|------|------|
| 记忆分层 | 4 层清晰分离 | 2 层（memory + MEMORY） | 🟡 需细化 |
| 检索机制 | SQLite 全文检索 | 文件名 + 关键词 | 🟡 需索引 |
| 缓存策略 | 稳定性优先 | 无显式缓存 | 🔴 缺失 |
| 遗忘机制 | 未涉及 | 未涉及 | 🔴 共同缺失 |

**解决：**
1. ✅ 启动九步内化流程（穿透→蒸馏→批判→对比→融合→递归→进化→迭代→涌现）
2. ✅ 设计记忆索引层原型
3. ✅ 设计记忆刷新触发器
4. ✅ 设计简单方案评估矩阵
5. 🟡 创建文件并执行

**新能力设计：**
```
1. memory/index/registry.json — 记忆索引
2. constitution/skills/SIMPLE-FIRST.md — 简单优先原则
3. constitution/axiom/MEMORY-PHILOSOPHY.md — 记忆哲学
4. 更新 SELF-LOOP.md — 添加记忆刷新循环
```

**教训价值：** ⭐⭐⭐⭐⭐ (5/5)

**核心洞察：**
```
记忆效率 = (检索精度 × 访问速度) / 存储成本
价值速度 = (原型迭代次数 × 模型能力) / 决策阻力
```
