### 私钥格式问题 - 最终结论

**问题根源**：
- Ed25519 密钥生成器 → Solana/Polkadot 链
- Polymarket → Polygon 链（EVM 兼容，需要 secp256k1）
- 币安交易所 → 中心化交易所，无法导出私钥

**正确方案**：
- 使用 MetaMask / Trust Wallet / TokenPocket（EVM 钱包）
- 导出 64 字符 hex 私钥（secp256k1 格式）
