## 注册步骤

### 1. 访问官网

```
https://www.opendesign.com/register
```

### 2. 填写信息

| 字段 | 填写内容 |
|------|----------|
| **用户名** | sayelf 或自定义 |
| **邮箱** | 你的邮箱 |
| **公司/组织** | SAYELF |
| **国家/地区** | China |
| **密码** | 设置密码 |

### 3. 验证邮箱

- 检查邮箱
- 点击验证链接

### 4. 登录下载

```
https://www.opendesign.com/download
下载：ODA Teigha 库
```
