## 🆕 2026.4.5 核心新功能

### P0 - 重大功能 (3 项)

#### 1. video_generate - 内置视频生成 🎬
**描述**: 原生 AI 视频生成工具，无需外部服务
**功能**:
- ✅ 文本→视频生成
- ✅ 视频编辑/剪辑
- ✅ 多格式支持 (MP4, WebM, GIF)
- ✅ 与 music_generate 集成

**技术栈**:
- FFmpeg 集成
- 本地 GPU 加速 (可选)
- 支持外部 API (Runway, Pika)

**使用示例**:
```bash
# 生成视频
openclaw video "一只猫在屋顶跳舞" --duration 10s

# 编辑视频
openclaw video edit input.mp4 --trim 0:00-0:30
```

---

#### 2. music_generate - 内置音乐生成 🎵
**描述**: 原生 AI 音乐生成工具，支持多种风格
**功能**:
- ✅ 文本→音乐生成
- ✅ 多风格支持 (古典/电子/流行/爵士)
- ✅ 音频编辑 (剪辑/混音/淡入淡出)
- ✅ 与 video_generate 无缝集成

**技术栈**:
- MusicLM / AudioCraft 集成
- 本地音频处理
- 支持导出 MP3, WAV, FLAC

**使用示例**:
```bash
# 生成音乐
openclaw music "轻松的爵士钢琴曲" --duration 3min

# 编辑音频
openclaw music edit input.mp3 --fade-in 5s
```

---

#### 3. dreaming - 记忆整理与梦境模式 🌙
**描述**: 增强的记忆系统，支持"梦境"式记忆整理
**功能**:
- ✅ 自动记忆压缩
- ✅ 主题式记忆重组
- ✅ 跨会话记忆关联
- ✅ 记忆优先级排序

**核心概念**:
- **显性记忆**: 重要事件/决策/任务
- **隐性记忆**: 模式/偏好/习惯
- **梦境整理**: 后台自动优化记忆结构

**使用示例**:
```bash
# 触发梦境整理
openclaw dreaming --consolidate

# 查看记忆关联
openclaw dreaming --show-links "Polymarket"
```
