## 💡 使用流程

### 新项目启动
1. 复制 `project-template/` 到 `projects/[项目名称]/`
2. 填写项目基本信息
3. 引用定额库文件
4. 应用标准模板

### 项目进行中
1. 预算文件使用定额库数据
2. 变更签证及时归档
3. 材料价格参考造价标准

### 项目完成后
1. 整理全部文件
2. 归档到 `archives/[年份]/`
3. 更新项目数据库

## 🔍 定额共享机制

### 共享原则
- ✅ 定额库集中管理
- ✅ 所有项目统一引用
- ✅ 避免重复存储
- ✅ 版本统一管理

### 引用方式
```
项目预算书 → 引用 → 定额库/重庆 2018/建筑工程
材料价格 → 引用 → 造价标准/材料价格库
费用计算 → 引用 → 定额库/配套文件/费用定额
```

## ⚠️ 注意事项

### 文件命名
- 统一格式：`日期_内容_版本`
- 示例：`20260326_预算书_v1.0.xlsx`

### 版本控制
- 初稿：`v0.1`
- 审核稿：`v1.0`
- 终稿：`v1.0-FINAL`

### 备份策略
- 本地备份：实时
- 云端备份：每日
- 归档备份：项目完成后
