```
1. 用 Excel 打开 templates/变更签证/变更台账总表 - 模板.csv

3. 阅读 Excel 公式复制手册.md
```

## 📋 使用流程

### 日常使用
```
每天:
├─ 打开变更台账.xlsm
├─ 录入新增变更
├─ 更新审批状态
└─ 查看预警看板

每周五:
├─ 运行"生成周报"宏
├─ 审核周报内容
└─ 发送给项目团队

每月末:
├─ 运行"生成月报"宏
├─ 审核月报内容
├─ 发送给业主/公司
└─ 运行"自动备份数据"
```

### 预警处理
```
发现🟠🔴预警:
├─ 查看预警清单
├─ 分配责任人
├─ 跟踪处理进度
├─ 超 7 天→商务经理催办
├─ 超 14 天→项目经理协调
└─ 超 21 天→上报业主分管领导
```
