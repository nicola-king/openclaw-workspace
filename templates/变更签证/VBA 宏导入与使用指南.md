## 📥 导入 VBA 宏

### 方法 1: 直接导入（推荐）

#### 步骤 1: 打开 VBA 编辑器
```
Excel 中按 Alt + F11
或：开发工具 → Visual Basic
```

#### 步骤 2: 导入模块
```
1. 在 VBA 编辑器中，右键点击左侧项目窗口
2. 选择"导入文件..."
3. 选择文件：templates/变更签证/变更台账自动化宏.bas
4. 点击"打开"
```

#### 步骤 3: 确认导入成功
```
展开"模块"文件夹
应该看到"变更台账自动化宏"模块
双击可查看代码
```

## 🔧 启用宏

### Excel 设置
```
1. 文件 → 选项 → 信任中心
2. 信任中心设置 → 宏设置
3. 选择"禁用所有宏，并发出通知"
4. 勾选"信任对 VBA 工程对象模型的访问"
5. 确定
```

### 保存为启用宏的工作簿
```
1. 文件 → 另存为
2. 文件类型选择："Excel 启用宏的工作簿 (*.xlsm)"
3. 保存
```

⚠️ **重要**: 必须保存为 `.xlsm` 格式，否则宏会丢失！

## ⏰ 设置定时任务

### 自动执行配置

#### 每日上午 9 点自动刷新
```vba
' 代码已包含在"设置定时任务"宏中
Application.OnTime TimeValue("09:00:00"), "一键刷新所有"
```

#### 每周五下午 5 点自动生成周报
```vba
' 代码已包含在"设置定时任务"宏中
If Weekday(Date) = vbFriday Then
    Application.OnTime TimeValue("17:00:00"), "生成周报"
End If
```

#### 每月最后一天自动生成月报
```vba
' 代码已包含在"设置定时任务"宏中
If Day(Date) = Day(DateSerial(Year(Date), Month(Date) + 1, 0)) Then
    Application.OnTime TimeValue("17:00:00"), "生成月报"
End If
```

### 启用定时任务
```
1. 按 Alt + F8
2. 选择"设置定时任务"
3. 点击"运行"
4. 关闭 Excel 时定时任务会取消
5. 下次打开需重新运行
```

⚠️ **注意**: 定时任务需要 Excel 保持打开状态

## ⚠️ 常见问题

### Q1: 宏无法运行，提示"宏被禁用"
**解决**: 
```
1. 文件 → 选项 → 信任中心 → 宏设置
2. 选择"禁用所有宏，并发出通知"
3. 重新打开文件
4. 点击"启用内容"
```

### Q2: 运行时提示"对象变量未设置"
**解决**:
```
检查工作表名称是否正确
确保"1️⃣台账总表"存在
检查 Outlook 是否已安装（邮件功能）
```

### Q3: 邮件无法发送
**解决**:
```
1. 检查 Outlook 是否已安装并配置
2. 检查代码中收件人邮箱是否正确
3. 邮件默认使用.Display 模式（预览）
4. 如需自动发送，将.Display 改为.Send
```

### Q4: 备份文件找不到
**解决**:
```
检查当前文件夹下是否有"备份"子文件夹
如果没有，手动创建
或运行一次"自动备份数据"宏会自动创建
```

### Q5: 定时任务不执行
**解决**:
```
1. 确认定时任务已设置（运行"设置定时任务"）
2. Excel 必须保持打开状态
3. 检查系统时间是否正确
4. 定时任务在关闭 Excel 后会取消
```
