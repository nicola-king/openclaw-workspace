## 🔧 核心公式（复制到第 2 行，然后向下填充）

### J2 核减金额
```excel
=H2-I2
```

### K2 核减率
```excel
=IF(H2=0,0,J2/H2)
```
**格式**: 百分比，保留 2 位小数

### N2 审批天数
```excel
=IF(AND(M2<>"",E2<>""),M2-E2,"")
```

### O2 超时天数
```excel
=IF(L2="已批",IF(N2>VLOOKUP(D2,系统设置!$A$2:$B$5,2,FALSE),N2-VLOOKUP(D2,系统设置!$A$2:$B$5,2,FALSE),0),TODAY()-E2-VLOOKUP(D2,系统设置!$A$2:$B$5,2,FALSE))
```

**简化版**（如果不使用系统设置表）：
```excel
=IF(L2="已批",IF(N2>7,N2-7,0),TODAY()-E2-7)
```

### P2 预警级别
```excel
=IF(O2<=3,"🟢正常",IF(O2<=7,"🟡关注",IF(O2<=14,"🟠预警","🔴警报")))
```

### R2 证据等级
```excel
=IF(Q2>=90,"🟢完整",IF(Q2>=70,"🟡基本完整",IF(Q2>=50,"🟠高风险","🔴缺失")))
```

### A2 序号（自动编号）
```excel
=ROW()-1
```

## 🎨 条件格式设置

### 预警级别列 (P 列) 颜色

1. 选中 P2:P1000
2. 开始 → 条件格式 → 新建规则 → 使用公式
3. 添加以下规则：

| 规则 | 公式 | 格式 |
|------|------|------|
| 🟢正常 | `=$P2="🟢正常"` | 绿色填充 |
| 🟡关注 | `=$P2="🟡关注"` | 黄色填充 |
| 🟠预警 | `=$P2="🟠预警"` | 橙色填充 |
| 🔴警报 | `=$P2="🔴警报"` | 红色填充，白色文字 |

### 证据等级列 (R 列) 颜色

1. 选中 R2:R1000
2. 开始 → 条件格式 → 新建规则 → 使用公式
3. 添加以下规则：

| 规则 | 公式 | 格式 |
|------|------|------|
| 🔴缺失 | `=$R2="🔴缺失"` | 红色填充，红色文字 |

### 超时天数列 (O 列) 颜色

1. 选中 O2:O1000
2. 开始 → 条件格式 → 新建规则 → 单元格值
3. 添加以下规则：

| 规则 | 条件 | 格式 |
|------|------|------|
| 正常 | 单元格值 <= 7 | 绿色文字 |
| 关注 | 单元格值 > 7 且 <= 14 | 橙色文字 |
| 警报 | 单元格值 > 14 | 红色文字，加粗 |

## 🔐 工作表保护

### 保护公式不被修改
1. 选中所有单元格 → 右键 → 设置单元格格式 → 保护 → 取消勾选"锁定"
2. 选中公式单元格 (J/K/N/O/P/R 列) → 右键 → 设置单元格格式 → 保护 → 勾选"锁定"
3. 审阅 → 保护工作表 → 设置密码

### 允许用户编辑的区域
- 变更名称、位置、原因、申报金额、审批金额、审批状态、审批日期、证据完整性

### 禁止用户编辑的区域
- 所有公式列 (J/K/N/O/P/R)、序号列

## ⚠️ 常见问题

### Q1: 公式显示为文本而不是结果
**解决**: 选中单元格 → 按 F2 → 按 Enter，或设置单元格格式为"常规"

### Q2: 条件格式不生效
**解决**: 检查公式中的单元格引用是否正确，确保使用$符号

### Q3: 下拉菜单不显示
**解决**: 数据 → 数据验证 → 检查序列来源是否正确

### Q4: TODAY() 函数不更新
**解决**: 按 F9 强制刷新，或关闭重开文件

### Q5: 文件过大运行缓慢
**解决**: 删除 1000 行以后的空行，或归档历史数据
