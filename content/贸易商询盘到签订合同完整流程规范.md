## 📥 第一阶段：询盘接收与评估 (1 天)

### 1.1 询盘接收标准流程

```
客户发送询盘
    ↓
系统自动记录 (CRM)
    ↓
销售初审 (30 分钟内)
    ↓
分类标记 (优先级)
    ↓
分配负责人
    ↓
详细评估
    ↓
回复客户
```

### 1.2 询盘登记表

| 字段 | 必填 | 说明 | 示例 |
|------|------|------|------|
| 询盘编号 | ✅ | 唯一标识 | INQ-20260411-001 |
| 接收时间 | ✅ | 精确到分钟 | 2026-04-11 10:30 |
| 客户名称 | ✅ | 公司全称 | ABC Trading Ltd. |
| 联系人 | ✅ | 姓名 + 职位 | John Smith, Purchaser |
| 邮箱 | ✅ | 企业邮箱优先 | john@abctrading.com |
| 电话 | ✅ | 含国家代码 | +1-234-567-8900 |
| 国家/地区 | ✅ | 完整名称 | United States |

| 数量 | ✅ | 明确数字 | 50 sets |
| 目标价格 | ⭕ | 如有 | USD 10-12/pc |
| 期望交期 | ⭕ | 如有 | 30 days |

| 负责人 | ✅ | 销售姓名 | 张三 |
| 处理状态 | ✅ | 待处理/已回复 | 待处理 |

### 1.3 询盘评估标准

**A 类询盘 (优质) - 立即回复**:

```
✅ 产品规格详细
✅ 数量明确 (≥MOQ)
✅ 目标价格合理
✅ 企业邮箱
✅ 公司信息完整
✅ 有明确交期要求
```

**B 类询盘 (一般) - 24 小时内回复**:

```
⭕ 规格基本清楚
⭕ 数量范围
⭕ 免费邮箱但信息完整
⭕ 首次接触
```

**C 类询盘 (低质) - 模板回复**:

```
❌ 只问价格无规格
❌ 数量模糊
❌ 信息不完整
❌ 群发询盘痕迹
```

### 1.4 客户背景调查清单

**基础调查 (必做)**:

```
□ Google 搜索公司名称
□ 查看公司网站
□ LinkedIn 核实联系人
□ 邮箱域名验证
□ 电话确认 (工作时间)
```

**深度调查 (大订单必做)**:

```
□ 海关数据查询 (进口记录)
□ 信用报告购买 ($100-500)
□ 同行评价收集
□ 公司注册信息核实
□ 财务状况评估
```

**高风险国家额外调查**:

```
□ 政治风险评估
□ 外汇管制情况
□ 法律环境调查
□ 过往纠纷记录
```

### 1.6 贸易商注意事项 ⚠️

> **注意 1**: 不要立即报价
> - 先了解客户真实需求
> - 评估客户信用
> - 确认产品可行性
> - 核算成本利润

> **注意 2**: 保护供应商信息
> - 不要透露工厂名称
> - 使用中性包装
> - 签订保密协议
> - 避免工厂直接联系客户

> **注意 3**: 确认贸易术语
> - EXW/FOB/CIF/DDP 明确
> - 责任划分清晰
> - 避免后续纠纷

> **注意 4**: 评估利润空间
> - 工厂价格核实
> - 物流成本计算
> - 关税税费预估
> - 预期利润确保 (15-30%)

## 🔍 第二阶段：供应商筛选与询价 (1-3 天)

### 2.1 供应商筛选流程

```
确定产品要求
    ↓
搜索潜在供应商
    ↓
初步筛选 (资质)
    ↓
发送询价单 (RFQ)
    ↓
收集报价
    ↓
对比评估
    ↓
选定 2-3 家备选
    ↓
样品测试
    ↓
确定供应商
```

### 2.2 供应商资质审核清单

**基础资质 (必须)**:

```
□ 营业执照 (三证合一)
□ 税务登记证
□ 组织机构代码证
□ 生产许可证 (如需要)
□ 出口许可证 (如需要)
```

**质量认证 (优先)**:

```
□ ISO 9001 质量管理体系
□ ISO 14001 环境管理体系
□ CE 认证 (出口欧洲)
□ UL 认证 (出口美国)
□ 其他行业认证
```

**生产能力 (核实)**:

```
□ 工厂面积
□ 员工数量
□ 生产线数量
□ 月产能
□ 主要设备清单
```

**出口经验 (确认)**:

```
□ 出口年限
□ 主要出口市场
□ 年出口额
□ 主要客户
□ 出口单证能力
```

### 2.6 贸易商注意事项 ⚠️

> **注意 1**: 不要只选最便宜
> - 质量稳定性更重要
> - 交期可靠性关键
> - 配合度影响效率
> - 综合评估非单一价格

> **注意 2**: 确认工厂真实性
> - 实地验厂 (或视频验厂)
> - 查看营业执照原件
> - 核实生产能力
> - 警惕贸易公司冒充工厂

> **注意 3**: 保护客户关系
> - 与工厂签保密协议
> - 避免工厂直接联系客户
> - 使用中性包装
> - 控制出货单据

> **注意 4**: 预留利润空间
> - 不要透露客户目标价
> - 报价留谈判余地
> - 考虑汇率波动
> - 核算所有成本

### 3.3 价格谈判策略库

**客户说"太贵了"**:

```
应对策略：强调价值

话术示例:
"我理解价格是您考虑的因素之一。
但是，我们的产品在 [质量/认证/服务] 方面有明显优势。
[具体举例]
从长期使用来看，我们的产品质量更稳定，
可以减少您的售后成本和客户投诉。"
```

**客户说"别家更便宜"**:

```
应对策略：差异化

话术示例:
"市场上确实有价格更低的产品。
但是，价格差异通常来自:
1. 材料质量不同 [具体说明]
2. 生产工艺不同 [具体说明]
3. 质量检验标准不同 [具体说明]
我们可以提供 [样品对比/测试报告]，
让您看到实际差异。"
```

**客户要求"量大优惠"**:

```
应对策略：阶梯报价

话术示例:
"我们理解大批量采购应该有价格优势。
我们的阶梯报价如下:
- 100-500 件：USD [XX]
- 501-1000 件：USD [XX] (优惠 5%)
- 1001 件以上：USD [XX] (优惠 10%)
您的订单数量是？我可以给您最优惠的价格。"
```

**客户说"预算有限"**:

```
应对策略：降级方案

话术示例:
"理解您的预算考虑。
我们可以提供几种方案供您选择:
方案 A: 标准配置 - USD [XX]
方案 B: 简化配置 - USD [XX] (便宜 15%)
方案 C: 经济配置 - USD [XX] (便宜 25%)
主要差异在于 [具体说明]，
核心功能是一样的。"
```

### 3.5 贸易商注意事项 ⚠️

> **注意 1**: 报价有效期
> - 设定有效期 (7-30 天)
> - 原材料波动大时缩短
> - 汇率波动大时注意
> - 过期重新报价

> **注意 2**: 价格保护
> - 不要透露工厂价格
> - 用"我们的价格"表述
> - 避免客户直接找工厂
> - 报价单注明"不得转售"

> **注意 3**: 付款条件
> - 新客户：30%+70% 或 L/C
> - 老客户：可协商 OA 30-60 天
> - 高风险国家：100% 前 T/T
> - 大额订单：L/C 优先

> - 大额订单锁定汇率
> - 或使用人民币报价
> - 约定汇率波动>5% 调整价格
