## 🎯 Skills 定义

```
每个 Skill = 状态×阶段的最优动作单元

64 状态 × 6 阶段 = 384 Skills
```

---

## 📊 MVP 优先策略

**Phase 1：16 Skills（高频×痛点）**

选择逻辑：
- 16 个最高频状态
- 每状态只取第 3 阶段（"开始怀疑"= 最痛点）

```
16 Skills = 用户最愿意付费的 16 个时刻
```

## 🚀 384 Skills 生产计划

### 批量生产脚本

```python
# scripts/generate_384_skills.py

def generate_all_skills():
    """生成 384 Skills"""
    
    # 64 状态定义
    states = load_64_states()
    
    # 6 阶段定义
    stages = load_6_stages()
    
    # 生成 384 Skills
    for state in states:
        for stage in stages:
            skill_id = f"{state['id']}-{stage}"
            
            # AI 生成初稿
            draft = ai_generate_skill(state, stage)
            
            # 心理学专家审核
            reviewed = psychology_review(draft)
            
            # 质量评分
            score = quality_score(reviewed)
            
            # 保存
            if score >= 45:
                save_skill({
                    'id': skill_id,
                    **reviewed,
                    'quality_score': score,
                    'status': 'published'
                })
    
    print(f"✅ 完成 384 Skills 生成")
```
