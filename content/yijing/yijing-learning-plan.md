## 📚 核心概念

### 1. 阴阳 (基础中的基础)

```
阳 ⚊ ─── 主动/刚强/光明/男性/天
阴 ⚋ ─ ─ 被动/柔和/黑暗/女性/地

核心思想：
- 阴阳互根 (相互依存)
- 阴阳转化 (物极必反)
- 阴阳平衡 (中庸之道)
```

### 2. 八卦 (8 个基本卦象)

| 卦名 | 符号 | 自然 | 属性 | 方位 |
|------|------|------|------|------|
| **乾** ☰ | 三连 | 天 | 刚健 | 西北 |
| **坤** ☷ | 六断 | 地 | 柔顺 | 西南 |
| **震** ☳ | 仰盂 | 雷 | 动 | 东 |

| **离** ☲ | 中虚 | 火 | 丽 | 南 |
| **艮** ☶ | 覆碗 | 山 | 止 | 东北 |
| **兑** ☱ | 上缺 | 泽 | 悦 | 西 |

### 3. 64 卦 (八卦两两相重)

```
上卦 (外卦)
  ↓
☰ 乾
☷ 坤

  ↓
☰ 乾
☷ 坤
...

组合示例：
天地否 ☰☷ (乾上坤下)
地天泰 ☷☰ (坤上乾下)
```

## 🔮 核心卦象速查 (精选 10 卦)

### 1. 乾卦 (第 1 卦)

```
䷀ 乾为天

卦辞：元亨利贞
象曰：天行健，君子以自强不息

解读：
- 纯阳之卦，至刚至强
- 适合：开创/领导/进取
- 警示：过刚易折

应用：
- 创业初期
- 职场晋升
- 领导力提升
```

### 2. 坤卦 (第 2 卦)

```
䷁ 坤为地

卦辞：元亨，利牝马之贞
象曰：地势坤，君子以厚德载物

解读：
- 纯阴之卦，至柔至顺
- 适合：配合/承载/包容
- 警示：过柔易弱

应用：
- 团队合作
- 家庭关系
- 资源整合
```

### 3. 泰卦 (第 11 卦)

```
䷊ 地天泰

卦辞：小往大来，吉亨
象曰：天地交，泰

解读：
- 天地交泰，阴阳和合
- 适合：合作/交流/发展
- 警示：盛极必衰

应用：
- 商业合作
- 婚姻关系
- 项目推进
```

### 4. 否卦 (第 12 卦)

```
䷋ 天地否

卦辞：否之匪人，不利君子贞
象曰：天地不交，否

解读：
- 天地不交，阴阳隔绝
- 适合：守成/等待/反思
- 警示：否极泰来

应用：
- 逆境应对
- 风险规避
- 战略收缩
```

### 5. 既济卦 (第 63 卦)

```
䷾ 水火既济

卦辞：亨小，利贞
象曰：水在火上，既济

解读：
- 事已成，阴阳各得其位
- 适合：守成/巩固
- 警示：初吉终乱

应用：
- 项目收尾
- 成果巩固
- 风险防范
```

### 6. 未济卦 (第 64 卦)

```
䷿ 火水未济

卦辞：亨，小狐汔济，濡其尾
象曰：火在水上，未济

解读：
- 事未成，阴阳未和
- 适合：继续奋斗
- 警示：慎始慎终

应用：
- 创业中期
- 持续改进
- 长期主义
```
