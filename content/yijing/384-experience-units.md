# 384 具体体验单元

> 创建：2026-03-31 10:09  
> 核心定义：**状态中的具体某个时刻**  
> 状态：✅ 内部文档（体验设计核心）

---

## 🎯 核心定义

```
64 状态 × 6 阶段 = 384 具体体验单元

每个体验单元 = 一种具体的人生情景 + 发展时刻
```

**体验单元是什么**:
- ✅ 用户在某个状态中的**具体时刻**
- ✅ 可描述、可感知、可共鸣的**真实体验**
- ✅ 有情绪、有认知、有行为模式的**完整切片**

**体验单元不是什么**:
- ❌ 抽象概念（如"积累未显期"）
- ❌ 笼统描述（如"你最近状态不好"）
- ❌ 空洞建议（如"继续努力"）

---

## 📐 计算公式

```
64 状态 × 6 阶段 = 384 体验单元

其中：
- 64 状态 = 64 种人生情景（宏观）
- 6 阶段 = 每种情景内的 6 个发展时刻（微观）
- 384 体验单元 = 384 个具体的"你正在经历什么"（具体时刻）
```

**类比理解**:
```
64 状态 = 64 部电影
6 阶段 = 每部电影的 6 幕
384 体验单元 = 384 个具体场景（观众能共鸣的画面）
```

---

## 🎬 体验单元结构

每个体验单元包含 5 个核心维度：

### 1. 体验描述（用户在经历什么）

```
【表层体验】
- 你在做什么
- 你看到了什么
- 你感受到了什么

【深层体验】
- 你在害怕什么
- 你在渴望什么
- 你在证明什么
```

### 2. 情绪状态（你的感受）

```
主要情绪：焦虑/迷茫/疲惫/期待/...
情绪强度：1-10 分
情绪来源：外部事件/内部冲突/...
```

### 3. 认知特征（你的想法）

```
核心信念： "我必须成功" / "我不够好" / ...
思维模式： 非黑即白 / 灾难化 / 过度概括 / ...
认知偏差： 读心术 / 预测未来 / 贴标签 / ...
```

### 4. 行为模式（你的行动）

```
典型行为： 过度努力 / 逃避 / 讨好 / 控制 / ...
行为频率： 每天 N 次 / 每周 N 次
行为后果： 短期缓解 / 长期恶化 / ...
```

### 5. 身体感受（你的身体）

```
身体信号： 紧绷/疲惫/疼痛/不适/...
身体部位： 头部/肩膀/胃部/胸口/...
身体状态： 失眠/食欲变化/精力下降/...
```

## 🎯 体验单元设计原则

### 1. 具体性原则

```
❌ 抽象："你最近状态不好"
✅ 具体："你每天工作 12 小时，但项目推进缓慢，开始怀疑方向"

❌ 笼统："你需要休息"
✅ 具体："你连续 3 周睡眠不足 6 小时，肩膀持续紧绷"
```

### 2. 共鸣性原则

```
❌ 说教："你应该放下执念"
✅ 共鸣："你在用错误路径证明自己，就像用漏桶打水"

❌ 分析："你的焦虑源于不安全感"
✅ 共鸣："你害怕停下来就会被落下，所以不敢休息"
```

### 3. 可执行原则

```
❌ 空洞："继续努力"
✅ 具体："写下 3 件今天做得好的事，无论多小"

❌ 模糊："调整心态"
✅ 具体："问自己：如果放下'必须成功'，我会怎么做？"
```

### 4. 体验优先原则

```
❌ 概念："这是乾卦初爻"
✅ 体验："你刚开始不对劲，努力但没有回报"

❌ 术语："这是三爻变"
✅ 体验："你开始怀疑路径，质疑当前方向"
```

## 🔧 体验单元应用

### 1. 情景测试匹配

```python
def match_experience_unit(answers: List[Dict]) -> Dict:
    """
    根据测试答案匹配体验单元
    
    输入：用户答案
    输出：匹配的体验单元（状态 + 阶段）
    """
    
    # Step 1: 确定状态（64 之一）
    state = determine_state(answers)
    
    # Step 2: 确定阶段（1-6）
    step = determine_step(answers)
    
    # Step 3: 获取体验单元详情
    experience_unit = get_experience_unit(state, step)
    
    return {
        'state': state,
        'step': step,
        'experience': experience_unit['experience_description'],
        'emotion': experience_unit['emotion_state'],
        'cognition': experience_unit['cognition_features'],
        'behavior': experience_unit['behavior_pattern'],
        'body': experience_unit['body_sensation'],
    }
```

```
✅ 被理解："这就是我正在经历的！"

```
