## 一、太极 (Taiji)

### 核心概念

```
无极 → 太极 → 阴阳
```

**太极图**:
```
        ☯
   阴中有阳，阳中有阴
   阴阳互根，相互转化
```

### 哲学内涵

| 概念 | 含义 | 现代应用 |
|------|------|---------|
| **无极** | 无形无象的混沌状态 | 创业前的空白期 |
| **太极** | 包含阴阳的统一体 | 事物的本源 |
| **阴阳** | 对立统一的两种力量 | 矛盾分析法 |

### 太极智慧

> "无极而太极，太极动而生阳，动极而静，静而生阴"
> 
> **现代解读**: 事物从混沌到有序，从单一到复杂的发展过程

## 二、两仪 (Liangyi - Yin & Yang)

### 阴阳符号

```
阳 ⚊ ───  (单数/奇数)
阴 ⚋ ─ ─  (双数/偶数)
```

### 阴阳属性对照表

| 维度 | 阳 | 阴 |
|------|-----|-----|
| **自然** | 天/日/火 | 地/月/水 |
| **方向** | 上/外/左 | 下/内/右 |
| **性质** | 刚/强/动 | 柔/弱/静 |
| **性别** | 男/父/子 | 女/母/女 |
| **时间** | 昼/春/夏 | 夜/秋/冬 |
| **颜色** | 白/红 | 黑/蓝 |
| **数字** | 奇数 (1,3,5,7,9) | 偶数 (2,4,6,8,10) |

### 阴阳三大规律

#### 1. 阴阳互根
```
孤阴不生，独阳不长
阴中有阳，阳中有阴
```

**现代应用**: 
- 团队合作：男女搭配，优势互补
- 个人发展：刚柔并济，张弛有度

#### 2. 阴阳消长
```
阳极生阴，阴极生阳
物极必反，否极泰来
```

**现代应用**:
- 商业周期：繁荣→衰退→复苏→繁荣
- 人生起伏：高峰时警惕，低谷时坚持

#### 3. 阴阳转化
```
重阴必阳，重阳必阴
寒极生热，热极生寒
```

**现代应用**:
- 危机转机：危险中包含机会
- 成功失败：成功中潜伏危机

## 三、四象 (Sixiang - Four Images)

### 四象生成

```
两仪生四象

阳 + 阳 = 太阳 ⚌ (老阳)
阳 + 阴 = 少阴 ⚍
阴 + 阳 = 少阳 ⚎
阴 + 阴 = 太阴 ⚏ (老阴)
```

| 四象 | 符号 | 阴阳 | 季节 | 方位 | 五行 | 时间 |
|------|------|------|------|------|------|------|
| **太阳** | ⚌ | 阳阳 | 夏 | 南 | 火 | 午 (11-13 点) |
| **少阴** | ⚍ | 阳阴 | 秋 | 西 | 金 | 酉 (17-19 点) |
| **少阳** | ⚎ | 阴阳 | 春 | 东 | 木 | 卯 (5-7 点) |
| **太阴** | ⚏ | 阴阴 | 冬 | 北 | 水 | 子 (23-1 点) |

### 四象演化

```
四象生八卦

太阳 (⚌) + 阳 = 乾 ☰ (天)
太阳 (⚌) + 阴 = 兑 ☱ (泽)

少阴 (⚍) + 阳 = 离 ☲ (火)
少阴 (⚍) + 阴 = 震 ☳ (雷)

少阳 (⚎) + 阳 = 巽 ☴ (风)
少阳 (⚎) + 阴 = 坎 ☵ (水)

太阴 (⚏) + 阳 = 艮 ☶ (山)
太阴 (⚏) + 阴 = 坤 ☷ (地)
```

## 五、64 卦 (64 Hexagrams)

### 生成原理

```
八卦两两相重 = 64 卦

上卦 (外卦)
  ↓
☰ 乾
☷ 坤
...
(8 种)

下卦 (内卦)
  ↓
☰ 乾
☷ 坤
...
(8 种)

8 × 8 = 64 卦
```

### 64 卦结构

```
每卦 6 爻 (从下往上数)

上爻 (第 6 爻) - 天道
五爻 (第 5 爻) - 君位
四爻 (第 4 爻) - 臣位
三爻 (第 3 爻) - 人位
二爻 (第 2 爻) - 地位
初爻 (第 1 爻) - 基础

每爻可以是：
- 阳爻 ⚊ (九)
- 阴爻 ⚋ (六)
```
