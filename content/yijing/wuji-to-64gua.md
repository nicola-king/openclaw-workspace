## 一、无极 (Wuji - 虚无混沌)

### 核心概念

```
无极 = 无 + 极
无：无形无象
极：极限、终点
```

**无极图**:
```
        ○
    空无一物
  混沌未分状态
```

### 哲学内涵

| 维度 | 含义 | 现代对应 |
|------|------|---------|
| **宇宙论** | 宇宙诞生前的混沌状态 | 大爆炸前的奇点 |
| **本体论** | 万物的本源 | 量子场/真空 |
| **认识论** | 超越认知的状态 | 未知领域 |
| **人生论** | 初心/本真状态 | 空杯心态 |

### 无极智慧

> "无极而太极，无形而生有形"
> 
> **现代解读**: 
> - 创业：从 0 到 1 的过程
> - 学习：空杯心态，才能容纳新知
> - 人生：回归初心，找到本源

### 无极应用

| 场景 | 应用 | 方法 |
|------|------|------|
| **创业** | 从 0 到 1 | 放下预设，洞察本质 |
| **学习** | 空杯心态 | 清空成见，开放接纳 |
| **决策** | 回归本源 | 问自己：最初的目标是什么 |
| **冥想** | 入静状态 | 放空思维，观照内心 |

## 二、太极 (Taiji - 阴阳统一)

### 核心概念

```
太极 = 太 + 极
太：最大、最初
极：极限、根本
```

**太极图**:
```
        ☯
   阴中有阳，阳中有阴
   阴阳互根，相互转化
```

### 太极三大规律

#### 1. 阴阳互根

```
孤阴不生，独阳不长
阴中有阳，阳中有阴
```

**图示**:
```
阳鱼 ⚪ 中有阴眼 ●
阴鱼 ● 中有阳眼 ⚪
```

**现代应用**:
- 团队：男女搭配，优势互补
- 个人：刚柔并济，张弛有度
- 管理：恩威并施，奖惩分明

#### 2. 阴阳消长

```
阳极生阴，阴极生阳
物极必反，否极泰来
```

**图示**:
```
阳长阴消 → 阳极 → 阴长阳消 → 阴极 → 阳长阴消
春→夏→秋→冬→春
```

**现代应用**:
- 商业周期：繁荣→衰退→复苏→繁荣
- 人生起伏：高峰时警惕，低谷时坚持
- 情绪管理：喜不过欢，悲不过伤

#### 3. 阴阳转化

```
重阴必阳，重阳必阴
寒极生热，热极生寒
```

**现代应用**:
- 危机转机：危险中包含机会
- 成功失败：成功中潜伏危机
- 健康管理：劳逸结合，防病未然

## 三、两仪 (Liangyi - 阴阳)

### 阴阳符号

```
阳爻 ⚊ ───  (单数/奇数/刚健)
阴爻 ⚋ ─ ─  (双数/偶数/柔顺)
```

### 阴阳属性完整对照表

| 维度 | 阳 (Yang) | 阴 (Yin) |
|------|----------|----------|
| **自然** | 天/日/火/暑 | 地/月/水/寒 |
| **方向** | 上/外/左/南 | 下/内/右/北 |
| **性质** | 刚/强/动/躁 | 柔/弱/静/缓 |
| **性别** | 男/父/子/夫 | 女/母/女/妻 |
| **时间** | 昼/春/夏/晨 | 夜/秋/冬/暮 |
| **颜色** | 白/红/黄 | 黑/蓝/绿 |
| **数字** | 奇数 (1,3,5,7,9) | 偶数 (2,4,6,8,10) |
| **形状** | 圆/方/凸 | 平/凹/曲 |
| **味道** | 辛/甘/酸 | 苦/咸 |
| **声音** | 高/亢/响 | 低/沉/轻 |
| **气味** | 香/腥 | 臭/臊 |
| **状态** | 生/长/散 | 杀/藏/收 |

### 阴阳诊断表

**自我检查**:

| 维度 | 过阳 | 平衡 | 过阴 |
|------|------|------|------|
| **工作** | 过度忙碌 | 劳逸结合 | 过度懒散 |

| **健康** | 上火炎症 | 阴阳平衡 | 寒湿内盛 |
| **社交** | 过度外向 | 内外平衡 | 过度内向 |
| **思维** | 过度理性 | 理性感性平衡 | 过度感性 |

**调整方法**:
- 过阳→补阴：静坐/冥想/休息
- 过阴→补阳：运动/社交/行动

