## 🧠 本质认知

### 产品本质

```
❌ 内容工具
✅ 情绪识别 + 决策映射系统
```

### 用户为什么付费

```
看到一句话
→ 被说中
→ 停下来
→ 想看更多
→ 付 1 元
```

### 为什么"准"

```
不是事件
而是：
→ 内在状态
→ 决策冲突
→ 自我评价
```

## 🔥 爆发原则

### 内容原则

| 对 | 错 |
|------|------|
| "我最近真的很累" | "你需要休息一下" |
| "好像走错方向了" | "建议调整方向" |
| "有点被说中" | "这个测试很准" |
| "你们呢？" | "快来下载" |

**核心**: 像人说话，不像工具/广告

### 开头句原则

```
✅ 像人在说心里话
❌ 像工具/广告/说教
```

### 卡片设计原则

```
✅ 白底黑字多留白
❌ 花哨/复杂/信息密集
✅ 像一句内心话
❌ 像一张海报
```

## 💰 转化闭环

```
看到内容 (爆点句)
  ↓ 共鸣
评论/点赞
  ↓ 好奇
私信/点击
  ↓ 测试

  ↓ 解锁
1 元付费
```

**关键转化句：**
```
"你现在是什么阶段？打出来👇"
"我一个个帮你们看"
"想测的我发你入口"
```

## 🚀 执行优先级

### P0 (必须做)

1. **选爆点句** - 前 3 条选 1

3. **做卡片** - 3 张变体

## ⚠️ 避坑指南

### 内容坑

```
❌ 太抽象："你需要调整认知结构"
❌ 太温和："可能你需要休息一下"
❌ 太说教："你应该这样做"
❌ 太长：超过 30 字没人看
```

### 设计坑

```
❌ 背景太花
❌ 文字太多
❌ 颜色太杂
❌ 不像人话
```

### 运营坑

```
❌ 发了不回评论
❌ 引导太硬
❌ 数据不追踪
❌ 不优化迭代
```

## 📝 每日检查清单

### 发布前

- [ ] 开头句像人话吗？
- [ ] 卡片简洁吗？
- [ ] 有引导评论吗？
- [ ] 发布时间对吗？(09:00/20:00)

### 发布后

- [ ] 前 30 分钟回复每条评论
- [ ] 记录数据 (曝光/点赞/评论/测试/付费)
- [ ] 准备次日内容

### 每周复盘

- [ ] 哪条内容爆了？为什么？
- [ ] 哪个时段数据最好？
- [ ] 哪种开头句转化率最高？
- [ ] 下周优化方向？
