## 为什么是 Polymarket？

Polymarket 是一个去中心化预测市场平台，用户可以：
- 对真实世界事件下注（天气、体育、加密、政治）
- 用 USDC 交易份额
- 24/7 全天候交易

**关键洞察：** 预测市场存在信息差和情绪差，AI 可以系统性地捕捉这些套利机会。

## 知几-E：气象套利策略

### 核心逻辑

1. **数据优势** - 同时接入 NOAA 和 WMO 气象数据
2. **置信度评分** - 96 分以上才执行（满分 100）
3. **优势阈值** - 净优势 ≥ 4.5%（覆盖 2.5% 成本后仍有 2% 利润）
4. **风控系统** - Quarter-Kelly 下注 + 日损熔断

### 技术参数

| 参数 | 值 |
|------|-----|
| 置信度阈值 | 96% |
| 优势阈值 | 4.5% |
| 最大暴露 | 5% |
| 日损熔断 | 10% |
| 下注策略 | Quarter-Kelly |

### 预期收益

- **月胜率**: 20-50%
- **单笔收益**: 2-5%
- **月交易频次**: 30-100 笔
