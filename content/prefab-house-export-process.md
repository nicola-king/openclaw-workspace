### 2.2 价格谈判要点

**买方砍价策略**:
```
1. 量大优惠："如果订购 XX 套，能给多少折扣？"
2. 长期合作："我们有持续采购计划，希望建立长期合作"
3. 竞争对手："其他供应商报价更低"
4. 付款条件："如果提高预付款比例，能否优惠？"
```

**卖方应对策略**:
```
1. 价值强调："我们的质量/服务/认证更有优势"
2. 阶梯报价："XX 套以上可以享受 XX 折扣"
3. 附加服务："可以赠送配件/延长质保"
4. 底线坚持："这已经是最低价，无法再降"
```

### 2.3 贸易术语选择

| 术语 | 卖方责任 | 买方责任 | 推荐场景 |
|------|---------|---------|---------|
| **EXW** | 工厂交货 | 全部物流 | 买方有物流能力 |
| **FOB** | 到港口 | 海运费 + 后续 | 常用 |
| **CIF** | 到目的港 | 清关 + 后续 | 买方无物流经验 |
| **DDP** | 门到门 | 收货即可 | 高端客户/样品单 |

### 3.2 中间贸易商合同

**三方协议要点**:
```
1. 贸易商角色定义
   - 协调沟通
   - 质量把控
   - 物流协调

2. 利润分配
   - 买方支付总价
   - 贸易商扣除佣金 (5-15%)
   - 剩余支付给厂家

3. 责任划分
   - 厂家：生产质量
   - 贸易商：协调沟通
   - 买方：按时付款

4. 保密条款
   - 厂家不得直接联系买方
   - 买方不得直接联系厂家
   - 违约赔偿
```

## 🏭 第四阶段：生产与质检 (15-45 天)

### 4.1 生产流程

```
收到定金
    ↓
原材料采购 (3-5 天)
    ↓
钢结构加工 (5-7 天)
    ↓
墙板生产 (5-7 天)
    ↓
预组装测试 (2-3 天)
    ↓
拆卸包装 (2-3 天)
    ↓
出厂检验 (1-2 天)
    ↓
等待发货
```

### 4.2 质量控制点

**原材料检验**:
```
□ 钢材厚度符合标准
□ 夹芯板密度达标
□ 门窗配件质量
□ 电线电缆认证
□ 五金件防锈处理
```

**生产过程检验**:
```
□ 钢结构焊接质量
□ 墙体垂直度
□ 门窗安装精度
□ 电路布线规范
□ 防水处理
```

**成品检验**:
```
□ 外观检查
□ 尺寸复核
□ 功能测试 (门窗开关、电路)
□ 包装完整性
□ 标识清晰度
```

### 4.3 第三方检验

**检验机构选择**:
| 机构 | 费用 | 时间 | 认可度 |
|------|------|------|--------|
| SGS | USD 300-500 | 1-2 天 | ⭐⭐⭐⭐⭐ |
| BV | USD 300-500 | 1-2 天 | ⭐⭐⭐⭐⭐ |
| TUV | USD 400-600 | 1-2 天 | ⭐⭐⭐⭐⭐ |
| 本地质检 | USD 100-200 | 1 天 | ⭐⭐⭐ |

**检验报告内容**:
```
INSPECTION REPORT

1. 基本信息
   - 合同号
   - 产品信息
   - 检验日期
   - 检验地点

2. 检验结果
   - 外观检查：Pass/Fail
   - 尺寸检查：Pass/Fail
   - 功能测试：Pass/Fail
   - 包装检查：Pass/Fail

3. 照片记录
   - 产品整体照
   - 细节照
   - 包装照
   - 铭牌照

4. 检验结论
   - 合格/不合格
   - 整改建议
   - 复检要求
```

## 🚚 第七阶段：运输与安装 (5-20 天)

### 7.1 内陆运输

**运输方式选择**:
```
平板车 (Flatbed Truck)
├── 适合：标准集装箱
├── 费用：中等
└── 时间：快

低平板车 (Lowboy)
├── 适合：超高货物
├── 费用：较高
└── 时间：快

特种运输
├── 适合：超大件
├── 费用：高
└── 时间：慢
```

### 7.2 安装流程

**安装前准备**:
```
□ 场地平整完成
□ 基础施工完成
□ 水电接口预留
□ 安装工具准备
□ 安装人员到位
□ 天气条件适宜
```

**安装步骤**:
```
1. 卸货 (1 天)
   - 吊车就位
   - 集装箱卸车
   - 摆放定位
```

## ⚠️ 风险控制

### 9.1 主要风险

| 风险类型 | 风险描述 | 风险等级 | 应对措施 |
|---------|---------|---------|---------|
| **付款风险** | 买方不付款 | 高 | 预付款 + 发货前付清 |
| **质量风险** | 产品不合格 | 中 | 第三方检验 + 质保 |
| **物流风险** | 货物损坏/丢失 | 中 | 保险 + 牢固包装 |
| **清关风险** | 无法清关 | 中 | 提前确认单证要求 |
| **汇率风险** | 汇率波动 | 中 | 锁定汇率/多币种 |
| **政治风险** | 政策变化 | 低 | 信保 + 多元化市场 |

### 9.2 付款方式对比

| 方式 | 卖方风险 | 买方风险 | 推荐度 |
|------|---------|---------|--------|
| **T/T 100% 前付** | 低 | 高 | ⭐⭐⭐ |
| **T/T 30% + 70%** | 中 | 中 | ⭐⭐⭐⭐⭐ |
| **L/C 信用证** | 低 | 低 | ⭐⭐⭐⭐ |
| **D/P 付款交单** | 高 | 低 | ⭐⭐ |
| **O/A 赊销** | 高 | 低 | ⭐ |
