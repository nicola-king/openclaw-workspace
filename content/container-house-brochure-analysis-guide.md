## 📋 典型宣传册结构（16-24 页）

### 第 1-2 页：封面 + 公司简介
**包含内容**:
- 公司 LOGO + 品牌名称
- 宣传语/Slogan
- 成立年限
- 生产基地位置
- 企业定位（如"模块化建筑领导者"）

**关键信息点**:
```
✓ 公司名称（全称）
✓ 成立时间（判断经验）
✓ 注册资本（判断实力）
✓ 工厂地址（判断产业带）
```

---

### 第 3-4 页：企业实力展示
**常见内容**:
- 工厂航拍图/车间照片
- 生产线数量
- 年产能数据
- 员工人数
- 研发团队

**典型话术**:
- "占地面积 XX 万平方米"
- "年产能 XXXX 套"
- "拥有专利 XX 项"
- "产品出口 XX 个国家"

**关键信息点**:
```
✓ 生产车间面积
✓ 生产线数量（自动化程度）
✓ 年产能（供货能力）
✓ 出口经验（外贸能力）
```

---

### 第 15-16 页：资质认证
**常见证书**:
| 证书类型 | 作用 | 重要性 |
|---------|------|--------|
| ISO9001 | 质量管理体系 | ⭐⭐⭐ |
| ISO14001 | 环境管理体系 | ⭐⭐ |
| CE 认证 | 欧盟市场准入 | ⭐⭐⭐⭐（出口必备） |
| 防火等级检测 | 防火性能证明 | ⭐⭐⭐⭐⭐ |
| 抗风抗震检测 | 结构安全证明 | ⭐⭐⭐⭐⭐ |
| 建筑资质 | 施工许可 | ⭐⭐⭐⭐ |

**关键信息点**:
```
✓ CE 认证（出口欧洲必需）
✓ 防火等级（A 级不燃）
✓ 抗风等级（12 级台风）
✓ 抗震等级（8 级地震）
```

### 第 17 页：服务流程
**标准流程**:
```
1. 需求沟通 → 2. 方案设计 → 3. 报价确认
       ↓
4. 签订合同 → 5. 工厂生产 → 6. 物流运输
       ↓
7. 现场安装 → 8. 验收交付 → 9. 售后服务
```

**关键时间节点**:
```
• 设计周期：3-7 天
• 生产周期：7-15 天
• 运输时间：视距离而定
• 安装周期：1-3 天（单套）
```

**关键信息点**:
```
✓ 交付周期（从下单到交付）
✓ 是否含安装
✓ 运输方式（整车/拼车）
✓ 质保期限
```

## 🔍 宣传册信息解读技巧

### 一、判断企业实力

| 指标 | 优质信号 | 警示信号 |
|------|---------|---------|
| **成立年限** | 5 年+ | 1 年以内 |
| **注册资本** | 1000 万+ | 100 万以下 |
| **工厂面积** | 1 万㎡+ | 无工厂（贸易公司） |
| **生产线** | 自动化生产线 | 纯手工组装 |
| **案例数量** | 50+ 项目 | 寥寥数个 |
| **认证证书** | CE/ISO 齐全 | 无证书或模糊 |

### 二、判断产品质量

| 关注点 | 优质信号 | 警示信号 |
|--------|---------|---------|
| **框架钢材** | 镀锌钢/厚度≥2mm | 普通钢/厚度<1.5mm |
| **墙板材料** | 岩棉（防火 A 级） | 泡沫（易燃） |
| **墙板厚度** | 75mm/100mm | 50mm 以下 |
| **窗户配置** | 塑钢双层玻璃 | 单层玻璃/铝合金 |
| **防水处理** | 多层防水说明 | 无防水说明 |
| **使用寿命** | 15-20 年 | 5-8 年 |

### 三、判断价格合理性

**价格对比参考**（2026 年行情）:

| 产品类型 | 低价位 | 合理区间 | 高价位 |
|---------|--------|---------|--------|
| 标准集装箱房 | <¥6,000 | ¥8,000-15,000 | >¥20,000 |
| 折叠箱房 | <¥8,000 | ¥10,000-20,000 | >¥30,000 |
| 太空舱 (单人间) | <¥20,000 | ¥25,000-40,000 | >¥60,000 |
| 太空舱 (套房) | <¥40,000 | ¥50,000-100,000 | >¥150,000 |
| 轻钢别墅 | <¥1,200/㎡ | ¥1,500-2,500/㎡ | >¥3,500/㎡ |

⚠️ **警惕超低价**: 可能偷工减料（钢材厚度/防火材料）

## 🎯 决策建议

### 适合合作的情况 ✅
- 成立 5 年+，有自有工厂
- CE/ISO 认证齐全
- 有类似项目案例
- 价格处于合理区间
- 沟通专业、响应及时

### 需要谨慎的情况 ⚠️
- 成立 1 年以内
- 无工厂（纯贸易）
- 无法提供资质证明
- 价格远低于市场价
- 催促快速下单

### 建议实地考察的情况 🔍
- 订单金额>50 万元
- 长期合作意向
- 定制化程度高
- 质量要求严格
