### 📋 10 个核心命令（建议收藏）

1️⃣ **查看状态**
```bash
openclaw status
```
→ Gateway 运行中/会话数/内存占用

2️⃣ **发送消息**
```bash
openclaw message send @team "开会了"
```
→ 微信/Telegram/飞书一键群发

3️⃣ **添加任务**
```bash
openclaw tasks add "写周报"
```
→ 自动加入待办 + 定时提醒

4️⃣ **搜索网络**
```bash
openclaw web-search "AI 趋势"
```
→ 10 条结果 + 摘要

5️⃣ **生成图片**
```bash
openclaw image-generate "赛博朋克城市"
```
→ 4 张图/10 秒

6️⃣ **查看会话**
```bash
openclaw sessions list
```
→ 活跃会话/子代理状态

7️⃣ **系统检查**
```bash
./scripts/heartbeat-cli.sh all
```
→ 一键全检（Gateway/通道/任务/Cron）

8️⃣ **读取文件**
```bash
openclaw read ~/project/README.md
```
→ AI 总结/提取关键信息

9️⃣ **执行脚本**
```bash
openclaw exec ./deploy.sh
```
→ 后台运行 + 日志追踪

🔟 **生成报告**
```bash
openclaw report daily
```
→ 日报/周报/月报自动生成
