## 🚀 快速上线

### 1. 导入项目

1. 下载微信开发者工具
2. 导入 `miniprogram/` 目录
3. 填写你的 AppID

### 2. 修改配置

编辑 `project.config.json`:
```json
{
  "appid": "你的小程序 AppID"
}
```

### 3. 上传代码

1. 点击"上传"按钮
2. 填写版本号和备注
3. 提交审核

### 4. 发布上线

1. 审核通过后
2. 点击"发布"
3. 用户可搜索使用

## 📊 核心功能

### 首页 (pages/index)
- 输入状态描述
- 随机抽取情景
- 跳转到测试页

### 测试页 (pages/test)
- 分析用户输入
- 匹配情景
- 跳转到结果页

### 结果页 (pages/result)
- 显示情景卡片
- 显示步骤进度
- 解锁按钮 (¥1)

### 解锁页 (pages/unlock)
- 完整分析
- 行动建议
- 会员购买 (¥19/月)
